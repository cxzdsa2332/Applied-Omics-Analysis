41  单细胞多组学：CITE-seq 与 10x Multiome 联合分析

Published

June 20, 2026

42 专题概述

细胞的身份并非只由转录组决定。蛋白质表面标志物（流式细胞术几十年来据以分类细胞的依据）和染色质可及性（顺式调控元件的活跃状态）携带了转录组无法提供的正交信息。单细胞多组学技术在同一细胞中同时测量多种模态：CITE-seq 通过抗体偶联寡核苷酸（ADT）在测 mRNA 的同时测表面蛋白；10x Genomics Multiome 从同一细胞核同时测量 RNA 与染色质可及性（scATAC）。本专题讲解两类多组学数据的分析框架，核心是 Seurat 的加权最近邻（Weighted Nearest Neighbor, WNN）算法——它根据每个细胞各模态的信息量自适应赋权，实现真正的多模态联合聚类；以及 Signac 对 scATAC 数据的专门处理流程。

学习目标：理解 CITE-seq 与 10x Multiome 的技术原理与各自的归一化要求；掌握 WNN 多模态聚类的加权逻辑与实操；理解 scATAC 数据稀疏性带来的 LSI 降维需求与质控指标；能完成 scATAC 的跨模态标签传递与 chromVAR motif 活性分析。

43 CITE-seq 多模态分析

43.1 技术原理与 ADT 归一化

CITE-seq（Cellular Indexing of Transcriptomes and Epitopes by Sequencing）将抗体与含条形码的寡核苷酸（Antibody-Derived Tag, ADT）偶联：抗体结合细胞表面蛋白后，其携带的 ADT 与该细胞的 mRNA 共享同一细胞条形码，因而在测序后可同时获得该细胞的转录组与表面蛋白丰度（Stoeckius et al., 2017）。由于表面蛋白标志物的细胞特异性通常强于单个 mRNA（且不受 dropout 困扰），CITE-seq 能解析传统 RNA 聚类难以区分的细胞亚群。

{{Wiki:Single-cell_sequencing}}

ADT 数据的统计性质与 RNA 不同，需要专门的归一化。ADT 计数受抗体非特异性结合产生的背景噪声影响较大，且不同抗体的滴度差异使原始计数不可直接比较。CITE-seq 的标准归一化是 CLR（Centered Log-Ratio，中心化对数比）：对每个细胞（或每个蛋白）取计数相对其几何均值的对数比，从而消除细胞间总 ADT 量的差异并稳定方差。

43.2 WNN 多模态聚类的逻辑

如何把 RNA 与 ADT 两种模态融合为统一的细胞聚类？简单地把两个特征矩阵拼接会失败，因为两种模态的尺度、维度与信息量都不同。Seurat 的 WNN 算法给出了一个自适应的解答（Hao et al., 2021）：对每个细胞，分别计算其在 RNA 空间和 ADT 空间的最近邻；然后评估”哪种模态更能预测这个细胞的邻居”，据此为该细胞的两种模态分配权重；最后用加权组合的距离构建统一的多模态最近邻图。

WNN 的精髓在于权重是逐细胞自适应的：对 RNA 已能清晰区分的细胞类型，RNA 权重高；对 RNA 相似但蛋白质不同的细胞（如 CD4+ 与 CD8+ T 细胞，转录组高度相似但 CD4/CD8 蛋白判然不同），ADT 权重高。这使 WNN 既不浪费任一模态的信息，又不被噪声较大的模态拖累。

43.3 CITE-seq WNN 实操

下面使用脐带血单核细胞 CITE-seq 数据集（cbmc，13 抗体 panel）演示 WNN 流程。

运行环境说明： 本节依赖 R 包 Seurat（v5）与 SeuratData。cbmc 数据集约 50 MB，含 RNA 与 ADT（13 种抗体）两个 assay。首次需 SeuratData::InstallData("cbmc")。

library(Seurat)
library(SeuratData)

# Load CITE-seq data with both RNA and ADT (antibody) assays
cbmc <- LoadData("cbmc")
cbmc <- UpdateSeuratObject(cbmc)

# --- RNA modality: standard log-normalization + PCA ---
DefaultAssay(cbmc) <- "RNA"
cbmc <- NormalizeData(cbmc) |> FindVariableFeatures() |> ScaleData() |> RunPCA()

# --- ADT modality: CLR normalization, then PCA on all proteins ---
DefaultAssay(cbmc) <- "ADT"
VariableFeatures(cbmc) <- rownames(cbmc[["ADT"]])   # use all 13 antibodies
cbmc <- NormalizeData(cbmc, normalization.method = "CLR", margin = 2) |>
    ScaleData() |> RunPCA(reduction.name = "apca")
#| label: fig-wnn-umap
#| fig-cap: "CITE-seq WNN 多模态聚类的 UMAP（cbmc 数据集）。基于 RNA 与 ADT 两种模态加权组合的最近邻图计算，按 WNN 聚类着色。相比单独使用 RNA，WNN 利用 CD4/CD8 等蛋白标志物更清晰地分辨了转录组相似的 T 细胞亚群。"
library(Seurat)

# Weighted Nearest Neighbor: learn per-cell modality weights and build joint graph
cbmc <- FindMultiModalNeighbors(
    cbmc,
    reduction.list = list("pca", "apca"),
    dims.list      = list(1:30, 1:18),     # RNA PCs and ADT PCs
    modality.weight.name = "RNA.weight"
)

cbmc <- RunUMAP(cbmc, nn.name = "weighted.nn", reduction.name = "wnn.umap")
cbmc <- FindClusters(cbmc, graph.name = "wsnn", resolution = 0.8)
DimPlot(cbmc, reduction = "wnn.umap", label = TRUE)

每个细胞的模态权重本身就是有信息的输出：可视化各细胞类型的 RNA 权重，能直观看到哪些细胞类型主要依赖蛋白质信息来界定。

# Cells whose identity is driven by protein (low RNA.weight) vs by transcriptome
VlnPlot(cbmc, features = "RNA.weight", group.by = "seurat_clusters", pt.size = 0) +
    Seurat::NoLegend()

何时 CITE-seq 的价值最大： WNN 的增益主要体现在”RNA 相似但蛋白不同”的细胞对上。对转录组本就泾渭分明的细胞类型（如 B 细胞 vs 单核细胞），ADT 几乎不提供额外分辨力。因此设计 CITE-seq 抗体 panel 时，应优先纳入能区分转录组近邻亚群的标志蛋白（如 T 细胞的 CD4/CD8/CD45RA/CD45RO），而非已能被 RNA 轻易区分的谱系标志。

44 10x Multiome 与 scATAC-seq 分析

44.1 scATAC-seq 的数据特性

10x Genomics Multiome 平台从同一细胞核同时测量 RNA 与染色质可及性。染色质可及性由 scATAC-seq（single-cell Assay for Transposase-Accessible Chromatin）测量：Tn5 转座酶优先在开放（无核小体占据）的染色质区域插入测序接头，因此 reads 的富集位置标记了活跃的顺式调控元件（启动子、增强子）。

{{Wiki:ATAC-seq}}

scATAC 数据与 scRNA 有本质不同，决定了完全不同的分析策略：

  • 极度稀疏：每个细胞只有数千个开放区域被采样，peak×cell 矩阵约 98% 为零，且大多为二值（0/1，因为一个区域在一个细胞中最多两个等位基因）。
  • 高维：peak 数量（数万至数十万）远多于基因数。
  • 质控指标不同：用 TSS 富集得分（reads 在转录起始位点的富集程度，反映信噪比）、核小体信号（片段长度分布是否呈现核小体周期性）、以及 FRiP（reads 落在 peak 内的比例）评估细胞质量。

44.2 LSI 降维与 Gene Activity

由于 scATAC 矩阵的稀疏二值性质，直接用 PCA 效果不佳。Signac（基于 Seurat 框架的 scATAC 分析包）采用 LSI（Latent Semantic Indexing，潜在语义索引）降维（Stuart et al., 2021）：先做 TF-IDF 加权（term frequency–inverse document frequency，提升稀有但有信息的 peak、压低普遍开放的 peak），再做 SVD 奇异值分解提取主成分。LSI 借自文本挖掘——把”细胞”类比为文档、“peak”类比为词，思路高度契合稀疏二值数据。

为了给 scATAC 细胞做初步注释，还需把不可直接解读的 peak 信号转换为类基因表达的量：Gene Activity Score 将每个基因的基因体加上游启动子区域内的 peak 信号汇总，得到一个”基因活性矩阵”，可与 scRNA 的标志基因对照，用于跨模态注释。

44.3 10x Multiome 联合分析实操

运行环境说明： 本节依赖 R 包 Seurat、Signac 及基因组注释包（如 EnsDb.Hsapiens.v86）。数据集 pbmcMultiome（SeuratData，11,909 个 PBMC，含配对 RNA + ATAC）约 200 MB。Signac 的部分步骤（如片段文件操作）需要 fragments.tsv.gz 文件，属较重的预处理。

library(Seurat)
library(Signac)
library(SeuratData)

# Load paired RNA + ATAC measured from the same nuclei
pbmc <- LoadData("pbmcMultiome", "pbmc.rna")
atac <- LoadData("pbmcMultiome", "pbmc.atac")
pbmc[["ATAC"]] <- atac[["ATAC"]]      # attach ATAC assay to the RNA object

# --- RNA modality: standard SCT pipeline ---
DefaultAssay(pbmc) <- "RNA"
pbmc <- SCTransform(pbmc) |> RunPCA()

# --- ATAC modality: TF-IDF + SVD (LSI), the standard scATAC dimensionality reduction ---
DefaultAssay(pbmc) <- "ATAC"
pbmc <- RunTFIDF(pbmc)
pbmc <- FindTopFeatures(pbmc, min.cutoff = "q0")
pbmc <- RunSVD(pbmc)                  # produces the 'lsi' reduction
# Joint WNN clustering over RNA (PCA) and ATAC (LSI), same algorithm as CITE-seq.
# LSI component 1 captures sequencing depth and is excluded (dims 2:30).
pbmc <- FindMultiModalNeighbors(
    pbmc,
    reduction.list = list("pca", "lsi"),
    dims.list      = list(1:30, 2:30)
)
pbmc <- RunUMAP(pbmc, nn.name = "weighted.nn", reduction.name = "wnn.umap")
pbmc <- FindClusters(pbmc, graph.name = "wsnn", resolution = 0.8)

44.4 跨模态标签传递与 chromVAR motif 活性

新产生的 scATAC 数据通常缺乏注释。可用 FindTransferAnchors() 以一个已注释的 scRNA-seq 数据集为参考，通过基因活性矩阵在两种模态间建立锚点，将细胞类型标签从参考”传递”到 scATAC 细胞上：

# Label transfer: annotate ATAC cells using a labeled scRNA-seq reference (PBMC 3k)
anchors <- FindTransferAnchors(
    reference = pbmc3k_annotated, query = pbmc,
    reduction = "cca"
)
predictions <- TransferData(anchorset = anchors,
                            refdata = pbmc3k_annotated$celltype)
pbmc <- AddMetaData(pbmc, metadata = predictions)

scATAC 的独特优势是能直接读出转录因子的调控活性。chromVAR 计算每个细胞中各 TF 结合 motif 所在 peak 的可及性偏差（相对于背景的富集），从而给出”TF motif 活性”——这是 scRNA 无法直接测量的染色质层面调控信号（Schep et al., 2017）：

#| label: fig-chromvar-motif
#| fig-cap: "chromVAR 推断的转录因子 motif 活性（pbmcMultiome scATAC 模态）。颜色编码每个细胞中该 TF motif 所在开放区域的可及性偏差（deviation z-score）。T 细胞亚群在 TBX21、EOMES 等 motif 上的高活性，与其谱系特异的转录因子调控程序一致，为 scRNA 的表达层面证据提供了染色质层面的正交验证。"
library(Signac)

# Add motif information and compute per-cell TF motif accessibility deviations
pbmc <- RunChromVAR(pbmc, genome = BSgenome.Hsapiens.UCSC.hg38)
DefaultAssay(pbmc) <- "chromvar"
FeaturePlot(pbmc, features = "MA0690.1", reduction = "wnn.umap")  # e.g., TBX21 motif

黄金法则： 多组学的价值不在于”测得更多”，而在于正交验证与因果链接。CITE-seq 用蛋白质验证 RNA 定义的细胞身份；10x Multiome 把基因表达（RNA）与其上游顺式调控（ATAC、TF motif 活性）在同一细胞内直接关联，从而能够检验”某增强子可及性升高是否伴随其靶基因表达升高”这类因果假设——这是任何单一模态都无法回答的。

45 方法总结与衔接说明

本专题展示了在同一细胞中整合多种分子模态的两条技术路线：CITE-seq 联合 RNA 与表面蛋白，10x Multiome 联合 RNA 与染色质可及性，二者都用 WNN 算法实现自适应加权的多模态聚类。这些多组学测量为前几个专题的计算推断提供了正交的实验证据——例如 scATAC 的 TF motif 活性可验证 SCENIC（专题 7.3）的 Regulon 推断。模块 A、B 至此完成了单细胞层面从整合、动态到调控、多组学的高级分析。模块 C（专题 7.5）将把视角扩展到空间维度，用本章产出的注释图谱作为参考，解析组织中细胞类型的空间分布。

46 课后自测 (Post-Lesson Quiz)

  1. CITE-seq 的 ADT（Antibody-Derived Tag）是如何实现”同一细胞中同时测量 mRNA 与表面蛋白”的？为什么表面蛋白标志物在区分细胞亚群时常优于单个 mRNA？

  2. 为什么 ADT 数据需要 CLR（中心化对数比）归一化而非 RNA 常用的对数归一化？CLR 消除了什么差异？

  3. WNN 算法如何为每个细胞的两种模态分配权重？为什么权重必须是”逐细胞自适应”而非全局固定的？

  4. 列举 scATAC-seq 区别于 scRNA-seq 的三个数据特性，并说明这些特性为何导致 scATAC 需要 LSI 而非 PCA 降维。

  5. 什么是 Gene Activity Score？它在 scATAC 分析流程中起什么作用？

  6. 在 10x Multiome 的 ATAC LSI 降维中，为什么通常排除第一个 LSI 成分（使用 dims 2:30）？

  7. chromVAR 计算的”TF motif 活性”是什么？为什么说它提供了 scRNA 无法直接测量的调控信号？

  8. 有人认为”既然多组学测得更多，就应该总是用多组学替代单组学”。请结合”正交验证与因果链接”以及 CITE-seq 抗体 panel 设计原则，评价这一观点。

参考答案 (Reference Answers)

  1. 抗体偶联了含细胞条形码体系的寡核苷酸（ADT），抗体结合细胞表面蛋白后，其 ADT 与该细胞的 mRNA 共享同一细胞条形码，测序后即可同时获得转录组与表面蛋白丰度。表面蛋白常优于单个 mRNA 是因为蛋白标志物的细胞特异性通常更强，且不受 dropout 困扰，测量更稳定。

  2. ADT 计数受抗体非特异性结合的背景噪声影响大，且不同抗体滴度差异使原始计数不可直接比较。CLR 对每个细胞取计数相对其几何均值的对数比，消除了细胞间总 ADT 量（测序/结合效率）的差异并稳定方差，使不同蛋白、不同细胞的 ADT 值可比。

  3. WNN 对每个细胞分别计算其在 RNA 与 ADT 空间的最近邻，评估哪种模态更能预测该细胞的邻居，据此为该细胞的两种模态分配权重，再用加权距离构建统一的多模态最近邻图。权重必须逐细胞自适应，因为不同细胞类型的信息来源不同：RNA 能清晰区分的细胞 RNA 权重高，而 RNA 相似但蛋白不同的细胞（如 CD4+/CD8+ T 细胞）ADT 权重高；全局固定权重无法兼顾。

  4. 三个特性：（1）极度稀疏（约 98% 为零）且多为二值；（2）高维（peak 数远多于基因数）；（3）质控指标不同（TSS 富集、核小体信号、FRiP）。稀疏二值的高维矩阵不适合直接 PCA，LSI 先用 TF-IDF 加权提升稀有但有信息的 peak、压低普遍开放的 peak，再用 SVD 提取主成分，更契合此类数据（类比文本挖掘中的文档-词矩阵）。

  5. Gene Activity Score 把每个基因的基因体加上游启动子区域内的 peak 信号汇总，得到类基因表达的”基因活性矩阵”。作用是把不可直接解读的 peak 信号转换为可与 scRNA 标志基因对照的量，用于 scATAC 细胞的初步注释与跨模态标签传递。

  6. 因为第一个 LSI 成分通常高度相关于每个细胞的测序深度（总片段数）这一技术变量，而非生物学变异。排除它（用 dims 2:30）可避免降维与下游聚类被测序深度主导。

  7. chromVAR 计算每个细胞中各 TF 结合 motif 所在 peak 的可及性偏差（相对背景的富集 z-score），即”TF motif 活性”。它提供 scRNA 无法直接测量的信号，因为它反映的是染色质层面 TF 结合位点的开放程度（上游调控可及性），而非 TF 或靶基因的 mRNA 表达；这是顺式调控活跃状态的直接读数。

  8. 该观点片面。多组学的价值不在”测得更多”，而在正交验证与因果链接：CITE-seq 用蛋白验证 RNA 定义的身份、Multiome 把表达与上游顺式调控在同一细胞关联以检验因果假设。但增益有条件——WNN 对转录组本就泾渭分明的细胞类型几乎无额外分辨力，CITE-seq 的 panel 必须针对性纳入能区分 RNA 近邻亚群的标志蛋白才有用。多组学成本更高、分析更复杂，应在单组学不足以回答科学问题时按需使用，而非无差别替代。
