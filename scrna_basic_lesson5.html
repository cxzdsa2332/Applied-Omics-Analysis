36  空间数据质控、归一化与空间高变基因分析

Published

June 20, 2026

37 专题概述

加载 Visium 数据之后，分析流程与 scRNA-seq 高度共享——质控、归一化、降维、聚类的逻辑一脉相承。但空间数据多了一个其他组学没有的维度：坐标。这使它既需要结合组织图像做空间质检，又催生了一类全新的核心分析——识别那些表达呈现非随机空间分布的基因。本专题讲解空间数据的质控与归一化要点，重点阐述空间高变基因（Spatially Variable Genes, SVG）的识别原理（Moran’s I 空间自相关）与空间聚类方法（普通聚类 vs 纳入空间坐标的 BayesSpace），完成 Visium 基础分析的闭环。

学习目标：掌握空间数据质控相比 scRNA-seq 的额外考量；理解 SVG 的概念及 Moran’s I 空间自相关统计量的原理；能用 FindSpatiallyVariableFeatures() 识别并可视化 SVG；理解普通聚类与 BayesSpace 在是否利用空间坐标上的区别；能生成有生物学意义的空间域可视化。

38 空间数据质控与归一化

38.1 结合图像的空间质检

空间数据的质控指标与 scRNA-seq 相同——每 Spot 检出基因数、每 Spot UMI 数、线粒体比例——但多了一层空间维度的判读。一个 Spot 即便表达量达标，若它位于组织边缘、脱落区域或折叠处，也应被剔除。因此空间质控必须结合 H&E 图像进行视觉检查，而非只看数值阈值。

library(Seurat)

# Standard per-Spot QC metrics, same as scRNA-seq
brain[["percent.mt"]] <- PercentageFeatureSet(brain, pattern = "^mt-")

# Inspect QC distributions spatially: low-count Spots often cluster at tissue edges
SpatialFeaturePlot(brain, features = c("nCount_Spatial", "percent.mt"))

别用同一阈值无差别过滤所有 Spot： Visium 中每个 Spot 的 UMI 总数还反映了真实的生物学差异——细胞密度高的区域（如脑的细胞密集层）天然有更高的 UMI，细胞稀疏区域（如白质、纤维区）UMI 本就低。若用 scRNA-seq 式的统一下限过滤，会系统性地删除低细胞密度的组织区域，造成空间偏差。空间质控应把”低 UMI 是技术问题还是生物学真实”结合图像区分对待。

38.2 归一化的空间考量

空间数据可沿用 scRNA-seq 的归一化策略，SCTransform() 同样适用且推荐。但需注意一个空间特有的混杂因素：Spot 的”测序深度”（总 UMI）不仅受技术捕获效率影响，还受该 Spot 覆盖的细胞数与细胞密度影响。SCTransform 的方差稳定能在一定程度上缓解，但在解读时应警惕——表达模式可能部分由细胞密度的空间梯度驱动，而非单纯的基因调控差异。

library(Seurat)

# SCTransform for spatial data: note the assay is "Spatial", not "RNA"
brain <- SCTransform(brain, assay = "Spatial", verbose = FALSE)

39 空间高变基因分析

39.1 什么是空间高变基因

scRNA-seq 的高变基因（专题 6.2）衡量基因在细胞间的总体变异，但忽略了空间结构。空间高变基因（Spatially Variable Genes, SVG）则专门识别那些表达呈现显著空间自相关的基因——即在组织上相邻的 Spot 倾向于有相似表达、表达模式不是空间随机散布的。这类基因往往标志着功能特异的组织层、解剖区域或局部微环境，是空间转录组最具特色的发现。

普通高变基因与 SVG 的区别在于：一个基因可以在细胞间高度变异（是 HVG），但其高低表达细胞在空间上随机分布（不是 SVG）；反之，一个总体变异不大的基因，若其表达在某个特定区域系统性升高，则是 SVG。

39.2 Moran’s I 空间自相关

识别 SVG 最常用的统计量是 Moran’s I 空间自相关指数。它衡量”相邻位置的取值是否相似”：

\[I = \frac{N}{\sum_{i}\sum_{j} w_{ij}} \cdot \frac{\sum_{i}\sum_{j} w_{ij}(x_i - \bar{x})(x_j - \bar{x})}{\sum_{i}(x_i - \bar{x})^2}\]

其中 \(x_i\) 为 Spot \(i\) 的表达量，\(\bar{x}\) 为均值，\(w_{ij}\) 为空间权重矩阵（相邻 Spot 权重大、远处为 0），\(N\) 为 Spot 总数。Moran’s I 的取值与解释类似相关系数：

  • \(I > 0\)：正空间自相关——相邻 Spot 表达相似，呈现空间聚集模式（典型的 SVG）。
  • \(I \approx 0\)：无空间自相关——表达空间随机分布。
  • \(I < 0\)：负空间自相关——相邻 Spot 表达相反（罕见，如棋盘格模式）。

{{Wiki:Spatial_analysis}}

空间权重矩阵 \(w_{ij}\) 的构建是关键：对 Visium 的六角网格，通常以每个 Spot 的直接相邻 Spot（六个邻居）为权重非零项，也可用 k-近邻定义。

#| label: fig-svg
#| fig-cap: "小鼠脑前切片识别出的空间高变基因（SVG）的空间表达（stxBrain Visium 数据）。展示 Moran's I 排名靠前的若干基因，每个 Spot 按表达量着色。这些基因呈现出与脑解剖结构吻合的清晰空间聚集模式（如特定皮层、海马区），印证了它们的高空间自相关性。"
library(Seurat)

# Identify spatially variable features using Moran's I autocorrelation
brain <- FindSpatiallyVariableFeatures(
    brain, assay = "SCT",
    features = VariableFeatures(brain)[1:1000],
    selection.method = "moransi"
)

# Visualize the top spatially variable genes on the tissue
top_svg <- head(SpatiallyVariableFeatures(brain, selection.method = "moransi"), 6)
SpatialFeaturePlot(brain, features = top_svg, ncol = 3)

除基于 Moran’s I 的方法外，SpatialDE 提供了另一种思路：用高斯过程（Gaussian Process）回归把每个基因的空间表达分解为”空间结构成分”与”非空间噪声成分”，据此检验是否存在显著的空间表达模式（Svensson et al., 2018）。两类方法可相互验证。

40 空间聚类

40.1 普通聚类 vs 空间感知聚类

识别组织的空间域有两种思路，区别在于是否利用 Spot 的坐标信息：

  • 普通聚类（忽略坐标）：完全沿用 scRNA-seq 流程——对 Spot 表达做 PCA、建 KNN 图、Leiden 聚类。它只用表达相似性，不考虑 Spot 是否空间相邻。优点是简单、与 scRNA-seq 一致；缺点是结果可能出现空间上不连贯的”碎片化”域（相邻 Spot 被分到不同簇）。
  • 空间感知聚类（纳入坐标）：BayesSpace 等方法在聚类时同时利用表达与空间坐标，通过马尔可夫随机场（Markov Random Field, MRF）先验，鼓励空间相邻的 Spot 归入同一域（Zhao et al., 2021）。它能产生更平滑、更符合解剖连续性的空间域，并支持亚 Spot 分辨率增强。
library(Seurat)

# Approach A: ordinary clustering (ignores coordinates), reusing the scRNA pipeline
brain <- RunPCA(brain, assay = "SCT") |>
    FindNeighbors(dims = 1:30) |>
    FindClusters(resolution = 0.5) |>
    RunUMAP(dims = 1:30)

# Visualize the resulting spatial domains on the tissue section
SpatialDimPlot(brain, label = TRUE, label.size = 3)

40.2 BayesSpace 的空间先验

BayesSpace 的核心是 MRF 先验：它假设组织在空间上是连续的，因此一个 Spot 属于某空间域的概率，应受其邻居所属域的影响——相邻 Spot 更可能属于同一域。这一先验把”空间连续性”这一组织的基本生物学性质编码进了聚类模型，从而抑制孤立的、空间不连贯的聚类噪声。其关键超参数是域数量 q，需结合已知解剖结构（如小鼠脑的主要分区数）选定。

何时需要空间感知聚类： 并非所有研究都需要 BayesSpace。若研究目标只是”组织里有哪些转录状态”，普通聚类已足够；若目标是精确划定空间域边界、研究组织的空间组织结构（如肿瘤微环境分区、发育中的形态发生场），则空间感知聚类的连续性约束更有价值。可先用普通聚类快速探索，再视需要用 BayesSpace 精细化。

#| label: fig-spatial-domains
#| fig-cap: "小鼠脑前切片的空间域聚类结果叠加在组织切片上（stxBrain Visium 数据）。每个 Spot 按其所属空间域着色，绘制在真实组织坐标上。聚类得到的空间域与已知小鼠脑解剖分区（如皮层各层、海马、白质）对应，验证了空间聚类捕捉组织结构的能力。"
library(Seurat)

# Overlay clusters on tissue and compare against known mouse-brain anatomy
SpatialDimPlot(brain, label = TRUE)

黄金法则： 空间聚类的结果必须接受解剖学检验——把空间域叠加到 H&E 图像上，看它是否与已知的组织结构（皮层分层、海马形态、器官分区）吻合。吻合则说明聚类捕捉到了真实的组织组织结构；若空间域杂乱、与任何解剖结构都对不上，则要么是分辨率/参数不当，要么是质控未清除的技术伪影在作祟。解剖一致性是空间分析可信度的终极判据。

41 方法总结与衔接说明

本专题完成了 Visium 空间转录组的基础分析闭环：结合图像的空间质控、SCTransform 归一化、基于 Moran’s I 的空间高变基因识别，以及普通聚类与 BayesSpace 空间感知聚类。这些方法与模块 A 的 scRNA-seq 流程在质控、归一化、聚类逻辑上高度统一，差异集中在”空间坐标”这一额外维度。至此，第6章建立了单细胞与空间转录组的基础分析能力。但 Visium 分析仍有一个核心问题悬而未决——每个 Spot 的混合信号究竟由哪些细胞类型、以何种比例构成？这一细胞解卷积问题，连同拟时序、细胞通讯、多组学等高级专题，将在第7章系统展开。

42 课后自测 (Post-Lesson Quiz)

  1. 空间数据的质控相比 scRNA-seq 多了哪一层判读？为什么即便表达量达标的 Spot 也可能需要剔除？

  2. 为什么不能用 scRNA-seq 式的统一 UMI 下限无差别过滤所有 Visium Spot？这样做会造成什么空间偏差？

  3. 什么是空间高变基因（SVG）？它与普通高变基因（HVG）的概念区别是什么？请举例说明一个基因可以是 HVG 却不是 SVG。

  4. Moran’s I 空间自相关指数衡量什么？\(I > 0\)、\(I \approx 0\)、\(I < 0\) 分别对应什么样的空间表达模式？

  5. 在 Moran’s I 计算中，空间权重矩阵 \(w_{ij}\) 起什么作用？对 Visium 的六角网格通常如何定义它？

  6. 普通聚类与 BayesSpace 空间感知聚类在是否利用空间坐标上有何区别？为什么普通聚类可能产生空间上”碎片化”的域？

  7. BayesSpace 的马尔可夫随机场（MRF）先验编码了组织的什么基本性质？它如何抑制空间不连贯的聚类噪声？

  8. 一位研究者用普通聚类得到的某个空间域在组织上呈零散点状分布、与任何已知解剖结构都不吻合。请说明应如何判断这是真实的生物学信号还是技术问题，并提出处理思路。

参考答案 (Reference Answers)

  1. 多了结合 H&E 图像的空间维度判读。因为一个 Spot 即便表达量（基因数/UMI）达标，若它位于组织边缘、脱落区域或折叠处，其信号也不可靠或不代表完整组织，应被剔除。这类问题只能通过结合图像的视觉检查发现，而非单看数值阈值。

  2. 因为 Visium 中 Spot 的 UMI 总数还反映真实生物学差异——细胞密集区天然高 UMI、细胞稀疏区（白质、纤维区）天然低 UMI。用统一下限过滤会系统性删除低细胞密度的组织区域，造成空间偏差，使分析偏向高密度区域。应结合图像区分”低 UMI 是技术问题还是生物学真实”。

  3. SVG 是表达呈现显著空间自相关（相邻 Spot 表达相似、非空间随机）的基因，标志功能特异的组织层/区域。它与 HVG 的区别在于 HVG 只衡量细胞/Spot 间的总体变异、不考虑空间结构。例子：一个基因在 Spot 间高度变异（是 HVG），但其高低表达 Spot 在空间上随机散布、无聚集模式，则它不是 SVG。

  4. Moran’s I 衡量相邻位置取值是否相似（空间自相关）。\(I>0\) 为正空间自相关，相邻 Spot 表达相似、呈空间聚集（典型 SVG）；\(I\approx 0\) 为无空间自相关，表达空间随机分布；\(I<0\) 为负空间自相关，相邻 Spot 表达相反（如棋盘格模式，罕见）。

  5. 空间权重矩阵 \(w_{ij}\) 定义了哪些 Spot 算”相邻”以及相邻的程度——相邻 Spot 权重大、远处为 0，从而把”空间邻近”信息引入统计量。对 Visium 六角网格，通常以每个 Spot 的六个直接相邻 Spot 为权重非零项，或用 k-近邻定义邻域。

  6. 普通聚类只用表达相似性、忽略坐标（PCA + KNN + Leiden）；BayesSpace 在聚类时同时利用表达与空间坐标，通过 MRF 先验鼓励相邻 Spot 归入同一域。普通聚类可能碎片化，是因为它不考虑空间连续性，表达上略有差异的相邻 Spot 可能被分到不同簇，产生空间上不连贯的零散域。

  7. MRF 先验编码了”组织在空间上是连续的”这一基本性质：一个 Spot 属于某空间域的概率受其邻居所属域影响，相邻 Spot 更可能属于同一域。它通过让孤立 Spot 的归属向其邻居靠拢，抑制了空间不连贯的、孤立的聚类噪声，产生更平滑、更符合解剖连续性的空间域。

  8. 零散点状、不吻合任何解剖结构，提示可能是技术问题或过度切分而非真实空间域。判断与处理：（1）把该域叠加到 H&E 图像，检查这些 Spot 是否对应组织折叠、边缘或损伤；（2）检查这些 Spot 的 QC 指标（UMI、基因数、线粒体比例）是否异常，若异常则为质控残留的低质量 Spot；（3）尝试降低聚类分辨率或改用 BayesSpace 的空间连续性约束，看该域是否消失或并入相邻域。若确认为伪影或技术噪声，应在质控中剔除相关 Spot 后重新聚类。
