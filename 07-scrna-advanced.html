38 整体概述

在细胞类型鉴定与空间聚类的基础上（第6章），单细胞和空间转录组数据还蕴藏着更深层次的生物学信息：来自不同样本与批次的数据如何整合为可比较的统一图谱？疾病与对照之间，哪些细胞类型的比例发生改变、哪些基因发生差异表达？细胞是如何分化的，分化过程是否具有方向性？不同细胞类型之间通过哪些分子对话相互协调？转录因子如何驱动细胞状态的转变？混合了多种细胞类型的 Visium Spot，各个细胞类型分别位于组织的哪些区域？本章围绕上述问题，系统讲解单细胞与空间转录组的高级计算分析方法。内容从多样本研究的统计基石出发，依次涵盖跨样本整合与多样本差异分析、拟时序与 RNA Velocity、细胞通讯与基因调控网络推断、单细胞多组学，以及以细胞解卷积和 MERFISH 为代表的空间转录组高级分析。

学习目标：掌握跨样本整合（Harmony/scVI）与拟bulk差异表达、差异丰度检验的统计逻辑；理解拟时序分析的算法逻辑与 Monocle 3 的实操方法；掌握 CellChat 配体-受体推断与 SCENIC 调控网络推断框架；能运行 CITE-seq WNN 多模态分析和 10x Multiome 联合分析；掌握 cell2location 细胞解卷积的原理与实操，了解 MERFISH 成像型空间转录组的分析流程。

39 结构导览

本章分为三个模块，共五个专题，在第6章的分析结果之上构建高级推断层次：

输入：第6章的细胞注释 Seurat 对象 + Visium 空间对象
  │
  ├── 模块 A：多样本统计与动态分析（专题 7.1–7.2）
  │     ↓ 跨样本整合 / 批次校正（Harmony / scVI）
  │     ↓ 拟bulk差异表达（pseudobulk + DESeq2）+ 差异丰度（miloR）
  │     ↓ 拟时序分析（Monocle 3 / Slingshot）
  │     ↓ RNA Velocity（scVelo）→ 轨迹方向性
  │
  ├── 模块 B：细胞通讯、调控与多组学（专题 7.3–7.4）
  │     ↓ 细胞通讯分析（CellChat）
  │     ↓ 基因调控网络推断（SCENIC）
  │     ↓ CITE-seq WNN 多模态聚类（cbmc 数据集）
  │     ↓ 10x Multiome RNA + ATAC 联合分析（pbmcMultiome 数据集）
  │
  └── 模块 C：空间转录组高级分析（专题 7.5）
        ↓ 细胞解卷积（cell2location / RCTD）
        ↓ 与 scRNA-seq 参考图谱整合 → 细胞类型空间定位
        ↓ MERFISH 成像型 ST 技术原理与分析流程

三个模块在方法论上递进，但各专题可以相对独立地学习；模块 C 的解卷积分析依赖模块 A、B 产出的经过注释的 scRNA-seq 参考图谱。多样本整合（专题 7.1）是其余所有高级分析的前置步骤——只有先消除批次效应、得到统一的细胞状态空间，下游的轨迹、通讯、调控与空间映射才具有可比性。

40 模块 A：多样本统计与动态分析

模块目标： 掌握跨样本整合与批次校正的方法选择逻辑；理解拟bulk差异表达与差异丰度检验为何是多样本研究的正确统计框架；理解拟时序推断的算法假设与适用场景；掌握 RNA Velocity 的动力学原理。

40.1 专题 7.1：跨样本整合、批次校正与多样本差异分析

现代单细胞研究极少只测一个样本。一旦纳入多个供体、多个批次或多种处理条件，技术批次效应便会与生物学信号混杂：直接合并多样本数据做 PCA 与聚类，往往得到的是”按样本分开”而非”按细胞类型分开”的聚类结果。跨样本整合（Integration）的目标是在保留真实生物学差异的前提下，将不同批次的同类细胞对齐到统一的低维空间。在得到整合后的细胞图谱之后，多样本研究的两个核心统计问题随之而来：不同条件之间，哪些细胞类型的比例发生了改变（差异丰度，Differential Abundance），以及某一细胞类型内部哪些基因发生了差异表达（差异表达，Differential Expression）。本专题强调一条贯穿始终的统计原则：以”细胞”为重复单位会严重低估方差、放大假阳性，正确的做法是以”生物学样本”为重复单位。

核心概念：

  • 批次效应的来源：供体个体差异、样本处理时间、测序批次、10x 化学版本；与生物学信号的混杂模式
  • 整合方法谱系：Seurat CCA/RPCA 锚点（Anchor）→ Harmony（PCA 空间的迭代聚类-校正）→ scVI（变分自编码器，深度生成模型）
  • Harmony 原理：在 PCA 嵌入上迭代执行”软聚类 + 批次特异性校正”，计算高效、对大数据友好
  • scVI 原理：以负二项分布建模原始计数，将批次作为协变量，输出批次校正后的隐变量空间与归一化表达
  • 整合质量评估：批次混合度（iLISI/kBET）与生物学保真度（cLISI）的权衡；过度整合（over-correction）的识别
  • 拟bulk差异表达（Pseudobulk DEA）：将同一样本同一细胞类型的计数加和为”拟bulk”样本，再用 DESeq2/edgeR 检验；为何优于逐细胞 Wilcoxon 检验
  • 差异丰度分析：miloR 的邻域（neighborhood）差异丰度检验；scCODA 的贝叶斯组成数据模型；组成数据的”此消彼长”约束
  • 重复单位的黄金法则：以生物学样本（而非细胞）为统计重复，避免伪重复（pseudo-replication）

实践内容：构造含批次效应的多样本 PBMC 数据，对比未整合 vs Harmony vs Seurat RPCA 的 UMAP；用 scater/scran 将注释后的细胞聚合为拟bulk矩阵并用 DESeq2 做条件间差异表达；运行 miloR 邻域差异丰度检验并用 beeswarm 图可视化各细胞状态的丰度变化。

40.2 专题 7.2：拟时序分析与 RNA Velocity

细胞分化是一个连续的动态过程，但 scRNA-seq 对每个细胞只产生一次”快照”。拟时序分析（Trajectory Inference, TI）通过假设转录组相似的细胞处于相近的分化状态，利用细胞间的转录组距离重建分化路径，给每个细胞赋予一个”伪时间”（Pseudotime）坐标。这一思路的核心在于：细胞在分化过程中的转录变化是渐进的，即使我们无法追踪单个细胞随时间的变化，大量细胞的横断面数据依然包含了足够的轨迹信息。RNA Velocity 从另一个角度补充了这一方法：通过比较 pre-mRNA（未剪接，spliced）与成熟 mRNA（已剪接，unspliced）的丰度比值，推断每个基因的转录速率方向，从而为拟时序轨迹赋予真实的时间方向性。

{{Wiki:Cell_potency}}

核心概念：

  • 拟时序方法演进：DDRTree（Monocle 2）→ 学习图（Learned Graph, Monocle 3）→ 主曲线（Slingshot）→ 最优传输（WaddingtonOT）
  • Monocle 3 的学习图策略：UMAP 空间中的最小生成树；choose_graph_segments() 手动指定根节点；graph_test() 拟时序差异检验
  • Slingshot 的主曲线拟合：多谱系支持；与现有 UMAP 嵌入的兼容性
  • RNA Velocity 的 ODE 模型：\(\frac{du}{dt} = \alpha - \beta u\)；\(\frac{ds}{dt} = \beta u - \gamma s\)；\(u\)（未剪接）稳态假设
  • scVelo 的概率扩展：动力学模式（Dynamical model）vs 稳态模式（Steady-state model）；基因级别的动力学参数估计
  • 轨迹可视化：速率流场（velocity stream）叠加 UMAP 的方向箭头；scv.pl.velocity_embedding_stream()
  • 拟时序分析的局限：对高度异质性数据和多谱系分化的敏感性；树形假设对环形轨迹的不适用

实践内容：PBMC 3k 数据 Monocle 3 拟时序分析（以单核细胞分化轨迹为例）；graph_test() 识别拟时序差异基因；scVelo Dynamical model 计算 RNA Velocity；速率流场可视化；对比 Monocle 3 与 scVelo 方向推断的一致性。

41 模块 B：细胞通讯、调控与多组学

模块目标： 能运行 CellChat 细胞通讯分析并解读信号通路网络；了解 SCENIC 基因调控网络推断框架；理解 CITE-seq 和 10x Multiome 的技术原理；掌握 Seurat WNN 多模态聚类与 Signac scATAC-seq 跨模态整合注释。

41.1 专题 7.3：细胞通讯分析与基因调控网络推断

多细胞生物的正常生理功能依赖于细胞间持续的信号交流，而细胞内部的状态又由转录因子（Transcription Factor, TF）构成的调控网络驱动。本专题将这两个互补的”关系推断”问题合并讲解：细胞通讯分析（Cell-Cell Communication）从配体-受体表达推断细胞之间的信号传递，基因调控网络推断（Gene Regulatory Network, GRN）从共表达与 motif 证据推断基因之间的调控关系。CellChat 与 SCENIC 分别是这两类分析的代表性工具，二者结合可以回答”哪些外部信号驱动了某细胞类型的状态、又由哪些转录因子在细胞内执行”这一完整的调控链条问题。

核心概念：

  • 细胞通讯推断逻辑：若细胞类型 A 高表达配体、B 高表达对应受体，则推断 A→B 存在潜在通讯
  • 配体-受体数据库：CellChat 的 CellChatDB（含复合体受体）；CellPhoneDB 数据库（含亚单位拆分）
  • CellChat 通讯概率计算：质量作用定律（mass action law）；多亚单位受体复合体的几何均值处理；computeCommunProb()
  • 通路抽象与可视化：信号通路（Signaling Pathway）层面的汇总；netVisual_aggregate() 弦图；中心性指标（信号发送者 vs 接收者）
  • SCENIC 三步流程：GRNBoost2 共表达网络构建 → RcisTarget motif 富集过滤 → AUCell 调控子活性打分
  • Regulon（调控子）的定义：一个转录因子及其经过 motif 验证的靶基因集合；AUCell 的基因集富集 AUC 打分
  • GRNBoost2 vs GENIE3：梯度提升树 vs 随机森林的重要性评分；pySCENIC 在大规模数据上的效率优势
  • 两类分析的整合解读：通讯分析给出”细胞间信号输入”，GRN 给出”细胞内转录响应”，二者共同刻画调控因果链

实践内容：PBMC 3k 注释数据运行 CellChat，弦图可视化主要细胞类型间通讯强度并识别 MHC II 信号通路核心通讯对；演示 pySCENIC 完整流程（GRNBoost2 → RcisTarget → AUCell），热图可视化各细胞类型 Regulon 活性谱，识别 T 细胞亚群关键转录因子。

41.2 专题 7.4：单细胞多组学分析（CITE-seq 与 10x Multiome）

细胞的表型不仅体现在转录组层面：蛋白质表面标志物（如流式细胞术的 CD 抗原）和染色质可及性（顺式调控元件的活跃状态）都携带转录组无法提供的信息。单细胞多组学技术在同一细胞中同时测量多种模态。CITE-seq（Cellular Indexing of Transcriptomes and Epitopes by Sequencing）通过抗体偶联寡核苷酸（ADT）在测 mRNA 的同时测表面蛋白；10x Genomics Multiome 则从同一细胞核同时测量 RNA 与染色质可及性（scATAC）。Seurat 的加权最近邻（Weighted Nearest Neighbor, WNN）算法根据每个细胞各模态的信息量自适应赋权，实现真正意义上的多模态聚类；Signac 则提供从 peak 矩阵到 motif 活性的完整 scATAC 工具链。

{{Wiki:ATAC-seq}}

核心概念：

  • CITE-seq 文库结构：ADT 与 RNA 文库的共同条形码索引；ADT 的 CLR（Centered Log-Ratio）归一化
  • Seurat WNN 算法：模态特异性邻图权重估计；FindMultiModalNeighbors()；wnn.umap 嵌入
  • 多模态聚类的优势：用 ADT/ATAC 信号区分 RNA 相似但表型不同的亚群（如 CD4+ 与 CD8+ T 细胞）
  • scATAC-seq 原理与质控：Tn5 转座酶在开放染色质优先插入；peak×cell 矩阵极度稀疏；TSS 富集得分、核小体信号、FRiP
  • LSI（Latent Semantic Indexing）降维：TF-IDF 加权 → SVD；替代 PCA 处理 scATAC 稀疏矩阵
  • Gene Activity Score：将 peak 信号汇总到基因体+启动子，生成类基因表达矩阵用于初步注释
  • 跨模态整合（label transfer）：FindTransferAnchors() 以已注释 scRNA-seq 为参考向 scATAC 传递标签
  • ChromVAR motif 活性分析：计算每个细胞的 TF motif 可及性偏差；RunChromVAR()

实践内容：加载 cbmc（SeuratData，CITE-seq）构建 RNA 与 ADT 双模态并运行 WNN 聚类，对比单模态 vs WNN 对 T 细胞亚群的分辨能力；加载 pbmcMultiome（SeuratData，RNA+ATAC），Signac scATAC QC 与 LSI 降维、WNN 联合聚类、以 PBMC 3k 为参考做 label transfer、ChromVAR 分析 TF motif 可及性差异。

42 模块 C：空间转录组高级分析

模块目标： 掌握细胞解卷积的方法论原理，能运行 cell2location 将 scRNA-seq 参考图谱映射至 Visium 数据；理解 MERFISH 成像型空间转录组的技术原理与细胞分割流程；了解空间多模态整合的前沿方向。

42.1 专题 7.5：细胞解卷积与成像型空间转录组

空间转录组学存在一对根本性的权衡：基于测序的 Visium 平台覆盖全转录组但分辨率受限（每个 Spot 直径 55 µm，混合多个细胞），而基于成像的 MERFISH 平台达到亚细胞分辨率但只能检测有限的基因 panel。本专题分别针对这两条技术路线的核心计算问题：对于 Visium，细胞解卷积（Cell Type Deconvolution）利用 scRNA-seq 参考图谱将 Spot 的混合信号拆分为各细胞类型的丰度估计，从而在低分辨率数据上恢复细胞类型的空间分布；对于 MERFISH，关键步骤前移到 RNA 检测之前的图像处理与细胞分割。cell2location 与 RCTD 是主流解卷积工具，二者均以模块 A、B 产出的注释 scRNA-seq 图谱为参考。

{{Wiki:Fluorescence_in_situ_hybridization}}

核心概念：

  • 细胞解卷积模型分类：基于线性回归（RCTD）→ 基于概率生成模型（cell2location）→ 基于深度学习（Tangram）
  • cell2location 两阶段框架：第一阶段从 scRNA-seq 学习细胞类型签名矩阵（NB 回归）；第二阶段层次贝叶斯推断每个 Spot 的细胞类型丰度
  • 技术批次处理与参考选择：批次作为协变量（batch_key）；参考图谱的细胞类型覆盖度与每类细胞数量要求（≥ 50）
  • RCTD：基于泊松/负二项最大似然的稳健回归；doublet_mode（每 Spot 至多两种主要细胞类型）
  • 解卷积结果可视化与评估：细胞类型丰度空间热图、与 H&E 叠加；模拟 Spot 定量基准与空间共定位验证
  • MERFISH 编码策略：基于汉明码的多轮荧光标记与误差纠正（HD ≥ 4 编码）；ON/OFF 编码矩阵
  • 细胞分割（Cell Segmentation）：基于 DAPI/poly-T 的边界划定；Cellpose 深度学习分割；mRNA 点检测与空白条形码假阳性控制
  • 成像型 vs 测序型的互补：MERFISH（高分辨率、有限基因）与 Visium（低分辨率、全转录组）的信息互补；Vizgen MERSCOPE 与 10x Xenium 平台对比

实践内容：使用 stxBrain（Visium 小鼠脑）与注释后的 scRNA-seq 参考运行 cell2location/RCTD 解卷积，可视化主要脑区细胞类型（兴奋性神经元、抑制性神经元、星形胶质细胞）的空间丰度热图并与解剖图谱对照；加载 vizgen.brain（SeuratData，MERFISH 小鼠脑）演示细胞分割后的表达矩阵构建、label transfer 注释与亚细胞 RNA 分布可视化，对比同一脑区 MERFISH 与 Visium 数据的互补信息。
