17 从 QTL 到 GWAS：一场由标记密度引发的革命

专题 3.1 和 3.2 系统讲解了经典分子标记与区间作图的技术逻辑（参见专题 3.1、3.2）。那一套框架的核心局限已清晰：人工构建的分离群体（F2、RIL）中历史重组次数有限，导致 QTL 定位分辨率始终徘徊在 10–30 cM，无法从统计意义上锁定因果基因。

改变这一局面的关键，并不仅仅是换一个统计方法，而是两个根本性的转变：

  1. 群体：从人工构建的分离群体，转向自然群体（自然人群或动植物育种群体）。自然群体中，每对位置相邻的碱基之间，经过数百至数千代的减数分裂已积累了大量历史重组事件。这意味着，只有在极短的物理距离内（数十至数百 kb），遗传变异之间才能维持稳定的共分离关系——这一现象称为连锁不平衡（Linkage Disequilibrium，LD）。

  2. 标记：从数十至数百个 SSR，转向覆盖全基因组的数十万至数百万个单核苷酸多态性（Single Nucleotide Polymorphism，SNP）。SNP 是基因组中最丰富的遗传变异形式，每 1 kb 约有 1–3 个 SNP；同时，SNP 只有两个等位体，天然适合芯片杂交的信号读取。

上述两个转变的交汇点，正是全基因组关联分析（Genome-Wide Association Study，GWAS）。本专题的任务是在进入 GWAS 统计模型之前，系统理解支撑这一框架的两项技术基础：SNP 的高通量鉴定技术（SNP 阵列或全基因组测序）以及决定标记覆盖率与检验功效的LD 图谱。

本专题与前后专题的关系

本专题是专题 3.2（QTL 定位）与专题 3.4（GWAS 统计模型）之间的桥梁。前者揭示了 QTL 定位的分辨率局限，本专题解释为什么自然群体的大规模 SNP 数据可以突破这一局限，以及这些 SNP 数据如何从测序仪或基因芯片转化为标准 VCF 格式，供后续分析使用。

18 SNP：基因组最丰富的遗传多态形式

18.1 什么是 SNP

单核苷酸多态性（Single Nucleotide Polymorphism，SNP）是基因组中单个碱基位置上存在的序列差异，且该差异在人群中的次等位基因频率（Minor Allele Frequency，MAF）通常要求 ≥ 1%（低于此阈值称为罕见变异，rare variant）。

图3.3.1：单核苷酸多态性（SNP）示意图。参考基因组（上）与携带替代等位体的个体（下）在同一位置存在单碱基差异。图片来源：维基百科 Single-nucleotide polymorphism，作者 David Eccles (Gringer)，CC BY 4.0。

一个典型的 SNP 位点如下：

参考序列（Reference）:  ...A T G [C] G A T...
个体 1（纯合参考）:    ...A T G [C] G A T...
个体 2（杂合）：       ...A T G [C/T] G A T...
个体 3（纯合替代）：   ...A T G [T] G A T...

在双等位体 SNP 中，每个个体的基因型只有三种可能：参考纯合（0/0）、杂合（0/1）、替代纯合（1/1）。这一简洁性使 SNP 极为适合芯片检测和统计建模。

18.2 SNP 的丰度与分布

人类基因组中已知的 SNP 超过 6 亿个（dbSNP 数据库 Build 156），其中 MAF > 5% 的常见 SNP 约 1500 万个。SNP 的分布并非均匀：

  • 外显子区：SNP 密度相对较低（负选择压力清除有害变异）
  • 内含子和基因间区：SNP 密度较高，但功能意义需结合调控元件注释判断
  • 重复元件区（如 LINE、SINE）：序列高度保守，SNP 密度低
  • 着丝粒和端粒附近：分型技术难度大，SNP 图谱不完整

SNP 与点突变的区别

“点突变”（Point Mutation）泛指单碱基替换，通常指特定个体中发生的新生突变（de novo），可能频率极低（MAF < 0.1%）。SNP 是群体水平上经过筛选后仍具有一定频率的点突变，是群体遗传多态性的主要形式，两个概念在使用场景上有明确区分。

19 SNP 的高通量鉴定：两种主流技术路线

获取群体规模 SNP 基因型数据的途径主要有两种：SNP 阵列（Genotyping Array）和全基因组测序（Whole-Genome Sequencing，WGS）。两者在成本、覆盖范围和数据格式上各有不同，但最终均产生以 VCF（Variant Call Format）格式存储的基因型数据，作为后续 GWAS 分析的统一输入。

维度 SNP 阵列 全基因组测序（WGS）
成本（每样本，2025 年） 约 50–150 美元 约 150–300 美元（30× 覆盖度）
检测变异数量 50 万–200 万个预设 SNP 全基因组所有变异（>3000 万）
稀有变异覆盖 极差（MAF < 1% 几乎不涉及） 全面（含低频变异和新生突变）
结构变异 无 部分（InDel、小 CNV）

19.1 SNP 阵列技术简介

SNP 阵列通过固相杂交或单碱基延伸，在一张芯片上同时检测数十万至数百万个预先设计好的 SNP 位点。以 Illumina BeadChip 平台为例，每个 SNP 设计两条荧光探针分别对应两种等位体，激光扫描仪读取荧光强度后，软件将样品判读为三种基因型（AA/AB/BB）。判读结果经格式转换后以 PLINK binary 格式（.bed/.bim/.fam）或 VCF 格式存储，供后续分析使用。

图3.3.2：DNA 微阵列实验工作流程示意图，包括样品制备、芯片杂交和荧光信号读取三个主要阶段。图片来源：维基百科 DNA microarray，Public Domain。

SNP 阵列的历史贡献与现实局限

SNP 阵列在人类 GWAS 发展史上功不可没——UK Biobank 的 50 万样本、中国生物银行等大规模队列均依赖芯片完成初步基因分型。然而，芯片只能检测预设的位点，无法发现稀有变异，且多数非模式物种缺乏商业化芯片。随着 WGS 成本持续下降，越来越多的新研究直接采用 WGS 方案，WGS SNP calling 流程已成为生物信息学的必备技能。

19.2 全基因组测序的 SNP 鉴定流程

内容回顾

WGS SNP calling 的完整流程（数据下载 → BWA-MEM 比对 → MarkDuplicates → GATK HaplotypeCaller → VariantFiltration）已在专题 2.3 中系统讲解，本节不再重复。此处仅梳理流程的核心逻辑与关键质量指标，供本专题后续分析参考。

WGS 产生的 FASTQ 文件，经过比对（Alignment）→ 去重（Deduplication）→ 变异检测（Variant Calling）→ 过滤（Filtration）四个步骤，最终输出标准 VCF 文件（参见专题 2.3）。

数据集适用范围说明

专题 2.3 中用于演示 SNP calling 流程的规范数据集（E. coli K-12 MG1655，SRR2584863）是单个细菌样本，且细菌为单倍体生物。其 VCF 输出不含群体频率信息，也不存在二倍体杂合基因型，不适用于 GWAS 和 LD 分析。本专题后续的 LD 计算、插补和 GWAS 演示均使用人类群体数据（1000 Genomes Project chr22，2504 个体）。GWAS SNP calling 应针对多样本人类队列，采用 GATK 联合基因型分型（Joint Genotyping，-ERC GVCF 模式）产生的多样本 VCF。

19.3 VCF 格式详解

内容回顾

VCF 格式（CHROM / POS / REF / ALT / QUAL / FILTER / INFO / FORMAT 各列含义、GT/GQ/DP/AD 字段解析）已在专题 2.3 中完整介绍，本节不再展开。如需复习，请参见专题 2.3”VCF 格式详解”一节。

变异调用格式（Variant Call Format，VCF）是存储基因组变异信息的行业标准文本格式。对于 GWAS 分析，以下两点是在前序基础上需要特别关注的延伸：

  • 多样本 VCF：GWAS 数据中，每个样本占一列，FORMAT 字段中的 GT（基因型）、DP（深度）、GQ（基因型质量）是后续质控过滤的核心字段。
  • 群体频率字段：INFO 列中的 AF（Allele Frequency）在多样本 joint calling 后代表群体内的等位基因频率，直接用于 MAF 过滤；单样本 VCF 中该值无群体意义，不应用于 GWAS 质控。

20 HapMap 计划：绘制人类基因组 LD 图谱

SNP 芯片的出现解决了高通量基因分型的技术问题，但仍有一个关键问题悬而未决：究竟哪些 SNP 需要被检测？ 理论上，人类基因组中有约 1500 万个常见 SNP，要全部检测既无必要也无可能。回答这一问题，需要先理解 SNP 之间的连锁不平衡（LD）结构。

HapMap 计划（International HapMap Project）正是为此目的而设计的。该计划于 2002 年启动，分三个阶段对来自欧洲（CEU）、非洲（YRI）、日本（JPT）和中国汉族（CHB）共约 270 个个体进行全基因组 SNP 分型，系统描绘了人类基因组的单倍型（Haplotype）结构和 LD 图谱。

HapMap 计划的核心发现是：人类基因组并非”每个 SNP 都独立”，而是呈现出LD 区块（LD Block）结构——相邻 SNP 之间往往高度相关，整个区块内少数”标签 SNP”即可代表所有遗传变异。这一发现奠定了 GWAS 芯片设计的理论基础：只需检测足够多的标签 SNP，就能通过 LD 关系”覆盖”到未检测的其他位点。对于 WGS 数据，虽然已直接获得全部变异，LD 结构同样决定了关联信号的扩散范围和独立检验数估算（参见第 3.4.5 节）。

从 HapMap 到 1000 Genomes 计划

HapMap 计划的样本量和 SNP 密度在当时已属领先，但随着测序技术进步，其局限也日益显现：样本量偏小（约 270 人），低频变异（MAF 1%–5%）覆盖不足。1000 Genomes Project（2008–2015）对来自 26 个人群的 2504 个体进行了全基因组测序，将参考面板扩展到约 8800 万个变异位点，成为此后 GWAS 插补和精细定位的主要参考资源。目前最全面的人类基因组变异参考面板是 TOPMed（约 250 万个体）和 gnomAD，分辨率已接近全部常见变异的完整捕获。

21 连锁不平衡：定量描述遗传变异的共分离

21.1 LD 的概念与直觉

专题 2.5 引入了 LD 的核心直觉：基因组上相邻位点的等位体并非随机独立，“知道了一个位点的等位体，就能在一定程度上推断附近位点的等位体”（参见专题 2.5”连锁不平衡：基因组的区块结构”）。这一直觉对应的数学含义是：两个位点等位体的观测共现频率，系统性地偏离了它们各自频率乘积所预测的随机期望值——偏离越大，LD 越强。

专题 2.5 还指出，LD 的强弱可以用 \(r^2\) 系数衡量，但未展开其计算方式。本节将从 \(r^2\) 的数学根基出发，说明它是如何从原始的等位体频率推导而来，以及为何能直接对应 GWAS 的统计功效。

形式化定义：设位点 A 有等位体 \(A_1\)（频率 \(p_A\)）和 \(A_2\)（频率 \(1-p_A\)），位点 B 有等位体 \(B_1\)（频率 \(p_B\)）和 \(B_2\)（频率 \(1-p_B\)）。若两位点完全独立，双位点单倍型 \(A_1B_1\) 的期望频率应为 \(p_A \cdot p_B\)；实际观测频率记为 \(f_{A_1B_1}\)，两者之差定义为D 值（连锁不平衡系数）：

\[D = f_{A_1B_1} - p_A \cdot p_B\]

\(D = 0\) 对应两位点完全独立（LD 不存在）；\(D \neq 0\) 则表明两位点的单倍型频率受共同遗传历史影响，偏离随机期望。同一条染色体上物理距离越近的位点，历史重组将其打散的概率越低，\(|D|\) 的期望值也越大。

21.2 r² 与 D’：两种主要的 LD 量化指标

D 值的绝对大小受等位体频率影响，不同位点间难以直接比较。因此实践中更多使用两个标准化指标：

21.2.1 r²（相关系数平方）

\[r^2 = \frac{D^2}{p_A(1-p_A) \cdot p_B(1-p_B)}\]

\(r^2\) 的取值范围为 \([0, 1]\)，具有直接的统计学含义： - \(r^2 = 1\)：两位点完全等价，知道一个即知道另一个；GWAS 中对某位点的检测功效可以 100% 传递到与之 \(r^2 = 1\) 的其他位点 - \(r^2 = 0\)：两位点完全独立，无 LD - 在 GWAS 中，若某 SNP 与未检测的因果变异的 \(r^2 = 0.8\)，则 GWAS 在该 SNP 处的统计功效约为检测因果变异本身的 80%

21.2.2 D’（标准化 D 值）

\[D' = \frac{D}{D_{\max}}\]

其中 \(D_{\max}\) 是在当前等位体频率约束下 \(D\) 的最大可能值。\(|D'|\) 的取值范围为 \([0, 1]\)： - \(|D'| = 1\)：位点间的四种可能单倍型组合（\(A_1B_1\), \(A_1B_2\), \(A_2B_1\), \(A_2B_2\)）中至多出现三种（称为”不可分解”或历史上只发生过一次重组事件） - \(|D'| < 1\)：四种单倍型均存在，说明两位点间已发生过至少一次重组

21.2.3 两个指标的使用场景

指标 优势 局限 主要用途
\(r^2\) 直接对应统计功效传递 低频 SNP 间 \(r^2\) 估计偏差大 GWAS 功效分析、标签 SNP 选取
\(D'\) 对重组历史更敏感 受样本量和等位体频率影响大，小样本时偏高 单倍型结构推断

小样本时 D’ 的系统性高估

当样本量较小（< 100 个体）时，\(D'\) 值趋向于 1，即便两个位点实际上并不处于高 LD 状态。这是因为小样本中四种单倍型组合往往不能全部被观测到。因此，在小样本研究中，\(r^2\) 是更可靠的 LD 度量。

21.3 R 代码：手工计算 LD 指标

以下代码演示从原始基因型数据出发，计算 \(r^2\) 和 \(D'\) 的完整流程，以增强对 LD 指标的直觉理解。

# R: manual calculation of LD statistics (r2 and D-prime)
# Input: two SNP loci genotyped in a sample of individuals
# Genotype coding: 0 = ref/ref, 1 = ref/alt, 2 = alt/alt

compute_ld <- function(snp_a, snp_b) {
  # Remove individuals with missing data at either locus
  valid <- !is.na(snp_a) & !is.na(snp_b)
  snp_a <- snp_a[valid]
  snp_b <- snp_b[valid]
  n <- length(snp_a)

  # ---- Allele frequencies ----
  # Additive coding: dosage = number of alt alleles (0/1/2)
  p_a <- sum(snp_a) / (2 * n)
  p_b <- sum(snp_b) / (2 * n)

  # ---- r-squared ----
  # cor(genotype_A, genotype_B)^2 equals haplotype-level r2 under HWE.
  # The naive method-of-moments estimator (0.5 * n_double_het) omits EM
  # iteration and severely underestimates r2 when LD is high.
  r2 <- cor(snp_a, snp_b)^2

  # ---- D statistic (recovered from r2 and allele frequencies) ----
  cov_sign <- sign(cov(snp_a, snp_b))
  D <- cov_sign * sqrt(r2 * p_a * (1 - p_a) * p_b * (1 - p_b))

  # ---- D-prime ----
  if (D >= 0) {
    D_max <- min(p_a * (1 - p_b), (1 - p_a) * p_b)
  } else {
    D_max <- min(p_a * p_b, (1 - p_a) * (1 - p_b))
  }
  D_prime <- D / D_max

  list(
    p_a = p_a, p_b = p_b,
    D = D, r2 = round(r2, 4), D_prime = round(abs(D_prime), 4)
  )
}

# ---- Example: two SNPs in high LD ----
# Simulate at the HAPLOTYPE level: each individual carries two haplotypes.
# Copying the allele from A to B on the same haplotype with low recombination
# creates haplotype-level LD.  Copying genotype dosages directly does NOT,
# because a heterozygous genotype (1) still carries two independent alleles.
set.seed(42)
n   <- 500
maf <- 0.35   # minor allele frequency

hap1_a <- rbinom(n, 1, maf)   # haplotype 1, locus A (0 or 1)
hap2_a <- rbinom(n, 1, maf)   # haplotype 2, locus A

# 2% recombination between A and B -> expected r2 ≈ (1 - 2*0.02)^2 ≈ 0.92
recomb_rate <- 0.02
hap1_b <- ifelse(runif(n) < recomb_rate, rbinom(n, 1, maf), hap1_a)
hap2_b <- ifelse(runif(n) < recomb_rate, rbinom(n, 1, maf), hap2_a)

snp_a <- hap1_a + hap2_a   # diploid additive dosage (0/1/2)
snp_b <- hap1_b + hap2_b

result_high_ld <- compute_ld(snp_a, snp_b)
cat("=== High LD Pair ===\n")
cat(sprintf("p(A_alt) = %.3f  p(B_alt) = %.3f\n",
            result_high_ld$p_a, result_high_ld$p_b))
cat(sprintf("D = %.4f  r2 = %.4f  D' = %.4f\n\n",
            result_high_ld$D, result_high_ld$r2, result_high_ld$D_prime))

# ---- Example: two SNPs in low LD (independent) ----
snp_c <- sample(c(0, 1, 2), n, replace = TRUE, prob = c(0.36, 0.48, 0.16))
snp_d <- sample(c(0, 1, 2), n, replace = TRUE, prob = c(0.25, 0.50, 0.25))

result_low_ld <- compute_ld(snp_c, snp_d)
cat("=== Low LD Pair (near-independent) ===\n")
cat(sprintf("p(C_alt) = %.3f  p(D_alt) = %.3f\n",
            result_low_ld$p_a, result_low_ld$p_b))
cat(sprintf("D = %.4f  r2 = %.4f  D' = %.4f\n",
            result_low_ld$D, result_low_ld$r2, result_low_ld$D_prime))

22 LD 衰减：距离越远，相关性越低

22.1 LD 衰减的成因

在自然群体中，两个位点间的 LD 水平随着物理距离的增加而下降，这一现象称为 LD 衰减（LD Decay）。其成因在于减数分裂中的重组：位点间的物理距离越大，历经多个世代后在两者之间发生重组的累积概率越高，原始的单倍型组合逐渐被打散，等位体频率趋向随机组合。

图3.3.3：连锁不平衡热图示例（GAM 方法）。色阶以黑色表示 \(r = 0\)（两位点独立），暖色（红白）趋向 \(r = +1\)，冷色（蓝白）趋向 \(r = -1\)；颜色越偏离黑色，位点间 LD 越强。注意：该图展示的是相关系数 \(r\) 而非 \(r^2\)，\(r\) 可为负值（等位体呈反向关联）。图片来源：维基百科 Linkage disequilibrium，作者 Negativeexistential，CC0 1.0。

LD 随距离的衰减速率在不同人群和物种中差异显著：

物种 / 人群 LD 衰减至 r² < 0.1 的距离 成因
非洲人群（YRI） ~20–50 kb 更古老的奠基群体，历史重组次数更多
欧洲人群（CEU） ~100–200 kb 较年轻的奠基者瓶颈效应降低了 LD 衰减速率
亚洲人群（CHB/JPT） ~100–150 kb 与欧洲人群类似
玉米（自交系间） ~2 kb 极高的遗传多样性，重组历史极丰富
大豆 ~150 kb 栽培大豆遗传基础窄，LD 延伸较远
小麦 > 1 Mb 强烈的驯化瓶颈和近亲繁殖

这一差异对 GWAS 设计有直接影响：非洲人群的 LD 衰减快，芯片研究需要更高密度的 SNP 才能实现相同的基因组覆盖率；而对于 WGS 研究，LD 衰减速率则决定了候选区域精细定位的分辨率——LD 衰减越快，可信集（Credible Set）越小，因果变异的定位越精确（参见专题 3.6）。

23 LD 区块与标签 SNP

23.1 LD 区块的概念

基因组并非以均匀的速率衰减 LD，而是呈现出区块化（Block-like）结构：某些区域内 SNP 间的 LD 极高（区块内），而相邻区块之间则存在重组热点，LD 急剧下降（区块间）。这种结构在 HapMap 数据中首次被系统描绘，并形成了”单倍型区块（Haplotype Block）“的概念。

在一个 LD 区块内，常见单倍型的数量通常远少于理论上的最大值（\(2^n\)，\(n\) 为 SNP 数量）。例如，一个含有 10 个 SNP 的区块，理论上有 \(2^{10} = 1024\) 种单倍型组合，但实际上往往只有 3–6 种在群体中频率较高的单倍型（major haplotypes）覆盖了 90% 以上的个体。

23.2 标签 SNP 的选取

标签 SNP（Tag SNP）是能够代表 LD 区块内全部变异信息的最小 SNP 集合。判定标准通常是：一个 Tag SNP 与区块内每一个未被直接检测的 SNP 的 \(r^2 \geq 0.8\)（即统计功效至少为直接检测的 80%）。

Haploview 软件是最常用的 LD 区块可视化与 Tag SNP 选取工具，其算法（Gabriel et al., 2002）基于 \(D'\) 的置信区间将基因组划分为 LD 区块。PLINK 的 --indep-pairwise 命令则提供了一种基于 \(r^2\) 阈值的 LD 修剪（LD Pruning）方法，常用于 GWAS 数据质控和 PCA 分析前的 SNP 筛选。

24 基因型插补：以 LD 为杠杆，扩展标记密度

24.1 插补的必要性与适用场景

商业 SNP 芯片通常检测 50 万–200 万个预设 SNP，而人类基因组中常见 SNP 约有 1500 万个。芯片未直接覆盖的 SNP，若与某个生物学效应有关，GWAS 可能就此错过。基因型插补（Genotype Imputation）利用 LD 结构和大型参考面板（如 1000 Genomes、HRC、TOPMed），统计推断芯片未直接检测位点的基因型剂量（Dosage）。插补后，可分析位点数从数十万扩展至数千万，覆盖几乎所有常见变异，大幅提升精细定位的分辨率。

WGS 数据无需插补

插补是芯片数据专有的补救策略。WGS 研究直接通过 SNP calling（如 GATK HaplotypeCaller）获得全部变异的真实基因型，无需推断。因此，随着 WGS 普及，插补的使用场景将逐渐收窄至：（1）利用已有大规模芯片队列数据（如 UK Biobank 原始芯片批次）；（2）在 WGS 数据补充前的过渡阶段；（3）与历史芯片队列进行元分析时统一变异集合。

24.2 插补的基本原理

插补依赖单倍型参考面板和隐马尔可夫模型（Hidden Markov Model，HMM）。以下是核心逻辑：

概念示意（3 个个体，5 个 SNP 位点）：

参考面板（WGS 测序，所有位点均已知）：
  单倍型 1:  A - C - G - T - A
  单倍型 2:  T - C - A - T - G
  单倍型 3:  A - T - G - C - A

目标个体（芯片数据，仅检测到 SNP 1、3、5）：
  芯片输出：   A - ? - G - ? - A

插补逻辑：
  1. HMM 在参考面板中寻找与目标个体在已知位点（1、3、5）最匹配的单倍型
  2. 单倍型 1（A-G-A）与目标个体完全吻合
  3. 推断：SNP 2 的基因型为 C，SNP 4 的基因型为 T
  4. 输出为基因型剂量（Dosage），插补质量指标 Rsq 反映该推断的置信度

实践中，插补质量的关键指标是 \(R^2\)（Imputation Accuracy），定义为插补基因型剂量与真实基因型的平方相关系数。通常要求 \(R^2 > 0.3\)（宽松）或 \(R^2 > 0.8\)（严格）的 SNP 才进入后续 GWAS 分析。

理解 LD 图谱之后，有必要明确 LD 在 GWAS 全流程中的具体作用——这将为后续专题的统计模型学习奠定直觉基础。

25.1 影响一：决定 GWAS 的有效基因组覆盖率

对于芯片研究，某个检测到的 SNP 能够”代表”多大范围内的变异，完全取决于其周围的 LD 衰减速率——LD 衰减越慢，标签 SNP 密度要求越低。若 LD 衰减慢（如小麦），200k SNP 芯片即可覆盖几乎全基因组；若 LD 衰减快（如非洲人群或玉米），则需要数百万 SNP 才能实现相同覆盖率。对于WGS 研究，LD 结构决定的不是覆盖率（WGS 已覆盖全部变异），而是关联信号在物理空间上的扩散宽度：LD 延伸越远，同一因果变异将导致更大范围的 SNP 出现显著信号，精细定位的分辨率相应降低。

25.2 影响二：产生”LD 拖尾”（LD Tailing）

当 GWAS 在某位点发现显著关联时，该位点附近所有与其处于高 LD 的 SNP 也会同时显著——在曼哈顿图上形成一个由高 LD 连接的”峰”（而非单点）。这是真实关联的特征，而非假阳性：整个峰代表一个 LD 区块，其中因果变异可能是峰顶 SNP，也可能是任何一个与之高 LD 的其他 SNP。从峰到因果变异的鉴定，是精细定位（Fine Mapping）的任务（参见专题 3.6）。

25.3 影响三：导致多重检验负担并非”百万次独立检验”

GWAS 对基因组上数百万个 SNP 同时进行关联检验，但由于 SNP 间存在 LD（非独立），真正独立的检验次数远少于 SNP 总数。人类全基因组显著性阈值 \(p < 5\times10^{-8}\) 正是基于”约 100 万个独立检验”（考虑 LD 后的有效独立 SNP 数）的 Bonferroni 校正，而非严格按照实际 SNP 总数（约 500 万–1000 万个常见 SNP）计算。这一阈值的推导逻辑将在专题 3.4 中详细展开。

# R: demonstrate LD inflation effect on association statistics
# Shows how LD causes neighboring SNPs to show inflated significance
# even when only one causal variant exists

simulate_gwas_ld <- function(n = 3000, n_snps = 30, causal_idx = 15,
                               effect_size = 0.5, ld_decay_rate = 0.3) {
  set.seed(42)

  # Simulate correlated SNPs using multivariate normal + threshold
  # LD between SNPs decays exponentially with distance
  snp_positions <- seq(0, 100, length.out = n_snps)  # in kb
  sigma_mat <- outer(snp_positions, snp_positions,
                     function(i, j) exp(-ld_decay_rate * abs(i - j)))

  library(MASS)
  latent <- mvrnorm(n, mu = rep(0, n_snps), Sigma = sigma_mat)
  # Convert to genotype dosage (0/1/2) via thresholding
  maf <- 0.3
  geno_mat <- (latent > qnorm(1 - maf))  +
              (latent > qnorm(1 - maf^2 - 2*maf*(1-maf)))  # rough diploid approx
  # Simpler: just use quantized continuous latent
  geno_mat <- apply(latent, 2, function(x) {
    q <- quantile(x, probs = c((1-maf)^2, 1 - maf^2))
    ifelse(x < q[1], 0, ifelse(x > q[2], 2, 1))
  })

  # Simulate phenotype driven by causal SNP only
  pheno <- geno_mat[, causal_idx] * effect_size + rnorm(n, 0, 1)

  # Run association tests (linear regression) for each SNP
  p_values <- sapply(1:n_snps, function(j) {
    fit <- lm(pheno ~ geno_mat[, j])
    coef(summary(fit))[2, 4]  # p-value for SNP coefficient
  })

  data.frame(
    snp     = paste0("SNP", 1:n_snps),
    pos_kb  = snp_positions,
    neg_log_p = -log10(p_values),
    is_causal = 1:n_snps == causal_idx
  )
}

results <- simulate_gwas_ld()

library(ggplot2)
ggplot(results, aes(x = pos_kb, y = neg_log_p, color = is_causal)) +
  geom_point(size = 3) +
  scale_color_manual(values = c("FALSE" = "steelblue", "TRUE" = "firebrick"),
                     labels = c("FALSE" = "Non-causal (LD tag)", "TRUE" = "Causal SNP")) +
  geom_hline(yintercept = -log10(5e-8), linetype = "dashed",

26.3 步骤三：计算成对 \(r^2\) 矩阵

# ---- 3. Compute pairwise LD (r2) using snpStats ----
ld_result <- ld(geno_region, stats = "R.squared", depth = n_selected - 1)
r2_matrix <- as.matrix(ld_result)

# Fill lower triangle (ld() returns upper triangle only)
r2_full <- r2_matrix
r2_full[lower.tri(r2_full)] <- t(r2_matrix)[lower.tri(r2_matrix)]
diag(r2_full) <- 1

snpStats::ld() 是计算成对 LD 的核心函数。参数说明：

  • stats = "R.squared"：计算 \(r^2\)（线性相关系数的平方），而非 \(D'\)。\(r^2\) 与 GWAS 功效传递比例存在直接对应关系（代理 SNP 对目标位点的功效传递效率约为 \(r^2\)），是功效分析和标签 SNP 选取的标准度量。
  • depth = n_selected - 1：计算每个 SNP 与所有其他 SNP 的 LD，即全成对矩阵（depth 控制计算半带宽，设为最大值时等于全矩阵）。对于较大的窗口（> 1000 个 SNP），可适当降低 depth（如 500）以节省计算时间，牺牲远距离 SNP 对的 \(r^2\) 信息。

ld() 出于存储效率只返回上三角矩阵（即 SNP_A 索引 < SNP_B 索引的 \(r^2\) 值），对角线为 NA。后续三行代码将上三角对称填充到下三角，并将对角线设为 1（每个 SNP 与自身完全相关），使矩阵成为完整的对称正定矩阵，方便后续热图绘制。

计算复杂度与内存

\(N\) 个 SNP 的全成对 \(r^2\) 矩阵有 \(N(N-1)/2\) 个元素。本例 \(N \approx 200\)，矩阵约 2 万个元素，计算和存储均无压力。若 \(N = 10000\)（约 5 kb 间距覆盖 50 Mb 区域），则矩阵含 5000 万个 double 值，占用约 400 MB 内存，此时应考虑分块计算或仅保存 \(r^2 > 0.1\) 的稀疏表示。

26.5 步骤五：LD 衰减分析

# ---- 5. LD decay within the selected region ----
positions <- bim$position[bim$snp.name %in% region_snps]
r2_long[, dist_kb := abs(positions[idx_a] - positions[idx_b]) / 1000]
r2_long[, dist_bin := cut(dist_kb,
                           breaks = c(0, 5, 10, 20, 50, 100, 200, 500),
                           labels = c("0-5", "5-10", "10-20",
                                      "20-50", "50-100", "100-200", "200-500"))]

decay_summary <- r2_long[dist_kb > 0,
                           .(mean_r2 = mean(r2, na.rm = TRUE)),
                           by = dist_bin]
cat("\n=== LD Decay within chr22:20-20.5 Mb ===\n")
print(decay_summary)

LD 衰减分析将每对 SNP 的 \(r^2\) 按物理距离分组，计算各距离区间内 \(r^2\) 的均值，从而得到 LD 衰减曲线（LD decay curve） 的离散近似。具体步骤如下：

  1. positions 从 .bim 文件中提取目标区域各 SNP 的物理坐标（单位：碱基对）。
  2. dist_kb 计算每对 SNP 的物理间距（单位：kb），排除 dist_kb == 0（SNP 与自身的距离）。
  3. cut() 按非等宽的距离区间分箱：短距离（0–20 kb）区间较密，因为这是 LD 变化最剧烈的区段，需要更精细的分辨率；长距离（50–500 kb）区间较宽，因为此时 \(r^2\) 已趋于平稳。
  4. data.table 的 by = dist_bin 分组计算，高效地在每个距离区间内取均值。

在 1000 Genomes 欧洲人群（CEU）中，预期的衰减模式为：0–5 kb 区间均值 \(r^2 \approx 0.6\)–0.8，10–20 kb 区间降至 \(\approx 0.3\)–0.5，100–200 kb 区间降至 \(\approx 0.05\)–0.1，并在更长距离趋于背景 LD 水平（\(\approx 0.02\)–0.05，主要来自低频 SNP 对的随机 LD）。若衰减速率明显偏快（如欧洲人群中 20 kb 处均值 \(r^2 < 0.1\)），可能提示 QC 过滤过于严格导致 SNP 密度不足；若衰减速率明显偏慢（100 kb 处均值 \(r^2 > 0.3\)），则需检查样本是否存在群体分层或亲缘关系未被过滤，这两者均会人为地维持远距离 LD。

黄金法则：LD 衰减曲线的群体间比较价值

对多个参考群体分别运行上述流程，将衰减曲线叠加比较，可以直观验证群体遗传学的核心结论：非洲人群（YRI）有效群体大（\(N_e \approx 10{,}000\)–15,000），LD 在 10–20 kb 就衰减至背景水平；欧洲/亚洲人群（\(N_e \approx 2{,}000\)–5,000）LD 可维持至 50–200 kb；而高度驯化的作物品系或近交系（\(N_e \approx 100\)–500）LD 可延伸至 Mb 尺度。这一对比是理解跨群体 GWAS 芯片设计差异的直观基础。

26.6 完整流程总结

上述五步构成了 LD 图谱分析的标准操作路径，各步骤的输入-输出关系如下：

PLINK 二进制文件 (.bed/.bim/.fam)
        │
        ▼  read.plink()

        │
        ▼  坐标筛选

        │

在实际 GWAS 研究中，上述分析通常在以下两个场景中重复使用：第一，研究开始前的数据质控阶段，通过 LD 衰减曲线验证样本群体来源与预期一致；第二，关联峰精细分析阶段，对显著信号区间绘制局部 LD 热图，判断哪些 SNP 处于同一 LD 区块中（共同代表同一因果变异），哪些 SNP 已跨越区块边界（需单独检验）。

  2. \(D'\) 的系统性高估来源于其定义：\(D' = D / D_{\max}\)，在小样本中四种单倍型组合往往不能全部被观测到，导致 \(D_{\max}\) 被高估，\(|D'|\) 趋向 1。\(r^2\) 直接反映两位点基因型向量的线性相关程度，统计性质更稳健，且与 GWAS 功效传递效率存在直接数学对应（功效传递比例 \(\approx r^2\)）。\(D'\) 适合用于单倍型结构推断和历史重组事件推断，在大样本、较高 MAF 的情景下可靠；\(r^2\) 适合 GWAS 功效分析和标签 SNP 选取。

  4. (a) “单点峰”是可疑信号，更可能是假阳性。真实关联信号由 LD 连接的一簇 SNP 共同显著，形成”峰形”；孤立的单点提示基因型分型错误（低 Call Rate、基因型聚类异常）或批次效应。(b) 验证步骤：① 检查该 SNP 的基因型质量（HWE、缺失率）；② 分析周围区域的 LD 结构；③ 在独立群体中重复验证；④ 若条件允许进行该区域局部重测序。(c) 是的。WGS 数据覆盖了周围所有变异，若为真实信号，高 LD 伙伴 SNP 将同时显著形成 LD 拖尾；芯片覆盖密度不足时，周围代表性 SNP 可能缺失，造成假性单点峰。
