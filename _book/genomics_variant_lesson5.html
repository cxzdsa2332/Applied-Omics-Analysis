20 专题 3.6：精细定位、功能注释与因果推断

20.1 学习目标

完成本专题学习后，学生应能够：

  1. 阐释 GWAS 精细定位（fine-mapping）的必要性及其与 LD 裁剪的本质区别；
  2. 理解贝叶斯精细定位方法（SuSiE、FINEMAP）的核心统计框架，解释可信集（Credible Set）的定义与生物学意义；
  3. 描述候选位点功能注释的标准流程：变异分类（ANNOVAR/VEP）、eQTL 查询（GTEx）、共定位分析（coloc）与表观基因组注释（ENCODE/Roadmap）；
  4. 描述孟德尔随机化（Mendelian Randomization，MR）的因果推断框架与三大工具变量假设，比较 IVW、MR-Egger、加权中位数方法的适用条件；
  5. 区分多基因风险评分（PRS）与单位点 GWAS 的应用场景，比较 C+T 与 LDpred2 方法；
  6. 解释多群体 GWAS 的科学意义与多群体精细定位的 LD 互补原理；
  7. 概述稀有变异聚合检验（SKAT、SKAT-O）的动机与统计策略；
  8. 阐释功能作图（Functional Mapping）的核心思想——将生物数学模型嵌入 QTL 似然函数，以动态曲线参数的基因型间差异量化遗传效应，理解 Logistic 均值模型与 AR(1) 协方差结构的数学意义及其相对于逐时间点 GWAS 的检验力优势。

20.2 精细定位的必要性

GWAS 中的显著信号通常对应一个 LD 区块，该区块内可能含有数十至数百个相互高度连锁的 SNP，其中只有少数（甚至仅一个）是真正的致因变异（causal variant）。精细定位（fine-mapping）的目标是在统计层面缩小因果候选变异的范围，为后续功能实验提供优先靶标。

精细定位 vs. LD 裁剪

LD 裁剪（clumping）识别的是统计独立的信号位点，目的是消除冗余；精细定位识别的是在已知信号区域内最可能为因果变异的集合，目的是缩小候选范围。二者互补，缺一不可。

{{Wiki:Linkage_disequilibrium}}

20.3 贝叶斯精细定位

20.3.1 统计框架

贝叶斯精细定位方法将每个 SNP 视为候选因果变异，计算其后验因果概率（Posterior Inclusion Probability，PIP）：

\[ \text{PIP}_j = P(\text{SNP } j \text{ 为因果变异} \mid \text{数据}) \]

早期方法（如 FINEMAP）假设候选区域内因果变异数量服从先验分布，通过马尔可夫链蒙特卡罗（MCMC）或 shotgun stochastic search 枚举可能的因果变异组合。

SuSiE（Sum of Single Effects） 方法采用变分贝叶斯（variational Bayes）推断，将多因果变异问题分解为多个”单效应”成分之和：

\[ \mathbf{y} = \sum_{l=1}^{L} \mathbf{X} \mathbf{b}_l + \boldsymbol{\varepsilon} \]

其中每个 \(\mathbf{b}_l\) 仅有一个非零元素（单效应假设），\(L\) 为预设的最大因果变异数。SuSiE 通过迭代贝叶斯岭回归（iterative Bayesian stepwise selection，IBSS）算法求解各成分的后验分布。

20.3.2 可信集（Credible Set）

对于每个效应成分 \(l\)，SuSiE 输出一个可信集（Credible Set）：包含所有 SNP 中，累积后验概率达到指定覆盖率（通常 95%）的最小变异子集。即：

\[ \sum_{j \in \mathcal{CS}_l} \text{PIP}_{lj} \geq 0.95 \]

可信集越小，精细定位的分辨率越高。若某个可信集仅含 1–2 个 SNP，则这些变异是高置信度的因果候选变异。

20.3.3 手动实现：单因果变异假设下的 PIP 计算

在介绍 SuSiE 之前，先从最简单的场景出发——假设候选区域内恰好只有一个因果变异——用纯 R 手动推导每个 SNP 的 PIP。这一推导去掉了所有工具包的封装，直接展示贝叶斯精细定位的核心计算逻辑。

统计模型：在单因果变异假设下，候选区域含 \(J\) 个 SNP，零假设 \(H_0\) 为无因果变异，备择假设 \(H_j\) 为 SNP \(j\) 是因果变异。给定 GWAS 汇总统计量（Z 分数 \(z_j\)）和 LD 矩阵 \(\mathbf{R}\)，各假设下 Z 分数向量的近似边际似然为：

\[ \log P(\mathbf{z} \mid H_j) \propto \frac{W}{1 + W} \cdot \frac{z_j^2}{2} \]

其中 \(W = n \sigma_0^2 / \sigma_e^2\) 为效应量先验方差与误差方差之比（即 Bayes Factor 的缩放因子，通常取 \(W = 1\)）。对每个 SNP 计算边际似然后，在等先验（\(P(H_j) = 1/(J+1)\)）下应用贝叶斯定理即得 PIP：

\[ \text{PIP}_j = \frac{P(\mathbf{z} \mid H_j)}{\sum_{k=0}^{J} P(\mathbf{z} \mid H_k)} \]

以下代码完整实现这一过程，并与 susieR 的输出进行对照：

# Manual Bayesian fine-mapping under the single-causal-variant assumption
# All calculations use only base R — no external packages required.
set.seed(2024)

# --- Step 1: Define the simulated region ---
# 150 SNPs; two "true" causal variants at positions 60 and 110.
# We deliberately set up two signals to show that single-CV assumption
# will partially recover both, but cannot fully separate them.
n_snp  <- 150
n_samp <- 30000
causal_pos <- c(60, 110)
beta_true  <- c(0.08, -0.06)   # effect sizes in SD units

# --- Step 2: Simulate an LD matrix with decaying correlation ---
# Use an AR(1) structure with rho = 0.85 within two blocks (1–80, 81–150)
# and near-zero correlation between blocks.
rho_in  <- 0.85
rho_out <- 0.05
R <- matrix(0, n_snp, n_snp)
for (i in seq_len(n_snp)) {
  for (j in seq_len(n_snp)) {
    same_block <- (i <= 80 & j <= 80) | (i > 80 & j > 80)
    rho        <- if (same_block) rho_in else rho_out
    R[i, j]   <- rho^abs(i - j)
  }
}
diag(R) <- 1.0

# --- Step 3: Generate Z-scores consistent with the LD structure ---
# Under the linear model y = X*beta + e, the marginal Z-score for SNP j is:
#   z_j ~ N( sqrt(n) * (R %*% beta)[j] / sigma_e,  1 )
# For simplicity set sigma_e = 1 and MAF = 0.25 for all SNPs.
maf     <- 0.25
# Expected signal: each causal SNP contributes to Z of all SNPs via LD
signal_vec          <- rep(0, n_snp)
signal_vec[causal_pos] <- beta_true
ncp_vec             <- sqrt(n_samp * 2 * maf * (1 - maf)) * (R %*% signal_vec)
z_scores            <- ncp_vec + rnorm(n_snp, sd = 1)

# --- Step 4: Compute approximate Bayes Factors (ABFs) for each SNP ---
# ABF_j = exp( W/(1+W) * z_j^2 / 2 ) / sqrt(1 + W)
# where W is the prior variance on effect size (in units of 1/n).
# We use W = 1 (corresponds to prior sd ~ 0.1 on standardised scale).
W   <- 1.0
log_abf <- (W / (1 + W)) * z_scores^2 / 2 - 0.5 * log(1 + W)

# Null model log-likelihood (z_j ~ N(0,1) everywhere): 0 by convention.
log_null <- 0

# --- Step 5: Convert ABFs to PIPs via Bayes theorem ---
# Assume equal prior probability for each hypothesis H_j (SNP j is causal)
# and H_0 (no causal variant).  Prior odds: 1/(J+1) for each.
# Posterior: normalise over all J+1 hypotheses on the log scale.
all_log_bf      <- c(log_null, log_abf)    # length J+1: H_0 first, then H_1..H_J
log_sum_bf      <- log(sum(exp(all_log_bf - max(all_log_bf)))) + max(all_log_bf)
posterior_probs <- exp(all_log_bf - log_sum_bf)

pip_manual <- posterior_probs[-1]   # drop H_0; keep PIP for each SNP

# --- Step 6: Build a 95% credible set (greedy, sorted by PIP) ---
order_by_pip   <- order(pip_manual, decreasing = TRUE)
cumulative_pip <- cumsum(pip_manual[order_by_pip])
cs_size        <- which(cumulative_pip >= 0.95)[1]
credible_set   <- sort(order_by_pip[seq_len(cs_size)])

cat(sprintf("P(H0: no causal variant) = %.4f\n", posterior_probs[1]))
cat(sprintf("Top-5 SNPs by PIP:\n"))
top5 <- order(pip_manual, decreasing = TRUE)[1:5]
for (i in top5) {
  flag <- if (i %in% causal_pos) "  <-- true causal" else ""
  cat(sprintf("  SNP %3d: PIP = %.4f, Z = %+.2f%s\n",
              i, pip_manual[i], z_scores[i], flag))
}
cat(sprintf("95%% credible set (%d SNPs): %s\n",
            cs_size, paste(credible_set, collapse = ", ")))

# --- Step 7: Visualise the PIP profile ---
library(ggplot2)
df_pip <- data.frame(
  pos    = seq_len(n_snp),
  pip    = pip_manual,
  z_abs  = abs(z_scores),
  causal = seq_len(n_snp) %in% causal_pos,
  in_cs  = seq_len(n_snp) %in% credible_set
)

ggplot(df_pip, aes(x = pos, y = pip)) +
  geom_col(aes(fill = in_cs), width = 0.9) +
  geom_point(data = subset(df_pip, causal),
             aes(y = pip + 0.03), shape = 25, size = 3,
             fill = "#D62728", color = "#D62728") +
  scale_fill_manual(values = c("FALSE" = "grey70", "TRUE" = "steelblue"),
                    labels = c("Outside CS", "In 95% CS"),
                    name = NULL) +
  geom_hline(yintercept = 0.5, linetype = "dashed",
             color = "grey30", linewidth = 0.5) +
  labs(x = "SNP position in region",
       y = "Posterior Inclusion Probability (PIP)",
       title = "Manual Bayesian Fine-mapping (Single Causal Variant Assumption)",
       subtitle = sprintf("True causal SNPs: %s | 95%% CS size: %d SNPs",
                          paste(causal_pos, collapse = " & "), cs_size)) +
  theme_classic(base_size = 12) +
  theme(legend.position = "top")

运行结果揭示了单因果变异假设的核心局限：两个真实信号（位置 60 和 110）都分别吸引了一些 PIP，但由于模型强制只有一个因果变异，95% 可信集必然同时将两个信号区域的 SNP 纳入，导致集合偏大。这正是多效应方法（SuSiE）的动机所在——它允许 \(L > 1\) 个独立效应，能分别为两个信号构建各自的可信集。

近似贝叶斯因子（Approximate Bayes Factor，ABF）的来源

上述 ABF 公式由 Wakefield（2009）推导：假设效应量在 \(H_j\) 下服从 \(N(0, \sigma_0^2)\) 先验，在 GWAS 汇总统计量框架下积分掉效应量后，得到仅依赖 Z 分数和先验方差的闭合解。\(W = n\sigma_0^2/\sigma_e^2\) 越大，先验越弥散，ABF 对 Z 分数的响应越平缓；\(W\) 越小，先验越集中于零，ABF 对弱信号更保守。实践中通常取 \(W \in \{0.5, 1, 2\}\) 后对 ABF 做平均（BMA），以降低结论对 \(W\) 选择的敏感性。

20.3.4 SuSiE R 实现

library(susieR)
library(tidyverse)

# 准备输入数据
# z: z-score 向量（GWAS 统计量）
# R: LD 矩阵（候选区域内所有 SNP 的 r 相关矩阵）
# n: 样本量

# 示例：从 GWAS summary stats 和参考 LD 矩阵运行 SuSiE
# （实际应用中 LD 矩阵由参考面板如 1000 Genomes 计算）
set.seed(42)
n_snp <- 200
n_samp <- 50000

# Simulate z-scores with one causal variant at position 100
causal_pos <- 100
z_scores <- rnorm(n_snp, mean = 0, sd = 1)
z_scores[causal_pos] <- 8.0  # strong signal at causal variant

# Simulate LD matrix (block structure)
R <- diag(n_snp)
for (i in 1:n_snp) {
  for (j in i:n_snp) {
    r_ij <- exp(-0.02 * abs(i - j))
    R[i, j] <- R[j, i] <- r_ij
  }
}

# Run SuSiE with summary statistics
fit_susie <- susie_rss(
  z    = z_scores,
  R    = R,
  n    = n_samp,
  L    = 5,          # max number of causal variants
  coverage = 0.95
)

# Extract results
cat("Number of credible sets:", length(fit_susie$sets$cs), "\n")

# PIP for each SNP
pip_df <- data.frame(
  snp_idx = 1:n_snp,
  pip     = fit_susie$pip
)

# Visualize PIP
ggplot(pip_df, aes(x = snp_idx, y = pip)) +
  geom_col(fill = "steelblue", width = 0.8) +
  geom_hline(yintercept = 0.9, color = "red", linetype = "dashed") +
  labs(x = "SNP Index",
       y = "Posterior Inclusion Probability (PIP)",
       title = "SuSiE Fine-mapping Result") +
  theme_classic()

# Print credible set SNPs
for (l in seq_along(fit_susie$sets$cs)) {
  cat(sprintf("Credible set %d: SNPs %s (coverage = %.3f)\n",
              l,
              paste(fit_susie$sets$cs[[l]], collapse = ", "),
              fit_susie$sets$coverage[l]))
}

SuSiE 的实际应用注意事项

精细定位的质量高度依赖于 LD 矩阵的准确性。应优先使用与 GWAS 样本来源相同的群体作为参考面板计算 LD；若使用公共参考面板（如 1000 Genomes EUR），需确认与研究样本的祖源高度匹配。样本量不足时，SuSiE 可信集可能包含大量 SNP，精细定位分辨率有限。

20.4 候选位点的功能注释

精细定位缩小了因果变异的统计候选范围，但统计概率本身无法说明变异通过何种分子机制影响表型。功能注释（functional annotation）是将统计候选变异与基因组功能信息整合、提出生物学假说的关键步骤。GWAS 发现的显著位点中约 90% 位于非编码区，其机制多涉及顺式调控（cis-regulation），需借助多层次基因组注释加以解析。

{{Wiki:Regulatory_element}}

20.4.1 变异功能分类：VEP

VEP（Variant Effect Predictor，Ensembl）是目前最广泛使用的变异功能注释工具，可对每个候选 SNP 及其 LD 伙伴变异（\(r^2 > 0.8\)）进行多数据库联合注释，输出标准化的序列本体论（Sequence Ontology）功能分类：

注释类型 含义 功能解读重要性
非同义变异（missense_variant） 导致氨基酸替换 高——直接改变蛋白序列

20.4.3 共定位分析（Colocalization）

eQTL 与 GWAS 信号的区域重叠不足以说明共享因果变异——两个信号可能来自同一 LD 区块内两个紧密连锁但不同的变异。共定位分析（colocalization） 通过贝叶斯框架量化两个性状的关联信号共享同一因果变异的后验概率。coloc 包计算五种假设的后验概率（PPH0–PPH4）：

假设 含义
H0 两性状均无信号
H1 仅 GWAS 有信号
H2 仅 eQTL 有信号
H3 两者有信号，但各由不同变异驱动
H4 两者共享同一因果变异

PPH4 > 0.8 是共定位的常用阈值，意味着有超过 80% 的后验概率支持 GWAS 与 eQTL 信号由同一遗传变异驱动，为该基因提供了强烈的致因基因证据。

H3（各由独立变异驱动）不代表”无关”——该情形提示同一 LD 区块内存在两个独立调控元件，分别影响基因表达和疾病风险，同样具有重要的生物学意义。SuSiE-coloc（coloc::coloc.susie()）在各数据集上先运行 SuSiE 精细定位识别多信号，再对每对信号分别做共定位，避免多信号区域中信号混叠导致的误判，是现代分析的推荐流程。

GWAS 的统计命中仅是起点，而非终点。候选基因的确立需要多组学证据的汇聚：精细定位（统计）→ 功能注释（基因组）→ eQTL/共定位（分子）→ 实验验证（细胞/动物）。任何单一证据都不足以建立因果结论。

20.5 多群体 GWAS 与跨祖源分析

{{Wiki:Population_stratification}}

20.5.1 单群体主导的历史局限

过去二十年间，已发表 GWAS 中超过 80% 的样本来自欧洲祖源人群。这一不均衡带来多重局限：基于欧洲人群构建的 PRS 在其他群体中预测性能显著下降；某些在特定群体中频率较高的致病变异可能被忽略；跨群体效应量比较受阻。UK Biobank 非白人子集、BBJ（日本生物样本库）、All of Us 研究计划等近年来显著改善了这一状况。

20.5.2 多群体分析策略

多群体 GWAS 主要有两种实施模式：

  1. 分群体分析后元分析：各祖源群体内独立运行 GWAS，对汇总统计量做跨群体元分析。METAL 的 IVW 模式（逆方差加权）为首选，自动对齐效应等位基因方向；对于 A/T 和 C/G 回文 SNP（palindromic SNPs），需依赖等位基因频率辅助链方向判断，或直接排除以避免方向反转错误。

  2. 联合分析（mega-analysis）：合并原始个体水平数据，统一模型中使用多祖源 PCA 或稀疏多祖源 GRM 校正群体结构。统计效率更高，但对数据共享协议要求严格。

多群体精细定位利用不同祖源群体 LD 结构的互补性缩小可信集。欧洲人群中 \(r^2 = 0.95\) 的两个 SNP 在东亚人群中可能 \(r^2 = 0.20\)，东亚样本的数据即可区分二者，将 PIP 集中在真正的因果变异上。SuSiE 多群体扩展版和 PAINTOR 均支持多群体联合精细定位，其多数据集似然为：

\[P(\text{all data} \mid \mathcal{C}) = \prod_{a=1}^{A} P(\text{data}_a \mid \mathcal{C}, \mathbf{R}_a)\]

20.5.3 跨群体异质性与健康公平性

Cochran’s Q 检验和 \(I^2\) 统计量量化各群体间效应量的异质性。高异质性（\(I^2 > 75\%\)）提示存在基因-环境交互或群体特异性 LD 代理效率差异，此时应报告随机效应元分析结果并通过亚组森林图展示各群体方向一致性。

PRS 跨群体迁移性差的根本原因在于 LD 代理关系的群体特异性：训练群体中的 lead SNP 通过 LD 代理因果变异，当 LD 结构不同时代理效率下降，效应量估计偏差。PRS-CSx 等多群体方法在贝叶斯框架下为每个群体各自估计收缩效应量，再线性加权组合，显著提升非欧洲群体预测性能，是推动基因组医学公平性的重要技术方向。

祖源描述的规范用语

应使用”欧洲祖源（European ancestry）“、”东亚祖源（East Asian ancestry）“等基于遗传相似性的描述，而非社会文化种族标签。遗传祖源是多维主成分空间中的连续分布，而非离散分类。研究报告应明确说明祖源分组的定义方法（通常基于与 1000 Genomes 参考超群的 PCA 比对），以保证可重复性。

20.6 孟德尔随机化（Mendelian Randomization）

20.6.1 因果推断的挑战

观察性流行病学研究中，暴露因素（如 BMI、LDL 胆固醇、炎症因子）与疾病的关联可能受到混杂因素的干扰，或受到反向因果（reverse causation）的影响，难以直接推断因果关系。

孟德尔随机化（Mendelian Randomization，MR） 借助遗传变异作为暴露因素的工具变量（instrumental variable，IV），利用遗传变异随机分配（类似随机对照试验）的特性推断因果效应。

{{Wiki:Mendelian_randomization}}

20.6.2 经典案例：HDL-C 与冠心病——一个改变药物研发方向的因果推断故事

在过去数十年间，流行病学研究持续观察到一个稳健的负相关：血浆高密度脂蛋白胆固醇（HDL-C）水平越高，冠心病（coronary heart disease，CHD）风险越低。这一关联在数十项大型队列研究中一再得到证实，效应量之大、重复性之强，令研究者几乎不加怀疑地将其解读为”HDL-C 对心脏的保护作用”。弗雷明汉心脏研究（Framingham Heart Study）等经典数据将 HDL-C 确立为心血管风险评估的核心指标，“好胆固醇”的通俗名称也由此深入人心。

这一观念在 2000 年代驱动了制药业数十亿美元的投入：若能通过药物提升 HDL-C，理应降低心脏病发作风险。胆固醇酯转运蛋白（cholesterol ester transfer protein，CETP）是 HDL 代谢的关键调控酶，抑制 CETP 可大幅提升 HDL-C 水平。辉瑞公司研发的 torcetrapib 在临床前和早期试验中表现亮眼，HDL-C 提升幅度高达 70%。然而，2006 年 ILLUMINATE 三期随机对照试验中期分析显示，torcetrapib 组患者的心血管事件和全因死亡率反而显著升高，试验被紧急叫停。此后，罗氏的 dalcetrapib、礼来的 evacetrapib 相继在大型三期试验中失败，全面宣告了”升高 HDL-C 即可降低心脏病风险”这一假说在药物干预层面的破产。

观察性证据为何失灵？ 其根本原因在于两种系统性偏倚的叠加：其一，混杂因素（confounding）——健康生活方式（运动、戒烟、合理饮食）同时提升 HDL-C 并降低 CHD 风险，真正的保护效应来自生活方式而非 HDL-C 本身；其二，反向因果（reverse causation）——慢性炎症和早期动脉粥样硬化既可降低 HDL-C，又是 CHD 的先兆，导致低 HDL-C 表面上”预测”了疾病，但方向颠倒。这两种机制在观察性数据中无法通过统计调整彻底去除。

孟德尔随机化的介入。 2012 年，Voight 等人在 The Lancet 上发表了迄今最具影响力的 HDL-C MR 研究。他们从 GWAS 中筛选出数十个强烈影响 HDL-C 水平但对 LDL-C 和甘油三酯影响极小的遗传位点作为工具变量，在超过 10 万名个体的汇总数据中分析遗传预测的 HDL-C 水平与心肌梗死风险的关系。结论清晰且令人震惊：遗传预测的 HDL-C 升高与心肌梗死风险无显著关联（OR 接近 1.0），与观察性估计完全背离。与此形成鲜明对比的是，遗传预测的 LDL-C 升高与 CHD 风险呈现强烈且一致的正相关——与他汀类药物的临床试验结果高度吻合。

观察性关联 vs. 因果效应：HDL-C 的教训

证据类型 HDL-C ↑ → CHD ↓ LDL-C ↑ → CHD ↑
观察性流行病学 强烈负相关（混杂驱动） 强烈正相关
孟德尔随机化 无因果效应 因果效应显著
随机对照试验（药物干预） CETP 抑制剂全线失败 他汀显著降低事件率

三类证据在 LDL-C 上高度一致；在 HDL-C 上，MR 与 RCT 共同推翻了观察性假说。这正是 MR 作为”自然界的随机对照试验”（nature’s randomized controlled trial）的核心价值——在无法进行伦理上可行的干预试验时，用遗传随机化替代实验随机化，区分真实因果与混杂相关。

这一案例深刻揭示了因果推断的核心难题，也奠定了 MR 在遗传流行病学中的方法论地位。自此，MR 分析成为评估生物标志物治疗靶点潜力的标准工具之一，在药物研发的靶点验证（target validation）阶段被广泛采用。

20.6.3 三个核心假设

有效工具变量必须满足三个假设：

  1. 相关性（Relevance）：遗传变异与暴露因素显著关联（通常要求 F 统计量 > 10）；
  2. 独立性（Independence）：遗传变异与混杂因素无关（随机化假设，由减数分裂的随机性保证）；
  3. 排他性（Exclusion restriction）：遗传变异仅通过暴露因素影响结局，不存在水平多效性（horizontal pleiotropy）。
遗传变异 (G) ──────────► 暴露 (X) ──────────► 结局 (Y)
     │                                              ▲
     │               混杂因素 (U)                   │
     │─────────────────────────────────────────────┘
                    (假设此路径不存在)

20.6.4 手动实现：从一个具体问题出发理解 MR 估计量

20.6.4.1 教学场景：BMI 升高会导致收缩压升高吗？

观察性研究中 BMI 与收缩压（SBP）之间存在稳健的正相关，但两者均受生活方式、社会经济因素等大量混杂因素影响。我们的问题是：BMI 的升高本身（因果地）是否会使收缩压升高？

以 FTO、MC4R 等基因区域的 GWAS 显著 SNP 作为工具变量——这些 SNP 影响 BMI，但理论上不直接影响血压（仅通过 BMI 这一中间路径），满足工具变量的三个核心假设。

G_j (体重相关 SNP)
     │
     │ γ_j（SNP→BMI 效应）
     ↓
    BMI (X)  ──── β_XY（因果效应？）────►  收缩压 (Y)
     ↑                                         ↑
     └──────── 混杂因素 U（饮食、运动）─────────┘
               （遗传变异绕过了 U 这条路）

MR 的核心思路是：如果 SNP \(j\) 使 BMI 平均升高 \(\hat{\gamma}_j\)，同时使收缩压升高 \(\hat{\Gamma}_j\)，那么用两者之比 \(\hat{\Gamma}_j / \hat{\gamma}_j\) 就可以估计 BMI 每升高一单位对收缩压的因果效应——Wald 比值（Wald ratio）。

20.6.4.2 第一讲：用三个 SNP 手算 Wald 比值

为了让计算完全透明，先从只有三个工具变量的极简情形出发：

SNP \(\hat{\gamma}_j\)（BMI 效应，kg/m²/等位基因） \(\hat{\Gamma}_j\)（SBP 效应，mmHg/等位基因） Wald 比值 \(\hat{\Gamma}_j / \hat{\gamma}_j\)
rs1421085 (FTO) 0.30 0.42 1.40
rs17782313 (MC4R) 0.20 0.26 1.30
rs12748679 (TMEM18) 0.15 0.22 1.47

每个 Wald 比值都是对同一因果效应 \(\beta_{XY}\) 的独立估计（单位：mmHg 每 kg/m² BMI）。三个值分别为 1.40、1.30、1.47，彼此接近，提示结果一致性好。若某个 SNP 的 Wald 比值与其他 SNP 严重偏离，则该 SNP 很可能存在水平多效性——它通过 BMI 以外的路径影响了血压。

# ---- Toy example: Wald ratios for 3 SNPs ----
# All numbers are illustrative; no external packages needed.
snp_names <- c("rs1421085 (FTO)", "rs17782313 (MC4R)", "rs12748679 (TMEM18)")

gamma_hat <- c(0.30, 0.20, 0.15)   # SNP -> BMI effects (kg/m² per allele)
Gamma_hat <- c(0.42, 0.26, 0.22)   # SNP -> SBP effects (mmHg per allele)
se_Gamma  <- c(0.06, 0.05, 0.07)   # standard errors of SNP -> SBP effects

wald_ratio <- Gamma_hat / gamma_hat
se_wald    <- se_Gamma / abs(gamma_hat)   # delta-method SE

cat("── Wald Ratio Estimates ──────────────────────────────\n")
for (i in seq_along(snp_names)) {
  cat(sprintf("  %-26s  ratio = %+.3f  SE = %.3f  95%%CI [%.3f, %.3f]\n",
              snp_names[i],
              wald_ratio[i],
              se_wald[i],
              wald_ratio[i] - 1.96 * se_wald[i],
              wald_ratio[i] + 1.96 * se_wald[i]))
}

预期输出：

── Wald Ratio Estimates ──────────────────────────────
  rs1421085 (FTO)            ratio = +1.400  SE = 0.200  95%CI [1.008, 1.792]
  rs17782313 (MC4R)          ratio = +1.300  SE = 0.250  95%CI [0.810, 1.790]
  rs12748679 (TMEM18)        ratio = +1.467  SE = 0.467  95%CI [0.552, 2.381]

三个 SNP 的估计均指向同一方向（BMI 每升高 1 kg/m²，SBP 约升高 1.3–1.5 mmHg），置信区间彼此高度重叠，一致性良好。现实分析中通常使用数十乃至数百个 SNP，此时需要将多个 Wald 比值合并为单一估计，三种主流方法各有侧重。

为什么不能直接回归 SBP ~ BMI？

设想用同一批人的 BMI 和 SBP 做普通线性回归。爱运动的人既有更低的 BMI，又有更低的 SBP——这个混杂因素 U（运动习惯）会使 BMI 系数被高估（或低估）。遗传变异天然地”随机化”了 BMI：携带高 BMI 等位基因的个体，其运动习惯与不携带者平均而言没有差异（因为等位基因在减数分裂时随机分配）。因此，遗传预测的 BMI 与 SBP 之间的关系，正是剔除了 U 之后的净因果效应。

20.6.4.3 第二讲：三种估计量的直觉与全流程模拟

现在用完整模拟（20 个 SNP，含 4 个多效性 SNP）演示三种估计量。继续沿用 BMI → SBP 场景：真实因果效应设为 \(\beta_{XY} = 1.30\)（mmHg / kg/m²），4 个”坏”SNP 存在水平多效性（直接影响血压，不经过 BMI），用于检验各方法的稳健性。

# Full MR simulation: BMI (X) -> systolic blood pressure (Y)
# Units: X = BMI (kg/m²), Y = SBP (mmHg), beta_xy = 1.30 mmHg per kg/m²
library(ggplot2)
set.seed(2025)

# ── Parameters ──────────────────────────────────────────────────────────────
n_iv    <- 20       # number of SNPs used as instruments
n_samp  <- 10000    # simulated sample size
beta_xy <- 1.30     # true causal effect: 1 kg/m² BMI -> 1.30 mmHg SBP
n_pleio <- 4        # number of invalid SNPs (horizontal pleiotropy)
pleio_sd <- 0.40    # SD of pleiotropic direct SNP -> SBP effects (mmHg)

# Each SNP's effect on BMI (gamma_j, kg/m² per risk allele)
gamma_j <- abs(rnorm(n_iv, mean = 0.25, sd = 0.08))

# Pleiotropic direct effects on SBP (alpha_j = 0 for valid IVs)
alpha_j  <- rep(0, n_iv)
pleio_idx <- sample(n_iv, n_pleio)
alpha_j[pleio_idx] <- rnorm(n_pleio, mean = 0, sd = pleio_sd)

数据生成模型完整地描述了每个人的观测值是怎样产生的。对每个 SNP \(j\)，独立模拟 10,000 名个体：

  • 基因型 \(G \in \{0,1,2\}\)（加性编码，MAF 随机）；
  • 未测量混杂因素 \(U\)（如运动习惯、饮食模式）；
  • 暴露 \(X = \gamma_j G + 0.4U + \varepsilon_X\)（BMI）；
  • 结局 \(Y = \beta_{XY} X + \alpha_j G + 0.4U + \varepsilon_Y\)（SBP）。

注意 \(Y\) 的方程中 \(\alpha_j G\) 一项：对有效 IV，\(\alpha_j = 0\)，SNP 只通过 BMI 影响 SBP；对多效性 SNP，\(\alpha_j \neq 0\)，SNP 还有一条绕过 BMI 的直接通路。

# ── Simulate individual data and extract summary statistics ──────────────────
# gamma_vec / alpha_vec: full vectors passed via MoreArgs; index by j inside.
# unname() strips the "G" attribute from coef(), keeping column names clean.
simulate_one_snp <- function(j, n, gamma_vec, alpha_vec, beta_xy) {
  maf <- runif(1, 0.10, 0.45)
  G   <- rbinom(n, 2, maf)                                    # genotype (0/1/2)
  U   <- rnorm(n)                                             # unmeasured confounder
  X   <- gamma_vec[j] * G + 0.4 * U + rnorm(n, sd = 2.5)    # BMI (kg/m²)
  Y   <- beta_xy * X + alpha_vec[j] * G + 0.4 * U +
         rnorm(n, sd = 12)                                    # SBP (mmHg)

  fit_x <- lm(X ~ G);  fit_y <- lm(Y ~ G)

  c(beta_X = unname(coef(fit_x)["G"]),
    se_X   = unname(summary(fit_x)$coefficients["G", "Std. Error"]),
    beta_Y = unname(coef(fit_y)["G"]),
    se_Y   = unname(summary(fit_y)$coefficients["G", "Std. Error"]))
}

ss <- as.data.frame(t(mapply(simulate_one_snp, seq_len(n_iv),
                             MoreArgs = list(n         = n_samp,
                                            gamma_vec = gamma_j,
                                            alpha_vec = alpha_j,
                                            beta_xy   = beta_xy))))

# Wald ratio and its delta-method SE for each SNP
ss$wald_ratio <- ss$beta_Y / ss$beta_X
ss$se_wald    <- ss$se_Y / abs(ss$beta_X)
ss$F_stat     <- (ss$beta_X / ss$se_X)^2
ss$is_pleio   <- seq_len(n_iv) %in% pleio_idx

cat(sprintf("Median F-statistic: %.1f  (all > 10: %s)\n",
            median(ss$F_stat),
            ifelse(all(ss$F_stat > 10), "YES", "NO")))

方法一：IVW（逆方差加权）——将所有 Wald 比值做加权平均，精度更高（SE 更小）的 SNP 权重更大。等价于以 \(1/\hat{\sigma}^2_{Y_j}\) 为权重、过原点的加权线性回归斜率。“过原点”意味着：IVW 假设所有 SNP 均有效（\(\alpha_j = 0\)），不允许系统性多效性存在。

# ── Method 1: IVW ────────────────────────────────────────────────────────────
# Intuition: precision-weighted average of all Wald ratios.
# The origin constraint assumes every SNP satisfies the exclusion restriction.
w_ivw    <- 1 / ss$se_wald^2
beta_ivw <- sum(w_ivw * ss$wald_ratio) / sum(w_ivw)
se_ivw   <- sqrt(1 / sum(w_ivw))
p_ivw    <- 2 * pnorm(-abs(beta_ivw / se_ivw))

cat(sprintf("[IVW]            beta = %+.3f mmHg/unit BMI  SE = %.3f  p = %.2e\n",
            beta_ivw, se_ivw, p_ivw))

方法二：MR-Egger——将”过原点”约束放开，允许回归直线有自由截距。截距捕捉的是：各 SNP 的多效性效应与该 SNP 工具变量强度是否存在系统性关联（方向性多效性）。若截距显著不为零，说明”坏”SNP 的多效性不是随机分布的，IVW 的估计会有偏，而 Egger 的斜率则在 InSIDE 假设成立时给出校正后的因果效应估计。代价是：自由截距消耗了一个自由度，估计的方差更大。

# ── Method 2: MR-Egger ───────────────────────────────────────────────────────
# Intuition: free the origin constraint -> the intercept absorbs average
# directional pleiotropy; the slope estimates the causal effect after correction.
w_egger        <- 1 / ss$se_Y^2
fit_egger      <- lm(beta_Y ~ beta_X, data = ss, weights = w_egger)
beta_egger     <- coef(fit_egger)["beta_X"]
se_egger       <- summary(fit_egger)$coefficients["beta_X", "Std. Error"]
intercept_egger <- coef(fit_egger)["(Intercept)"]
p_intercept    <- summary(fit_egger)$coefficients["(Intercept)", "Pr(>|t|)"]

cat(sprintf("[MR-Egger]       beta = %+.3f mmHg/unit BMI  SE = %.3f\n",
            beta_egger, se_egger))
cat(sprintf("  Intercept = %+.3f  p = %.3f  %s\n",
            intercept_egger, p_intercept,
            ifelse(p_intercept < 0.05,
                   "<-- directional pleiotropy detected!", "")))

方法三：加权中位数——将所有 Wald 比值从小到大排列，找到累积权重恰好达到 50% 的那个值。这个”中间人”不受任何一侧极端值的拉偏。关键性质：只要有效工具变量贡献了总权重的一半以上，估计量就保持一致性——即便剩余一半的 IV 全部无效。标准误无法用 delta 法推导，改用自助法（bootstrap）模拟抽样分布。

# ── Method 3: Weighted median ────────────────────────────────────────────────
# Intuition: sort all Wald ratios by value; find the 50% cumulative-weight
# quantile. Robust as long as the majority (by weight) of IVs are valid.
weighted_median_mr <- function(ratios, weights) {
  w      <- weights / sum(weights)          # normalise to sum = 1
  ord    <- order(ratios)
  cum_w  <- cumsum(w[ord])
  mid    <- which(cum_w >= 0.5)[1]
  # edge case: cumulative weight hits exactly 0.5
  if (cum_w[mid] == 0.5 && mid < length(ratios))
    return(mean(ratios[ord[mid + 0:1]]))
  ratios[ord[mid]]
}

set.seed(42)
wm_boot <- replicate(1000, {
  idx <- sample(n_iv, replace = TRUE)
  weighted_median_mr(ss$wald_ratio[idx], 1 / ss$se_wald[idx]^2)
})

beta_wm <- weighted_median_mr(ss$wald_ratio, 1 / ss$se_wald^2)
se_wm   <- sd(wm_boot)
p_wm    <- 2 * pnorm(-abs(beta_wm / se_wm))

cat(sprintf("[Weighted median] beta = %+.3f mmHg/unit BMI  SE = %.3f  p = %.2e\n",
            beta_wm, se_wm, p_wm))
cat(sprintf("\n[True causal effect] beta_XY = %.2f mmHg per kg/m² BMI\n", beta_xy))

可视化：MR 散点图——横轴为每个 SNP 对 BMI 的效应（\(\hat{\gamma}_j\)），纵轴为对 SBP 的效应（\(\hat{\Gamma}_j\)）。若 MR 假设成立，所有点应沿过原点、斜率为 \(\beta_{XY}\) 的直线分布。多效性 SNP（红色）会偏离这条直线，拉动 IVW 直线（蓝色实线）向自身倾斜；MR-Egger（红色虚线）通过自由截距对此做了校正；加权中位数（橙色点划线）则忽略了少数偏离点。

ss$snp_type <- ifelse(ss$is_pleio, "Invalid IV (pleiotropic)", "Valid IV")

IVW 的等价形式

IVW 估计量等价于：以 \(1/\hat{\sigma}_{Yj}^2\) 为权重，对过原点约束的加权线性回归 \(\hat{\Gamma}_j \sim \hat{\gamma}_j\) 求斜率。这一等价性揭示了 IVW 的核心假设：残差误差完全来自抽样误差，排除系统性水平多效性——即假设 \(\alpha_j = 0\) 对所有 \(j\) 成立（排他性假设严格成立）。

20.7 多基因风险评分（PRS）

20.7.1 基本原理

单个 GWAS 显著位点的效应量通常很小（优势比 OR 多在 1.05–1.20 之间），无法单独用于个体风险预测。然而，复杂性状的遗传结构往往是高度多基因性（highly polygenic）的——数千乃至数万个散布于全基因组的低效应位点共同贡献了相当大比例的遗传力。多基因风险评分（Polygenic Risk Score，PRS） 正是将这些位点的微小效应线性叠加，将个体的全基因组遗传信息压缩为一个具有连续分布的风险分数：

\[ \text{PRS}_i = \sum_{j=1}^{M} \hat{\beta}_j \cdot G_{ij} \]

其中 \(\hat{\beta}_j\) 为第 \(j\) 个 SNP 在 GWAS 训练集（base dataset）中估计的效应量（通常为 log OR 或线性回归系数），\(G_{ij}\) 为个体 \(i\) 在该 SNP 的加性基因型编码（0/1/2，按效应等位基因计数）。PRS 的本质是对个体基因组中所有风险等位基因的加权求和，权重由人群水平的 GWAS 汇总统计量提供。

PRS 的多基因性假设

PRS 框架隐含两个基本假设：（1）各位点效应在统计上近似可加——不考虑上位性（epistasis）和显性偏差（dominance deviation）；（2）效应量估计可从训练群体迁移至目标个体。前者在大多数复杂性状中是合理的近似，后者则是 PRS 跨群体泛化问题的核心挑战（参见本章多群体 GWAS 部分）。

20.7.2 PRS 的统计框架：从遗传力到预测准确性

理解 PRS 的理论上限，需从SNP 遗传力（SNP heritability，\(h^2_\text{SNP}\)）出发。\(h^2_\text{SNP}\) 是所有常见 SNP 共同解释的表型方差比例，代表理想条件下 PRS 可解释的最大方差。现实中，PRS 的实际预测准确性（\(R^2\) 或 AUC）受以下因素制约：

  1. 训练样本量：效应量估计误差随 \(n\) 减小而增大，PRS 的 \(R^2\) 近似正比于 \(n \cdot (h^2_{\text{SNP}})^2 / \bigl(n \cdot h^2_{\text{SNP}} + M_{\text{eff}}\bigr)\)，其中 \(M_{\text{eff}}\) 为有效独立 SNP 数目；
  2. SNP 数目与效应量截断：纳入的 SNP 越多，捕捉的多基因信号越完整，但同时也引入更多噪声；
  3. LD 校正的质量：连锁 SNP 之间效应量高度相关，若不处理 LD 会导致重复计数，高估具有大量代理变异的位点的贡献；
  4. 群体匹配程度：训练集与目标集的祖源差异越大，LD 代理关系越不一致，预测性能下降越明显。

不同性状的当前 PRS 预测能力差异很大：身高的 PRS \(R^2\) 可达 0.25–0.35（接近家系遗传力 0.80 的 30–40%）；2 型糖尿病的 AUC 约为 0.60–0.65；精神分裂症的 PRS 在病例-对照设计中 Nagelkerke \(R^2\) 约为 0.07–0.10。这些差异反映了各性状的多基因结构、GWAS 训练集规模以及环境因素比重的综合作用。

20.7.3 构建流程：数据分工与过拟合防范

PRS 分析涉及三个必须严格分离的数据集：

Base dataset (GWAS 汇总统计量)
   │  提供效应量 β̂_j（通常来自大规模 GWAS 联盟）
   │
   ▼
Target dataset（目标基因型数据）
   ├── Validation set（验证集）：调参——选最优 p 值阈值或超参数
   └── Test set（测试集）：报告最终预测性能

最常见的 PRS 过拟合错误

将目标数据集（target dataset）直接用于调参（如选择最优 p 值截断阈值），再在同一数据上报告预测性能，会产生严重的乐观偏倚。正确做法是将目标数据严格划分为独立的验证集（调参）和测试集（评估），或使用交叉验证。若目标样本量不足以分割，则仅使用 base GWAS 的超参数默认值，不在目标数据上调参。

20.7.4 LD 裁剪 + p 值阈值（C+T）方法

C+T（Clumping + Thresholding）是最直观的 PRS 构建方法，分为两步：首先通过 LD 裁剪去除相互连锁的 SNP（保留每个 LD 区块中关联最强的代表 SNP），再按 p 值阈值筛选纳入 PRS 计算的 SNP 集合。C+T 无需参考 LD 矩阵进行效应量校正，计算速度快，是快速原型验证的首选方案。其主要缺陷是 p 值阈值的选择依赖验证集，且丢弃了阈值以下的大量微效位点信息。

20.7.5 贝叶斯收缩方法：LDpred2 与 PRS-CS

C+T 方法在统计上是次优的，原因有二：第一，它用硬截断丢弃了大量具有信息量的微效位点；第二，它用 LD 裁剪粗暴地选择代表 SNP，而未对效应量进行 LD 感知的校正。贝叶斯收缩方法从理论上解决了这两个问题。

核心思想。 真实 GWAS 效应量估计 \(\hat{\beta}_j\) 包含信号与噪声两部分，以及由 LD 引起的”效应膨胀”（neighbouring SNPs in LD inflate each other’s estimates）。贝叶斯方法对真实效应量 \(\beta_j\) 施加先验分布，通过最大化后验概率得到收缩估计 \(\tilde{\beta}_j\)，再用 \(\tilde{\beta}_j\) 替代原始 \(\hat{\beta}_j\) 构建 PRS：

\[ \text{PRS}^{\text{Bayes}}_i = \sum_{j=1}^{M} \tilde{\beta}_j \cdot G_{ij} \]

LDpred2 假设效应量服从点-正态混合先验（spike-and-slab）：以概率 \(p\) 为因果变异（效应来自 \(N(0, h^2_\text{SNP} / (M \cdot p))\)），以概率 \(1-p\) 效应为零。其自动版本（ldpred2-auto）可从数据中联合估计 \(h^2_\text{SNP}\) 和因果变异比例 \(p\)，无需外部参数调优，是目前应用最广泛的贝叶斯 PRS 方法之一。

PRS-CS 采用连续收缩先验（continuous shrinkage prior，基于马蹄先验的变体），全局收缩参数 \(\phi\) 同样可从数据自动估计（--phi auto），对不同多基因性程度的性状均有良好的适应性。

方法 先验类型 LD 参考面板 参数调优 适用场景
C+T 无（硬截断） 仅用于裁剪 需验证集选阈值 快速原型、小样本目标集
LDpred2-auto 点-正态混合 需要（同祖源） 自动估计 \(h^2\)、\(p\) 通用，训练集样本量 ≥ 5 万
PRS-CS-auto 连续收缩（马蹄） 需要（EUR/EAS/AFR） 自动估计 \(\phi\) 通用，多群体支持好
PRS-CSx 多群体连续收缩 多祖源联合 自动 非欧洲祖源目标人群

20.7.6 PRS 的评估指标

PRS 的预测性能因表型类型不同而采用不同指标：

连续表型（如身高、BMI、血压）：使用 \(R^2\)（在独立测试集中 PRS 解释的表型方差比例）。增量 \(R^2\)（incremental \(R^2\)）是更规范的报告方式，即在协变量（年龄、性别、前十个 PCA 主成分）调整后，PRS 额外解释的方差量。

二元表型（如疾病状态）： - AUC（area under the ROC curve）：衡量 PRS 区分病例与对照的能力，不受患病率影响； - Nagelkerke \(R^2\)：logistic 回归中 PRS 解释的伪决定系数，受患病率影响，跨研究比较时需用与训练集相同的患病率归一化； - 分位数相对风险（quantile relative risk）：将人群按 PRS 分位数（如底部 20% 对比顶部 20%，或顶部 1% 对比中间 40–60%）分组，报告各组间疾病相对风险——这是临床沟通中最直观的报告形式。

20.7.7 PRS 的临床转化与伦理考量

PRS 在临床场景中的应用尚处于早期阶段，但已在以下方向展现出实际价值：风险分层——将高 PRS 个体纳入更密集的筛查方案（如乳腺癌高 PRS 女性提前进行钼靶筛查）；与家族史互补——PRS 可识别无家族史但遗传风险高的个体，填补传统临床风险评估的盲区；药物研发中的患者分层——在临床试验中按 PRS 分层分析，识别对特定干预获益更大的亚组。

然而，PRS 的临床应用也面临若干系统性局限与伦理挑战。首先，当前绝大多数大规模 GWAS 基于欧洲祖源人群，PRS 在非欧洲群体中的预测准确性显著下降，存在加剧健康不平等的风险。其次，高 PRS 并非疾病的决定因素，而是风险概率的上调——大多数高 PRS 个体不会患病，大多数患者的 PRS 处于中等水平，过度强调 PRS 结果可能导致不必要的医学焦虑或过度干预。此外，PRS 信息在保险和就业场景中的潜在歧视性使用是亟待监管框架覆盖的伦理议题。

黄金法则：PRS 不等于命运

PRS 描述的是群体层面的概率分布，而非个体层面的确定性预测。在解读高 PRS 结果时，始终应结合环境因素、生活方式、临床检测结果综合判断，并以”概率性风险信息”而非”遗传诊断”的框架传达给被测者。PRS 的最大价值在于推动从”已病则治”向”未病先防”的预防医学范式转变，而非制造遗传决定论的新焦虑。

20.8 稀有变异聚合检验

20.8.1 稀有变异分析的动机

标准 GWAS 主要针对常见变异（MAF > 1%），对稀有变异（MAF < 1%）的统计检验力极低：在有限样本量下，稀有变异在任何基因型组别中仅有少数个体携带，单 SNP 检验几乎无法达到显著。

然而，稀有变异（尤其是功能性稀有变异，如 Loss-of-Function 变异）往往具有较大的效应量，其在基因水平的聚合分析可显著提升检验力。

20.8.2 SKAT 与 SKAT-O

序列核关联检验（Sequence Kernel Association Test，SKAT） 将同一基因（或基因组窗口）内所有稀有变异的效应量视为服从先验分布（均值为零）的随机变量，通过核函数（kernel function）将多个稀有变异的基因型信息聚合为单一统计量：

\[ Q_{\text{SKAT}} = (\mathbf{y} - \hat{\boldsymbol{\mu}})^\top \mathbf{K} (\mathbf{y} - \hat{\boldsymbol{\mu}}) \]

其中 \(\mathbf{K} = \mathbf{G} \mathbf{W} \mathbf{G}^\top\) 为加权核矩阵，\(\mathbf{G}\) 为基因型矩阵，\(\mathbf{W}\) 为基于 MAF 的权重矩阵（稀有变异权重更高）。

SKAT-O（Optimal SKAT） 通过线性组合负担检验（burden test）和 SKAT 统计量，自适应地在”方向一致的稀有变异效应”和”效应方向混合的稀有变异”两种场景之间取得最优检验力：

\[ Q_\rho = (1-\rho) \cdot Q_{\text{SKAT}} + \rho \cdot Q_{\text{burden}}, \quad \rho \in [0, 1] \]

20.9 动态性状的 GWAS：功能作图方法

{{Wiki:Quantitative_trait_locus}}

20.9.1 纵向表型的映射挑战

标准 GWAS 以单时间点表型（如某次测量的身高、血压值）为分析对象，将其视为固定标量与基因型关联。然而，许多生物学性状天然具有时间动态性（time-varying nature）：生长速率、药物代谢曲线、疾病进展轨迹、季节性表达波动……这些性状的生物学核心不在于某一时刻的表型值，而在于整条动态轨迹的遗传控制机制。对这类性状逐时间点独立做 GWAS 有两大缺陷：其一，大量低效的重复检验造成严重的多重检验负担；其二，完全丢失了时间点之间的生物学依赖结构，等于抛弃了最具信息量的动态信息。

功能作图（Functional Mapping，FM） 正是为解决上述问题而提出的统计框架。其核心思想是：将生物数学模型（描述性状动态规律的参数方程）直接嵌入 QTL/GWAS 关联分析的似然函数，用数量性状的发育参数而非原始表型值作为遗传效应的载体。这一框架由邬荣领（Rongling Wu）、马长兴（Changxing Ma）和 George Casella 于 2002 年在 Genetics 上正式提出，随后在 Nature Reviews Genetics（2006）上系统综述，发展为功能基因组学中动态性状遗传解析的基础理论之一。

20.9.2 功能作图的数学框架

设 \(n\) 个个体在 \(T\) 个时间点 \(t_1, t_2, \ldots, t_T\) 上各测量一次表型。个体 \(i\) 的观测向量为 \(\mathbf{y}_i = (y_{i1}, y_{i2}, \ldots, y_{iT})^\top\)。在 QTL/SNP 关联分析框架下，将个体按基因型 \(g \in \{AA, Aa, aa\}\)（加性模型下简化为 0/1/2）分组，每组的期望轨迹由参数化曲线描述：

\[E[y_{it} \mid g_i = g] = f(t;\, \boldsymbol{\theta}_g)\]

其中 \(\boldsymbol{\theta}_g\) 是第 \(g\) 组基因型的曲线参数向量（如 Logistic 模型的渐近值 \(a\)、增长率 \(r\)、拐点 \(t_0\)），不同基因型之间曲线参数的差异即为 QTL 效应。

均值模型——三参数 Logistic 生长曲线：

\[f(t;\, a, r, t_0) = \frac{a}{1 + e^{-r(t - t_0)}}\]

  • \(a\)：渐近值（最终体量上限）；
  • \(r\)：增长率（曲线陡峭程度）；
  • \(t_0\)：拐点时刻（增长速率最大处）。

在加性模型下，两个纯合基因型（\(g = 0\) 和 \(g = 2\)）的参数向量 \(\boldsymbol{\theta}_0 = (a_0, r_0, t_{0,0})\) 与 \(\boldsymbol{\theta}_2 = (a_2, r_2, t_{0,2})\) 直接反映等位基因替换对生长轨迹各维度的加性效应。

协方差结构——结构化时间相关：

纵向测量之间存在时间自相关，不满足独立性假设。功能作图采用AR(1) 自回归结构对残差协方差建模：

\[\Sigma_{jk} = \sigma^2 \rho^{|t_j - t_k|}\]

其中 \(\sigma^2\) 为残差方差，\(\rho\) 为相邻时间点间的自相关系数（\(0 < \rho < 1\)）。AR(1) 结构的闭合形式逆矩阵和行列式使得计算复杂度从 \(O(T^3)\) 降至 \(O(T)\)，是功能作图计算可行的关键技巧。

混合模型似然函数：

\[\ln L(\boldsymbol{\Theta}) = \sum_{i=1}^{n} \ln \left[\sum_{g=0}^{2} p_g \cdot \phi(\mathbf{y}_i;\, \boldsymbol{\mu}_g,\, \boldsymbol{\Sigma})\right]\]

其中 \(p_g\) 为各基因型频率（由 Hardy-Weinberg 平衡和等位基因频率确定），\(\phi(\cdot;\, \boldsymbol{\mu}_g, \boldsymbol{\Sigma})\) 为多元正态密度函数，均值向量 \(\boldsymbol{\mu}_g = (f(t_1;\boldsymbol{\theta}_g), \ldots, f(t_T;\boldsymbol{\theta}_g))^\top\)。通过 EM 算法迭代估计 \(\boldsymbol{\Theta} = \{\boldsymbol{\theta}_0, \boldsymbol{\theta}_2, \sigma^2, \rho\}\)，计算似然比统计量（LRT）检验 \(H_0: \boldsymbol{\theta}_0 = \boldsymbol{\theta}_2\)（无 QTL 效应）。

参数的生物学可解释性是功能作图的核心优势

传统多时间点 GWAS 报告的是”在时间点 \(t_k\) 处，等位基因 A 将表型平均提升 \(\beta_k\) 个单位”——这些 \(\beta_k\) 是统计描述，缺乏生物学含义的整合。功能作图报告的是”等位基因 A 将渐近体量 \(a\) 提升 \(\Delta a\)，同时不影响生长速率 \(r\)“——每个参数对应一个具有生物学意义的发育维度，检验效力也因参数聚合了整条曲线信息而显著高于逐点检验。

20.9.3 正交 Legendre 多项式：非参数化扩展

当研究者不确定数据适合哪种生物数学曲线时，可改用正交 Legendre 多项式（Legendre Orthogonal Polynomials，LOP） 作为均值函数的非参数化基底：

\[f(t;\, \boldsymbol{\phi}) = \sum_{k=0}^{K} \phi_k P_k(\tilde{t})\]

其中 \(P_k(\tilde{t})\) 为 \(k\) 阶 Legendre 多项式（\(\tilde{t} \in [-1, 1]\) 为归一化时间），\(\phi_k\) 为各基底的展开系数。阶数 \(K\) 通过 AIC/BIC 模型选择确定（通常 \(K = 3\)–\(5\) 即可捕捉主要动态特征）。不同基因型的 LOP 系数向量 \(\boldsymbol{\phi}_g\) 之差即为遗传效应的非参数估计。Legendre 多项式基底因正交性使设计矩阵数值稳定，且可通过 \(R\) 的 poly() 函数直接生成。

20.9.4 R 模拟实现

以下代码从零开始模拟一个功能作图场景：100 个个体在 10 个时间点上的纵向生长数据，含一个影响 Logistic 增长曲线参数的 QTL，演示模型拟合与基因型轨迹差异可视化。

# Functional mapping simulation and fitting
# Simulates longitudinal growth data with a QTL affecting curve parameters
set.seed(2024)
library(ggplot2)

# --- 1. Parameters ---
n        <- 150          # sample size
t_obs    <- seq(1, 10)   # 10 time points
T        <- length(t_obs)
maf      <- 0.35         # minor allele frequency of QTL

# Logistic growth: f(t; a, r, t0) = a / (1 + exp(-r*(t - t0)))
logistic_growth <- function(t, a, r, t0) a / (1 + exp(-r * (t - t0)))

# Genotype-specific curve parameters (additive model)
# Genotype AA (g=0): slower, lower asymptote
# Genotype Aa (g=1): intermediate (midpoint)
# Genotype aa (g=2): faster, higher asymptote
theta <- list(
  "0" = c(a = 8.0, r = 0.5, t0 = 5.0),   # AA
  "1" = c(a = 9.5, r = 0.7, t0 = 4.5),   # Aa
  "2" = c(a = 11.0, r = 0.9, t0 = 4.0)   # aa
)

# --- 2. Simulate genotypes under HWE ---
p    <- maf
q    <- 1 - p
freq <- c(q^2, 2*p*q, p^2)   # P(AA), P(Aa), P(aa)
geno <- sample(0:2, n, replace = TRUE, prob = freq)

# --- 3. AR(1) covariance matrix ---
rho   <- 0.75
sigma <- 1.2
ar1_cov <- function(T, sigma, rho) {
  Sigma <- matrix(0, T, T)
  for (i in seq_len(T))
    for (j in seq_len(T))
      Sigma[i, j] <- sigma^2 * rho^abs(i - j)
  Sigma
}
Sigma <- ar1_cov(T, sigma, rho)
L     <- t(chol(Sigma))   # Cholesky for simulation

# --- 4. Simulate phenotype trajectories ---
sim_data <- lapply(seq_len(n), function(i) {
  g  <- geno[i]
  mu <- logistic_growth(t_obs,
                        a  = theta[[as.character(g)]]["a"],
                        r  = theta[[as.character(g)]]["r"],
                        t0 = theta[[as.character(g)]]["t0"])
  y  <- mu + as.vector(L %*% rnorm(T))
  data.frame(id = i, genotype = g, time = t_obs, y = y)
})
dat <- do.call(rbind, sim_data)

# --- 5. Visualize observed trajectories by genotype ---
geno_labels <- c("0" = "AA (g=0)", "1" = "Aa (g=1)", "2" = "aa (g=2)")
dat$geno_label <- factor(geno_labels[as.character(dat$genotype)],
                          levels = geno_labels)

# Mean trajectory per genotype
mean_traj <- aggregate(y ~ time + geno_label, dat, mean)

ggplot(dat, aes(x = time, y = y, group = id, color = geno_label)) +
  geom_line(alpha = 0.15, linewidth = 0.3) +
  geom_line(data = mean_traj,
            aes(x = time, y = y, group = geno_label, color = geno_label),
            linewidth = 1.4, inherit.aes = FALSE) +
  scale_color_manual(values = c("steelblue", "darkorange", "firebrick")) +
  labs(x = "Time", y = "Phenotype",
       title = "Functional Mapping: Longitudinal Trajectories by Genotype",
       subtitle = sprintf("n=%d, MAF=%.2f; QTL affects a, r, and t0 of logistic curve",
                          n, maf),
       color = "Genotype") +
  theme_classic() +
  theme(legend.position = "top")

# --- 6. Fit functional mapping model via numerical optimization ---
# Log-likelihood under logistic mean + AR(1) covariance
neg_loglik_fm <- function(params, dat, t_obs, T) {
  # params: a0, r0, t0_0, a2, r2, t0_2, sigma, rho
  a0   <- params[1]; r0  <- params[2]; tt0_0 <- params[3]
  a2   <- params[4]; r2  <- params[5]; tt0_2 <- params[6]
  sig  <- exp(params[7])   # log-parameterized for positivity
  rho_ <- plogis(params[8]) * 0.999 + 0.001   # [0.001, 1)

  # AR(1) covariance and its inverse
  Sigma <- ar1_cov(T, sig, rho_)
  tryCatch({
    Sigma_inv <- solve(Sigma)
    log_det   <- determinant(Sigma, logarithm = TRUE)$modulus
  }, error = function(e) return(1e9))

  maf_est <- mean(dat$genotype) / 2
  freq_g  <- c((1-maf_est)^2, 2*maf_est*(1-maf_est), maf_est^2)

  # Mean curves for each genotype
  mu_list <- list(
    logistic_growth(t_obs, a0, r0, tt0_0),
    logistic_growth(t_obs, (a0+a2)/2, (r0+r2)/2, (tt0_0+tt0_2)/2),  # Aa midpoint
    logistic_growth(t_obs, a2, r2, tt0_2)
  )

  # Mixture log-likelihood (sum over individuals)
  ll <- 0
  for (i in unique(dat$id)) {
    y_i  <- dat$y[dat$id == i]
    g_i  <- dat$genotype[dat$id == i][1]
    comp <- numeric(3)
    for (g in 0:2) {
      resid <- y_i - mu_list[[g + 1]]
      comp[g+1] <- freq_g[g+1] *
        exp(-0.5 * (t(resid) %*% Sigma_inv %*% resid +
                    log_det + T * log(2 * pi)))
    }
    ll <- ll + log(sum(comp) + 1e-300)
  }
  -ll   # return negative log-likelihood for minimization
}

# Initial values near true parameters
init_params <- c(a0 = 8.5, r0 = 0.55, tt0_0 = 4.8,
                 a2 = 10.5, r2 = 0.85, tt0_2 = 4.2,
                 log_sigma = log(1.0), logit_rho = qlogis(0.7))

cat("Fitting functional mapping model (this may take ~30 seconds)...\n")
fit <- optim(init_params, neg_loglik_fm,
             dat = dat, t_obs = t_obs, T = T,
             method = "Nelder-Mead",
             control = list(maxit = 2000, reltol = 1e-6))

# Extract fitted parameters
p_hat   <- fit$par
a0_hat  <- p_hat[1]; r0_hat  <- p_hat[2]; t0_0_hat <- p_hat[3]
a2_hat  <- p_hat[4]; r2_hat  <- p_hat[5]; t0_2_hat <- p_hat[6]
sig_hat <- exp(p_hat[7])
rho_hat <- plogis(p_hat[8]) * 0.999 + 0.001

cat(sprintf("Fitted AA  curve: a=%.2f, r=%.3f, t0=%.2f\n",
            a0_hat, r0_hat, t0_0_hat))
cat(sprintf("Fitted Aa  curve: a=%.2f, r=%.3f, t0=%.2f\n",
            (a0_hat+a2_hat)/2, (r0_hat+r2_hat)/2, (t0_0_hat+t0_2_hat)/2))
cat(sprintf("Fitted aa  curve: a=%.2f, r=%.3f, t0=%.2f\n",
            a2_hat, r2_hat, t0_2_hat))
cat(sprintf("Covariance: sigma=%.3f, rho=%.3f\n", sig_hat, rho_hat))

# --- 7. LRT: test QTL effect (H0: theta_0 == theta_2) ---
# H0: fit single curve for all genotypes
neg_loglik_null <- function(params, dat, t_obs, T) {
  a  <- params[1]; r  <- params[2]; t0 <- params[3]
  sig <- exp(params[4]); rho_ <- plogis(params[5]) * 0.999 + 0.001
  Sigma     <- ar1_cov(T, sig, rho_)
  Sigma_inv <- solve(Sigma)
  log_det   <- determinant(Sigma, logarithm = TRUE)$modulus
  mu <- logistic_growth(t_obs, a, r, t0)
  ll <- 0
  for (i in unique(dat$id)) {
    y_i   <- dat$y[dat$id == i]
    resid <- y_i - mu
    ll    <- ll - 0.5 * (t(resid) %*% Sigma_inv %*% resid +
                          log_det + T * log(2*pi))
  }
  -ll
}

fit_null <- optim(c(9.5, 0.7, 4.5, log(1.0), qlogis(0.7)),
                  neg_loglik_null,
                  dat = dat, t_obs = t_obs, T = T,
                  method = "Nelder-Mead",
                  control = list(maxit = 2000))

lrt_stat <- 2 * (fit_null$value - fit$value)
df_lrt   <- 6 - 3   # alternative (6 params) vs null (3 params)
p_lrt    <- pchisq(lrt_stat, df = df_lrt, lower.tail = FALSE)
cat(sprintf("\nLRT statistic = %.2f (df=%d), p = %.4e\n",
            lrt_stat, df_lrt, p_lrt))

# --- 8. Overlay fitted curves on data plot ---
t_fine    <- seq(1, 10, length.out = 100)
a1_hat  <- (a0_hat + a2_hat) / 2
r1_hat  <- (r0_hat + r2_hat) / 2
t0_1_hat <- (t0_0_hat + t0_2_hat) / 2
fit_curves <- rbind(
  data.frame(time = t_fine, y = logistic_growth(t_fine, a0_hat, r0_hat, t0_0_hat),
             geno_label = "AA (g=0)", type = "Fitted"),
  data.frame(time = t_fine, y = logistic_growth(t_fine, a1_hat, r1_hat, t0_1_hat),
             geno_label = "Aa (g=1)", type = "Fitted"),
  data.frame(time = t_fine, y = logistic_growth(t_fine, a2_hat, r2_hat, t0_2_hat),
             geno_label = "aa (g=2)", type = "Fitted")
)
fit_curves$geno_label <- factor(fit_curves$geno_label,
                                 levels = c("AA (g=0)", "Aa (g=1)", "aa (g=2)"))

ggplot(mean_traj, aes(x = time, y = y, color = geno_label)) +
  geom_point(size = 2.5) +
  geom_line(linetype = "dashed", linewidth = 0.8) +
  geom_line(data = fit_curves,
            aes(x = time, y = y, color = geno_label),
            linewidth = 1.3, inherit.aes = FALSE) +
  scale_color_manual(values = c("steelblue", "darkorange", "firebrick")) +
  annotate("text", x = 2, y = 10.5,
           label = sprintf("LRT = %.1f, p = %.2e", lrt_stat, p_lrt),
           hjust = 0, size = 3.5, color = "grey20") +
  labs(x = "Time", y = "Mean phenotype",

20.9.6 功能作图的延伸与局限

功能作图框架在过去二十年中延伸至多个方向，包括：反应规范映射（reaction norm mapping）——以环境梯度（温度、光照）替换时间轴，映射基因型-环境交互效应；网络功能作图——以系统发育树或代谢网络约束为先验，映射发育网络的遗传控制；高维纵向 GWAS——结合 LASSO 或 Elastic Net 正则化在全基因组范围内扩展功能作图。

然而，功能作图的实践也存在若干值得关注的局限，其中一项根本性约束来自其历史设计背景：Wu 等（2002）的原始框架是为受控实验交配群体（如回交 BC、F2、重组自交系 RIL）量身设计的。在这类群体中，各基因型频率由孟德尔分离比例精确确定（无需从群体等位基因频率估算），且实验设计本身消除了群体分层——所有个体来自同一受控杂交，亲缘关系均一可知，因此既无群体结构混杂（Q），也无隐性亲缘关系干扰（K）。

当功能作图被移植至自然群体或人类 GWAS 场景时，这一假设前提不再成立。自然群体中个体间存在不同程度的亲缘关系与祖源分层，若不加以校正，纵向表型中嵌入的家系内相关或亚群体表型差异会被错误地归因于 SNP 效应，产生系统性假阳性。解决方案是将功能作图的似然框架与混合线性模型的 Q+K 校正相结合：将主成分（Q 矩阵）作为固定效应协变量纳入均值方程，将全基因组亲缘关系矩阵（K 矩阵，即 GRM）引入残差协方差结构，以消除群体分层和隐性亲缘的双重混杂。这一整合尚未在主流功能作图软件中得到充分实现，是将该方法推广至人类纵向队列 GWAS 的重要方法论挑战。

此外，功能作图还面临其他实践局限：其一，对生物数学模型的选择具有主观性，错误的曲线模型会导致参数偏差；其二，AR(1) 协方差假设未必适合所有纵向数据（如非等间隔时间点或缺失数据较多时）；其三，计算开销远高于标准 GWAS，在基因组规模（数百万 SNP）下需要并行计算或两阶段筛选策略。对于现代大规模 GWAS 数据，通常先用标准单点 GWAS 初步筛选候选区域，再对候选 SNP 子集运行功能作图以获取动态效应的精细描述。

功能作图的实际应用场景

功能作图特别适合以下研究设计：（1）动植物育种——作物产量的生长曲线、体重增长轨迹的 QTL 定位；（2）药物基因组学——不同基因型对药物的时序反应曲线（药效动力学 PD 轨迹）；（3）纵向队列研究——老龄化研究中认知能力下降曲线、BMI 随年龄变化轨迹的遗传架构；（4）生理节律研究——昼夜节律、季节性表型波动的遗传基础。

参考答案 (Reference Answers)

  1. LD 裁剪（clumping）的目的是在全基因组范围内识别统计独立的信号位点，通过将 LD 区块内相关 SNP 合并为一个代表性位点来消除冗余，输出独立命中列表。精细定位（fine-mapping）的目的是在已知信号区域内，基于贝叶斯框架估计每个 SNP 为因果变异的后验概率，缩小因果候选变异的集合。在分析流程中，LD 裁剪先识别独立信号区域，再对每个区域进行精细定位，以获取高置信度的因果候选变异。

  2. 95% 可信集定义为：从 PIP 最高的 SNP 开始累加，直到累积 PIP 达到 0.95 为止的最小 SNP 集合。若该集合仅含一个 SNP，表明在当前数据和 LD 结构下，有超过 95% 的后验概率支持该变异为唯一因果变异，精细定位分辨率极高，该 SNP 是高优先级功能实验靶标。

  3. 三个假设为：（1）相关性：IV（遗传变异）与暴露因素显著关联；（2）独立性：IV 与混杂因素无关（由孟德尔随机化遗传原理保证）；（3）排他性：IV 仅通过暴露影响结局，不存在其他路径。水平多效性违背的是排他性假设。常用敏感性分析方法包括：MR-Egger 回归（检测平均多效性，截距不为零提示多效性）、加权中位数法（在 50% 以下 IV 有效时仍稳健）、加权众数法和 MR-PRESSO（检测并校正异常 IV）。

  4. C+T 方法优点是简单透明、计算快速，缺点是阈值以下小效应 SNP 被完全丢弃，且对 LD 校正不充分。LDpred2 优点是通过贝叶斯收缩利用全基因组 SNP 信息，在高多基因性性状上预测准确性更高，缺点是计算复杂度高、依赖参考 LD 矩阵质量。PRS 预测性能通常用连续表型的 Nagelkerke R²（或 Pearson \(r^2\)）以及二元表型的 AUC（曲线下面积）评估，均应在独立验证集中评估以避免过拟合。

  5. 稀有变异（MAF < 1%）在群体中携带者极少，单 SNP 关联检验在有限样本中的等位基因组数量太小，导致统计量方差极大，几乎无法在全基因组显著性阈值下达到显著。SKAT 将同一基因内所有稀有变异的基因型信息通过核矩阵聚合为单一统计量，相当于在基因水平上汇聚多个稀有变异的微小信号，从而实现检验力的提升，且允许各变异效应方向不同（与 burden test 的区别）。

  6. 分析流程：（1）从 LDL 胆固醇 GWAS（如 CARDIoGRAMplusC4D 或 UK Biobank）提取显著 SNP 作为工具变量，进行 LD 裁剪保留独立 IV；（2）在心肌梗死 GWAS 中提取对应 SNP 的效应量；（3）使用 TwoSampleMR 运行 IVW（主估计）、加权中位数法和 MR-Egger（敏感性）；（4）通过 MR-Egger 截距检验和 Cochran Q 统计量评估多效性和异质性。主要局限性：PCSK9 区域 SNP 已知仅通过 LDL 水平影响心血管风险，但若纳入其他区域的 LDL 相关 SNP，需检查是否存在通过 HDL、甘油三酯等其他路径的水平多效性；GWAS 样本重叠（winner’s curse 效应）也可能导致 IV 效应高估。

  7. （1）在候选区域（lead SNP ± 500 kb）内运行 SuSiE-RSS，获得每个 SNP 的 PIP 值和 95% 可信集，将 PIP > 0.1 的 SNP 列为优先候选。（2）在 GTEx 中查询可信集内 SNP 是否为候选基因（尤其是区域内或邻近基因）在相关组织中的 cis-eQTL；若存在 eQTL 重叠，运行 coloc 分析评估 PPH4 是否 > 0.8。（3）利用 ENCODE 或 Roadmap 的 H3K27ac、ATAC-seq 峰数据，通过 bedtools intersect 检查可信集 SNP 是否位于相关细胞类型的活跃增强子或启动子区域；进一步通过 Hi-C 数据确认该调控元件与哪个基因启动子存在染色质互作环，建立”SNP→调控元件→靶基因”的调控假说。

  8. 欧洲人群 lead SNP 通常不是因果变异本身，而是通过 LD 代理因果变异的标记物。当目标群体（非欧洲）的 LD 结构不同时，同一 lead SNP 与因果变异的 \(r^2\) 降低，代理效率下降，导致效应量估计偏向零，PRS 预测性能随之下降。多群体精细定位的原理：设两群体的 LD 矩阵分别为 \(\mathbf{R}_1\)（欧洲）和 \(\mathbf{R}_2\)（东亚），多群体似然为各群体似然之积 \(P(\text{data}_1 | \mathcal{C}, \mathbf{R}_1) \times P(\text{data}_2 | \mathcal{C}, \mathbf{R}_2)\)；若欧洲群体无法区分 SNP A 和 B（\(r^2 = 0.95\)），但东亚群体中 A 与 B 的 \(r^2 = 0.10\)，则东亚数据可有效区分二者，将 PIP 集中于真正的因果变异，实现可信集缩小。
