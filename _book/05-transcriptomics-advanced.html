25  Bulk 转录组高级分析：转录本结构与调控网络

Published

June 22, 2026

26 整体概述

标准 Bulk RNA-seq 分析（第4章）通常以基因表达量为主要对象，回答”哪些基因表达升高或降低”。然而，同一个基因可以产生多个转录本异构体；这些异构体可能具有不同的剪接结构、3’ 端长度、蛋白质结构域或转录后调控元件。因此，高级转录组分析需要进一步回答两个层次的问题：转录本结构如何变化，基因之间又如何以网络形式协同变化。本章围绕五个专题展开：先从可变剪接、基因融合与可变多腺苷酸化进入转录本结构层面，再介绍长读长转录组技术对全长异构体重建的补充价值，随后进入 WGCNA 共表达网络与 GENIE3/SINDy 调控网络推断，最后以扰动稳态方法讨论如何在明确实验设计下学习非线性基因调控动力系统。全章仍以第4章的 airway 数据集和已生成的表达矩阵、BAM 文件为主要实践基础；涉及扰动稳态的部分则以论文模型与模拟数据为主，强调机制建模对实验设计的额外要求。

学习目标：理解可变剪接、基因融合与可变多腺苷酸化的生物学意义；掌握 PSI、PDUI、ΔPDUI 等转录本结构指标的解释；能够解读 rMATS、STAR-Fusion、DaPars2 与 QAPA 的核心输出；理解 WGCNA 如何从表达矩阵构建共表达模块并关联表型；了解 GENIE3/dynGENIE3、ODE 与 SINDy 在基因调控网络推断中的适用场景与局限；理解扰动稳态数据如何将非线性调控模型转化为可估计的稀疏参数问题。

27 结构导览

本章当前包含五个专题，按”转录本结构 → 共表达模块 → 调控方向 → 扰动稳态机制模型”的逻辑组织：

输入：第4章产生的 clean FASTQ、BAM 文件、原始计数矩阵和归一化表达矩阵
  │
  ├── 模块 A：转录本结构分析（专题 5.1–5.2）
  │     │
  │     ├── 专题 5.1  可变剪接差异分析与基因融合鉴定
  │     │           rMATS → PSI / ΔPSI / FDR
  │     │           STAR-Fusion / Arriba → 融合转录本候选事件
  │     │
  │     └── 专题 5.2  可变多腺苷酸化分析与长读长转录组展望
  │                 DaPars2 / QAPA → PDUI / ΔPDUI / PAS 使用变化
  │                 PacBio IsoSeq / Nanopore direct RNA-seq / SQANTI3
  │
  └── 模块 B：表达网络与调控网络（专题 5.3–5.5）
        │
        ├── 专题 5.3  加权基因共表达网络分析
        │           WGCNA → 软阈值选择 → 模块识别 → 模块-表型关联 → 枢纽基因
        │
        ├── 专题 5.4  基因调控网络推断
        │           GENIE3 / dynGENIE3 → 有向调控边
        │           ODE / SINDy → 时间序列调控方程辨识
        │
        └── 专题 5.5  扰动稳态下的非线性调控模型学习
                    过表达 / 敲低 / 敲除 → 新稳态表达谱
                    热力学 ODE 模型 → 稳态线性化 → LASSO 稀疏项选择
                    补充：Regulon 思想（共表达 + motif 证据 → 直接调控）

模块 A 主要使用第4章产生的 BAM 文件或原始 FASTQ，关注单个基因内部的转录本结构变化。模块 B 主要使用归一化表达矩阵，关注基因之间的协同表达、调控方向与动态机制。两类分析互为补充：结构分析解释”同一基因产生了什么转录本”，网络分析解释”多个基因如何共同响应实验条件”。

28 各专题简介

28.0.1 专题 5.1：可变剪接差异分析与基因融合鉴定

本专题首先介绍可变剪接（Alternative Splicing）的基本事件类型，并以 PSI（Percent Spliced-In）量化外显子纳入比例。rMATS 部分重点讲解 junction reads 计数、Beta 二项分布建模、ΔPSI 解释和 JC/JCEC 输出文件差异。专题后半部分转向基因融合（Gene Fusion），说明融合转录本的产生机制、STAR-Fusion 的两步过滤策略，以及 Arriba 可视化的基本思路。

核心概念：

  • 七种可变剪接事件：SE、A5SS、A3SS、RI、MXE、AFE、ALE
  • PSI 与 ΔPSI：外显子纳入比例及其组间变化
  • rMATS 输出字段：IJC_SAMPLE、SJC_SAMPLE、IncLevelDifference、FDR
  • 基因融合：嵌合读段、融合断点、框内融合与框外融合
  • STAR-Fusion 与 Arriba：融合事件检测、过滤与可视化

实践内容：使用 airway 数据的 STAR 比对结果运行 rMATS，筛选差异剪接事件并绘制火山图与 Sashimi 图；演示 STAR-Fusion 和 Arriba 的融合转录本检测流程，学习解读高置信度融合候选事件。

28.0.2 专题 5.2：可变多腺苷酸化分析与长读长转录组展望

本专题围绕转录本 3’ 端结构变化展开。可变多腺苷酸化（Alternative Polyadenylation, APA）通过选择近端或远端多腺苷酸化位点（Polyadenylation Site, PAS），产生不同长度的 3’ 非翻译区（3’ UTR），从而影响 miRNA 调控、mRNA 稳定性和翻译效率。专题比较 DaPars2 与 QAPA 两类分析思路，并简要介绍 PacBio IsoSeq、Oxford Nanopore direct RNA-seq 和 SQANTI3 在全长转录本重建中的作用。

核心概念：

  • PAS、近端 PAS、远端 PAS 与 3’ UTR 长度变化
  • PDUI 与 ΔPDUI：远端 PAS 使用比例及其组间差异
  • DaPars2：基于 3’ UTR 覆盖度变化点发现潜在 PAS
  • QAPA：基于已注释 PAS 的精确定量
  • 长读长转录组：全长异构体、RNA 修饰直接检测、SQANTI3 结构分类

实践内容：使用第4章的 HISAT2 BAM 文件演示 DaPars2 工作流，筛选显著 APA 基因并绘制 ΔPDUI 结果图；展示 QAPA 的参考构建与 Salmon 定量思路；通过长读长转录组展望说明短读长方法在全长异构体重建中的局限。

28.0.3 专题 5.3：加权基因共表达网络分析

本专题从单基因差异表达转向基因模块层面。加权基因共表达网络分析（Weighted Gene Co-expression Network Analysis, WGCNA）根据多样本表达矩阵中的基因相关性构建网络，并将高度共表达的基因划分为模块。每个模块可用模块特征基因（Module Eigengene, ME）概括，再与实验条件或表型变量建立关联，从而发现与特定生物状态相关的功能基因集和枢纽基因（Hub gene）。

核心概念：

  • 软阈值（Soft Thresholding）与无标度拓扑拟合
  • 拓扑重叠矩阵（Topological Overlap Matrix, TOM）
  • 层次聚类、动态树切割与模块合并
  • 模块特征基因（Module Eigengene, ME）与模块-表型关联
  • 模块成员度（kME）、基因显著性（Gene Significance, GS）与枢纽基因识别

实践内容：使用 airway 数据集的 DESeq2 VST 表达矩阵运行 WGCNA，完成样本离群检测、软阈值选择、模块构建、模块-表型关联热图绘制和枢纽基因筛选；进一步将 WGCNA 模块与差异表达基因结果整合解释。

28.0.4 专题 5.4：基因调控网络推断——从随机森林到动力系统

本专题在 WGCNA 的基础上进一步讨论调控方向。WGCNA 给出的是无向共表达关系，而基因调控网络（Gene Regulatory Network, GRN）推断试图回答”哪个转录因子可能调控哪个靶基因”。GENIE3 将每个靶基因的表达预测建模为随机森林回归任务，用特征重要性为候选调控边排序；dynGENIE3 则利用时间序列先后关系增强方向约束。专题后半部分介绍 ODE 参数化建模与 SINDy 稀疏辨识，将调控网络从有向图扩展到动态方程。

核心概念：

  • GRN 推断的高维变量选择本质
  • GENIE3：逐靶基因随机森林回归与特征重要性边权
  • dynGENIE3：时间序列表达数据中的滞后调控关系
  • WGCNA 与 GENIE3 的互补：共表达模块与有向调控边整合
  • ODE、Hill 函数、SINDy 与弱 SINDy：动态调控方程辨识
  • AUROC 与 AUPR：稀疏调控网络评估指标

实践内容：以 airway VST 表达矩阵为输入，演示 GENIE3 的候选转录因子筛选、权重矩阵构建、Top 调控边提取和网络可视化；使用模拟时间序列数据展示 dynGENIE3 与 SINDy 的基本思想，并讨论其对真实时序 RNA-seq 实验设计的要求。

28.0.5 专题 5.5：扰动稳态下的非线性基因调控模型学习

本专题基于 Meister 等人提出的扰动稳态方法，讨论如何在没有密集时间序列的情况下学习一个非线性基因调控动力系统。该方法从 Bintu 型热力学转录模型出发，将 RNA 聚合酶结合概率写成有理函数；再利用扰动后稳态条件，把原本非线性的 ODE 参数估计问题转化为对参数线性的约束优化问题。其核心价值在于把实验设计、可解释模型和稀疏参数选择连接在一起。专题最后以 SCENIC 的 Regulon 思想作为通用补充，说明如何用顺式调控 motif 这一独立的机制证据，把共表达网络中的统计关联精炼为候选的直接调控关系（仅讨论通用思想，不涉及单细胞特有处理）。

核心概念：

  • 扰动稳态（Perturbed Steady-State）：过表达、敲低或敲除后系统重新达到的表达平衡
  • Bintu 型热力学转录模型：RNAP 结合概率、有理函数、激活、抑制和协同调控
  • 稳态线性化：利用 \(dx/dt=0\) 与 \(dy/dt=0\) 将非线性模型转化为线性参数方程
  • 带约束 LASSO：\(0\leq b_i\leq c_i\)、\(c_i(0)=1\) 与稀疏项选择
  • 非可辨识性：仅用稳态数据时自调控项可能存在等价方程，需要额外动态信息打破平局
  • 与 GENIE3/SINDy 的关系：稳态扰动数据、时间序列数据和纯表达矩阵的建模差异
  • Regulon 思想（SCENIC）：共表达候选 + motif 富集过滤 + 靶基因群活性打分；用正交机制证据剔除间接边

实践内容：围绕论文提出的扰动稳态框架推导关键方程；用小型模拟数据演示如何把候选非线性调控项转化为线性设计矩阵，并用简化 LASSO 恢复主要调控项；讨论该方法在真实 RNA-seq 或 Perturb-seq 实验中的设计要求与局限；并以 Regulon 思想说明 motif 证据如何精炼共表达网络。
