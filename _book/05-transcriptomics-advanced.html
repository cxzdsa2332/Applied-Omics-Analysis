25  Bulk 转录组高级分析：结构变异、长读长与共表达网络

Published

June 9, 2026

26 整体概述

标准 Bulk RNA-seq 分析（第4章）聚焦于基因表达量的变化，但转录组所携带的信息远不止于此。同一基因可以通过可变剪接（Alternative Splicing）产生具有不同功能的蛋白质异构体；染色体重排或转录通读可以产生融合转录本（Fusion Transcript），在肿瘤研究中具有重要的诊断意义；多个基因在转录层面的协同调控形成共表达模块（Co-expression Module），揭示了潜在的调控因子与生物学通路。短读长测序虽然能够检出这些结构变异的信号，但在全长异构体的精确重建方面存在固有局限。长读长测序技术（PacBio IsoSeq、Oxford Nanopore direct RNA-seq）能够直接读取全长 cDNA 或 RNA 分子，从根本上解决短读长拼接带来的歧义问题。本章在第4章基础上，系统讲解上述三个高级分析方向，并介绍在无参考基因组情形下进行 de novo 转录组组装的方法。

学习目标：掌握可变剪接五种事件类型及 rMATS 的 PSI 差异检验框架；理解长读长转录组学的工作流程与 SQANTI3 的结构分类体系；能运用 WGCNA 识别与表型相关的共表达基因模块；了解 Trinity de novo 转录组组装的适用场景与评估方法。

27 结构导览

本章分为三个模块，共五个专题，围绕”超越表达量”的高级分析主题组织：

输入：来自第4章的清洁 reads（BAM）+ 原始 FASTQ + 计数矩阵
  │
  ├── 模块 A：转录组结构分析（专题 5.1–5.2）
  │     ↓ rMATS → 可变剪接差异事件（PSI 检验）
  │     ↓ STAR-Fusion / Arriba → 基因融合鉴定
  │
  ├── 模块 B：长读长转录组学（专题 5.3–5.4）
  │     ↓ IsoSeq3 / Nanopore dRNA-seq → minimap2 比对
  │     ↓ SQANTI3 结构分类与质量过滤
  │     ↓ Trinity（无参场景）→ de novo 转录组 + BUSCO 评估
  │
  └── 模块 C：共表达网络分析（专题 5.5）
        ↓ WGCNA 软阈值选择 → 模块构建 → 模块-表型关联 → 枢纽基因

模块 A 和模块 C 以第4章产出的比对结果和计数矩阵为输入；模块 B 是独立的长读长分析路径，与短读长结果互为验证。

28 模块 A：转录组结构分析

模块目标： 理解可变剪接与基因融合的生物学机制，掌握基于短读长数据的结构变异检测方法，能解读 rMATS 的 PSI 差异检验结果与 STAR-Fusion 的输出报告。

28.1 专题 5.1：可变剪接差异分析

可变剪接（Alternative Splicing）是真核生物转录组多样性的主要来源之一。人类基因组中超过 90% 的多外显子基因发生可变剪接，同一基因的不同异构体可编码具有不同结构域组成、亚细胞定位甚至拮抗功能的蛋白质。从测序数据中检测可变剪接需要在外显子-外显子连接处（junction reads）与外显子内部读段（skipping reads）之间进行统计比较，核心指标是外显子纳入百分比（Percent Spliced-In, PSI）。rMATS 是目前最广泛使用的可变剪接差异分析工具，支持五种基本剪接事件类型的系统检测，并使用基于重复样本的贝叶斯统计模型控制假阳性。

{{Wiki:Alternative_splicing}}

核心概念：

  • 可变剪接五种基本事件类型：外显子跳跃（Skipped Exon, SE）、可变 5’ 剪接位点（A5SS）、可变 3’ 剪接位点（A3SS）、内含子保留（Retained Intron, RI）、互斥外显子（Mutually Exclusive Exons, MXE）
  • PSI 值的定义：\(\Psi = \frac{\text{inclusion junction reads}}{\text{inclusion junction reads} + \text{skipping junction reads}}\)
  • rMATS 统计框架：Beta 二项分布模型；\(\Delta\Psi\) 的贝叶斯估计；FDR 控制
  • 结果文件解读：.JC.txt（只用 junction reads）vs .JCEC.txt（junction + exon body reads）；IncLevelDifference 与 FDR 字段含义
  • Sashimi 图可视化：rmats2sashimiplot；连接弧线的粗细代表 junction reads 数量
  • 下游功能分析：差异剪接基因的 GO 富集；蛋白质域结构影响预测

实践内容：对 pasilla 数据集（dex 处理 vs 对照）运行 rMATS；识别显著差异的 SE 事件（|ΔPSI| > 0.1，FDR < 0.05）；生成代表性事件的 Sashimi 图；分析差异剪接基因的 GO 功能类别分布。

28.2 专题 5.2：基因融合鉴定

基因融合（Gene Fusion）是两个原本独立的基因的转录本被拼接在一起形成嵌合体转录本的现象，通常源于染色体易位、倒置或缺失等结构变异，也可由转录通读产生。在肿瘤研究领域，特定基因融合事件具有高度的诊断与预后价值（如慢性髓系白血病的 BCR-ABL1、前列腺癌的 TMPRSS2-ERG）。STAR-Fusion 通过两步过滤策略——先由 STAR 识别跨基因边界的嵌合比对，再利用 FusionInspector 在局部重建的融合转录本上进行读段重比对验证——实现了高精度的融合事件检测。

核心概念：

  • 基因融合的产生机制：染色体易位（translocation）、倒置（inversion）、缺失（deletion）与转录通读（read-through transcription）
  • STAR 嵌合比对参数：--chimSegmentMin（最小嵌合片段长度）；嵌合读段（chimeric reads）与 Chimeric.out.junction 文件
  • STAR-Fusion 过滤策略：FusionInspector 局部重比对验证；FFPM（Fusion Fragments Per Million）计数；已知假阳性数据库（FusionFilter/CTAT library）过滤
  • 融合事件分类：框内融合（in-frame）vs 框外融合；保留功能域的融合与截短融合
  • Arriba 替代工具：直接从 STAR 比对结果检测融合，内置高质量可视化模块
  • 验证策略：RT-PCR + Sanger 测序；扩增子测序验证低表达融合

实践内容：在含已知融合事件的模拟 RNA-seq 数据上运行 STAR-Fusion；解读输出结果中的关键字段；使用 Arriba 可视化代表性融合转录本的结构。

29 模块 B：长读长转录组学

模块目标： 理解长读长测序在全长异构体分析中的技术优势与误差特征，掌握 IsoSeq3 与 Nanopore 的分析流程，能使用 SQANTI3 对转录本进行结构分类与质量评估，了解 Trinity de novo 组装的适用场景。

29.1 专题 5.3：PacBio IsoSeq 与 Oxford Nanopore 全长转录组

短读长测序（150 bp）在重建全长转录本时面临本质性困难：当一个基因有多个相似的异构体时，短读段无法可靠地归属到特定转录本，尤其是在内含子较多或异构体共享大量外显子的情况下。PacBio IsoSeq 技术通过环形一致性测序（CCS）生成平均长度 2–15 kb、准确率高于 99% 的长读段，能够直接覆盖整条转录本从 5’ 帽到 poly-A 尾。Oxford Nanopore direct RNA-seq 则省去逆转录步骤，直接对 RNA 分子测序，保留天然的碱基修饰信息，但原始读段错误率较高（约 5–10%）。两种技术的分析流程既有共性，也有各自的特殊处理步骤。

{{Wiki:Pacific_Biosciences}}

核心概念：

  • PacBio CCS（Circular Consensus Sequencing）：SMRTbell 文库结构；多轮测序取均值的准确率提升机制；HiFi reads 的定义（≥ 3 次亚读段，准确率 ≥ Q20）
  • IsoSeq3 工作流：CCS → lima（primer/barcode 去除）→ isoseq3 refine（poly-A 修剪，FLNC 过滤）→ isoseq3 cluster（聚类去冗余）→ pbmm2 比对 → isoseq3 collapse（同一基因座转录本合并）
  • Oxford Nanopore direct RNA-seq：纳米孔电流信号直接读取 RNA；dorado 碱基识别（替代旧版 guppy）；minimap2 -ax splice -uf 比对参数（-uf 指定正链偏向）
  • 长读长比对的挑战：高错误率导致的剪接位点模糊；--junc-bed 利用已知剪接位点锚定比对
  • StringTie2 长读长模式：-L 参数启用长读长优化；转录本组装与表达量估计
  • 短读长辅助（Hybrid approach）：利用 Illumina 高精度数据校正长读长剪接位点

实践内容：演示 IsoSeq3 完整工作流（从 CCS BAM 到 collapsed GFF）；演示 Nanopore dRNA-seq minimap2 比对与 StringTie2 组装；生成两种技术检测到的转录本数量与长度分布对比图。

29.2 专题 5.4：SQANTI3 质量评估与 Trinity de novo 组装

长读长组装产生的转录本需要系统的质量评估：哪些是已知的注释转录本（FSM/ISM），哪些是新型连接（NNC），哪些可能是文库构建产生的伪影（Intrapriming、RT-switching）。SQANTI3 是专为长读长转录本设计的综合评估工具，提供详细的结构分类、质量指标计算与 ML 过滤。在无参考基因组的场景下，Trinity 提供了基于短读长的 de novo 转录组组装路线，通过 de Bruijn 图算法重建转录本序列（de Bruijn 图原理参见第2章基因组组装）。

核心概念：

  • SQANTI3 结构分类：FSM（Full Splice Match）、ISM（Incomplete Splice Match）、NIC（Novel in Catalog）、NNC（Novel Not in Catalog）、Antisense、Intergenic、Genic Genomic、Fusion
  • SQANTI3 质量指标：CAGE peak 支持（5’ 端真实性）、poly-A signal 支持（3’ 端真实性）、覆盖深度、RT-switching 评分
  • SQANTI3 filter：随机森林分类器识别 Intrapriming 与 RT-switching 伪异构体；rules 模式 vs ML 模式
  • Trinity de novo 组装：基于 k-mer de Bruijn 图；Trinity 特有的 Butterfly 模块处理可变剪接
  • Trinity 输出质量评估：BUSCO（转录组完整性，参见第2章）；TransRate（读段支持度）；N50 指标的局限性
  • de novo 转录组的适用场景：非模式生物研究、宏转录组分析

实践内容：SQANTI3 对 IsoSeq3 输出转录本进行结构分类，生成分类统计图与质量分布可视化；演示 SQANTI3 filter 去除低质伪异构体前后的转录本数量变化；展示 Trinity de novo 组装小型转录组并用 BUSCO 评估完整性。

30 模块 C：共表达网络分析

模块目标： 理解 WGCNA 的理论框架，能正确选择软阈值、构建共表达模块并进行模块-表型关联分析，从显著模块中识别枢纽基因。

30.1 专题 5.5：加权基因共表达网络分析

差异表达分析聚焦于单个基因层面的变化，而基因的功能往往通过协同调控网络来实现。加权基因共表达网络分析（Weighted Gene Co-expression Network Analysis, WGCNA）将多样本的基因表达矩阵转化为基因-基因相关性网络，并通过层次聚类将相关基因划分为模块（Module）。每个模块可通过模块特征基因（Module Eigengene, ME）——该模块中基因表达的第一主成分——与样本表型信息建立关联，从而识别与特定生物学状态相关的功能基因集，进而发现模块内连接度最高的枢纽基因（Hub gene）作为候选调控因子。

{{Wiki:Gene_co-expression_network}}

核心概念：

  • WGCNA 前提条件：样本数要求（通常 ≥ 15）；表达量归一化（vst 或 rlog 转换）；离群样本检测（层次聚类 + Euclidean distance + Z score 过滤）
  • 软阈值（Soft Threshold）选择：幂律拟合（scale-free topology fit index）；pickSoftThreshold() 函数；\(R^2 \geq 0.85\) 准则
  • 相关性度量：Pearson 相关（默认）vs 双权中点相关（bicor，对异常值更鲁棒）
  • 模块构建：层次聚类 + 动态树切割（Dynamic Tree Cut）；minModuleSize（最小模块大小）；mergeCutHeight（模块合并阈值）
  • 模块-表型关联：ME 与样本性状的 Pearson 相关矩阵；模块成员度（kME）；基因显著性（Gene Significance, GS）
  • 枢纽基因识别：模块内连接度（Intramodular Connectivity）；kME > 0.8 且 GS > 0.5 的交集作为枢纽基因候选
  • WGCNA 与 DEA 的整合：差异基因在各模块中的富集检验

实践内容：使用 pasilla 数据集（DESeq2 vst 归一化表达矩阵）运行 WGCNA；选择软阈值并构建共表达网络；识别与 dex 处理状态相关的核心模块；绘制模块-表型关联热图；从显著模块中筛选枢纽基因并进行 GO 富集验证。
