14 变异功能注释

14.1 从坐标到生物学意义

变异的可视化检测只是第一步，真正的价值在于理解其生物学后果。

14.1.1 基因组注释资源

{{Wiki:Gene_annotation}}

变异注释依赖于：

  1. 基因组注释文件 (GTF/GFF)
    • 包含所有已知基因、转录本、外显子、内含子的坐标
    • 常用数据库：Ensembl, RefSeq, GENCODE
  2. 蛋白质序列 (FASTA)
    • 用于判断同义/错义突变
  3. 保守性评分 (Conservation)
    • PhyloP, GERP：评估碱基的进化保守性
    • 保守位点突变更可能造成功能损伤
  4. 注释数据库
    • ClinVar：临床相关变异
    • gnomAD：全球人群频率
    • dbSNP：已知 SNP
    • COSMIC：肿瘤体细胞变异

14.1.2 变异的功能效应分类

效应类型 说明 重要性
同义突变 (Synonymous) 改变 DNA 不改变蛋白质（如 GCT → GCC，都编码 Ala） ✓ 通常低
错义突变 (Missense) 改变蛋白质序列（如 ATG (Met) → GTG (Val)） ✓✓✓ 高
无义突变 (Nonsense) 产生终止密码子，蛋白质截断（如 TGG (Trp) → TAG (Stop)） ✓✓✓ 非常高

内含子/2kb 外 (Intron/Intergenic) 不直接编码，功能未知 ✓ 低

14.2 GATK Best Practices 补充：BQSR 与群体层面 QC

本专题与专题 2.4 共享同一套 GATK 流程框架（FASTQ → BWA → MarkDuplicates → HaplotypeCaller → VariantFiltration，参见专题 2.4 各步骤代码），但有两个关键环节在上一专题中未能深入展开：碱基质量分数重校正（BQSR）和群体层面的变异质控（HWE 检验与 MAF 过滤）。这两步在人类 WGS 队列分析中是标准必做流程，理解其原理对后续 GWAS 章节至关重要。

{{Wiki:GATK}}

14.2.1 BQSR：碱基质量分数重校正

测序仪报告的碱基质量分数（Phred Q 值）往往存在系统性偏差：特定的读段位置、特定的碱基上下文（如 GCC 三联体）、或特定的碱基化学类型（如 G 碱基）会出现质量分持续高估或低估的现象。这种偏差不是随机误差，而是由测序化学反应机制决定的系统性偏差（systematic bias），会导致 HaplotypeCaller 在计算似然时使用的质量权重失真，进而影响变异检测的准确性。

{{Wiki:Base_quality_score_recalibration}}

BQSR 的核心思路是利用已知变异数据库作为”地面真相”：在已知变异位点处观测到的碱基差异归因于真实多态性，排除后剩余的差异才视为测序错误，据此统计每种上下文下的真实错误率，建立一张从”原始 Q 值 + 上下文特征”到”校正 Q 值”的映射表。

BQSR 考虑的协变量：

协变量 典型偏差模式
读段位置 (ReadPosition) read 末端错误率通常更高，Q 值被高估
碱基上下文 (Context) GGC 等特定三联体化学结构不稳定，错误率偏高
循环数 (Cycle) 测序循环越多，错误累积，但 Q 值不一定下调
读段方向 (Strand) 正反向读段化学特性不完全对称 
# 以 E. coli SRR2584863 数据为例（人类数据将 --known-sites 替换为 dbsnp.vcf）
# 注：细菌基因组 BQSR 通常不做（无已知变异库），此处以概念演示为主

# Step 1: 统计错误模式，生成重校正表
gatk BaseRecalibrator \
    -I ecoli_rg.bam \
    -R reference/ecoli_K12.fasta \
    --known-sites variants.vcf \       # 用已检测到的高置信变异作为已知位点
    -O recal_data.table

# Step 2: 可视化校正前后的质量分布（可选）
gatk AnalyzeCovariates \
    -bqsr recal_data.table \
    -plots bqsr_report.pdf

# Step 3: 应用校正，生成新 BAM
gatk ApplyBQSR \
    -I ecoli_rg.bam \
    -R reference/ecoli_K12.fasta \
    --bqsr-recal-file recal_data.table \
    -O ecoli_recalibrated.bam

细菌基因组的 BQSR 取舍

BQSR 在人类 WGS 中效果显著（可将 SNP 检测精度提升约 1–3%），但对细菌基因组收益有限：一方面，细菌无已知变异参考库，只能用本次检测结果”自举”，形成循环依赖；另一方面，细菌基因组变异率极低，BQSR 训练样本量不足。实践中，对模式生物细菌（如 E. coli）通常跳过 BQSR 步骤，直接使用原始质量分进行变异检测。

14.2.2 Hardy-Weinberg 平衡检验与 MAF 过滤

通过技术过滤（QUAL、DP、FS 等）后，还需对变异进行生物学层面的质量检验。这一步骤在群体遗传学分析和 GWAS 中尤为关键，但也适用于队列测序的常规质控。

Hardy-Weinberg 平衡（HWE）是群体遗传学的基本定律：在理想种群中，等位基因频率保持稳定，基因型频率满足：

\[P(\text{AA}) = p^2, \quad P(\text{Aa}) = 2pq, \quad P(\text{aa}) = q^2\]

{{Wiki:Hardy–Weinberg_principle}}

一个位点若显著偏离 HWE（通常以 \(p < 10^{-6}\) 为阈值），往往提示以下技术问题而非真实生物学信号：

HWE 偏离方向 最可能的原因
杂合子过少（heterozygote deficit） 基因分型错误、等位基因丢失（allele dropout）、近亲交配
杂合子过多（heterozygote excess） 样本混合、体细胞嵌合、比对错误

MAF（Minor Allele Frequency，次等位基因频率） 过滤则用于排除统计功效不足的极低频变异：

# 使用 PLINK 进行 HWE 检验和 MAF 过滤（常用于 GWAS 前处理）
plink \
  --vcf filtered.vcf.gz \
  --hwe 1e-6 \          # 移除 HWE p-value < 1e-6 的位点
  --maf 0.01 \          # 移除 MAF < 1% 的位点（极低频变异）
  --geno 0.05 \         # 移除缺失率 > 5% 的位点
  --mind 0.10 \         # 移除缺失率 > 10% 的个体
  --make-bed \
  --out filtered_qc

# 查看过滤统计
plink --bfile filtered_qc --freq --out allele_freq
# R: 手工计算 HWE p 值（卡方检验）
hwe_test <- function(n_hom_ref, n_het, n_hom_alt) {
  n_total <- n_hom_ref + n_het + n_hom_alt
  p <- (2 * n_hom_ref + n_het) / (2 * n_total)   # ref allele freq
  q <- 1 - p

  expected_hom_ref <- n_total * p^2
  expected_het     <- n_total * 2 * p * q
  expected_hom_alt <- n_total * q^2

  chi2 <- (n_hom_ref - expected_hom_ref)^2 / expected_hom_ref +
          (n_het     - expected_het)^2     / expected_het +
          (n_hom_alt - expected_hom_alt)^2 / expected_hom_alt

  p_value <- pchisq(chi2, df = 1, lower.tail = FALSE)

  cat(sprintf("Observed  : AA=%d  Aa=%d  aa=%d\n",
              n_hom_ref, n_het, n_hom_alt))
  cat(sprintf("Expected  : AA=%.1f  Aa=%.1f  aa=%.1f\n",
              expected_hom_ref, expected_het, expected_hom_alt))
  cat(sprintf("Chi2 = %.4f  p-value = %.2e  %s\n",
              chi2, p_value,
              ifelse(p_value < 1e-6, "FAIL HWE", "PASS HWE")))
}

# 示例：正常位点
cat("--- Normal locus ---\n")
hwe_test(n_hom_ref = 400, n_het = 450, n_hom_alt = 150)

# 示例：杂合子严重缺失（疑似基因分型错误）
cat("\n--- Suspected genotyping error ---\n")
hwe_test(n_hom_ref = 600, n_het = 50, n_hom_alt = 350)

MAF 阈值的选择

MAF < 1% 的变异通常称为低频变异（low-frequency variant），MAF < 0.1% 称为罕见变异（rare variant）。常规 GWAS 一般过滤 MAF < 1%–5%，保留常见变异（common variant）进行关联分析；若专门研究罕见变异对疾病的聚合效应，则需专用方法（如 SKAT、BURDEN test），不宜用简单 MAF 过滤。

14.3 变异注释与优先级评估

14.3.1 SnpEff 实战注释

SnpEff 内置了数千个物种的基因组数据库，可直接按物种标识符调用。对于本节使用的 E. coli K-12 MG1655 参考基因组，内置数据库标识符为 Escherichia_coli_k_12。

# 1. 查询 SnpEff 内置数据库（确认 E. coli 版本）
snpEff databases | grep -i "coli"

# 2. 对 E. coli variants.vcf 进行基础功能注释
snpEff \
    -stats ecoli_snpeff_report.html \
    -csvStats ecoli_snpeff_stats.csv \
    Escherichia_coli_k_12 \
    variants.vcf > annotated.vcf

# 3. 统计各效应类型的变异数量
grep -v "^#" annotated.vcf | \
    grep -oP "(?<=ANN=)[^;]+" | \
    cut -d'|' -f2 | sort | uniq -c | sort -rn

# 4. 提取高影响（HIGH impact）变异——优先关注无义突变和移码突变
grep -v "^#" annotated.vcf | \
    awk -F'\t' '{
        if ($8 ~ /IMPACT=HIGH/ || $8 ~ /HIGH/) print $0
    }' > high_impact_variants.vcf

echo "High impact variants: $(grep -v "^#" high_impact_variants.vcf | wc -l)"

对于人类 WGS 数据（如 GIAB HG002 基准集），SnpEff 结合 VEP 能提供更丰富的注释，包括 SIFT/PolyPhen 预测评分和 ClinVar 临床分类：

# VEP 注释（人类 hg38，需提前下载 cache）
vep \
    --input_file variants_hg38.vcf \
    --output_file annotated_hg38.vcf \
    --assembly GRCh38 \
    --cache --offline \
    --sift b \          # SIFT 预测：氨基酸替换对蛋白功能的影响
    --polyphen b \      # PolyPhen-2：错义突变致病性预测
    --regulatory \      # 注释调控区域（启动子、增强子）
    --variant_class \   # 标注变异类型（SNV/indel/structural）
    --fork 4            # 并行线程数

14.3.2 注释字段解读

注释后的 VCF 会添加 ANN 字段：

##ANN=<ID=ANN,Number=.,Type=String,Description="Annotation">

ANN=T|missense_variant|MODERATE|TP53|ENSG00000141510|transcript|ENST00000269305|
protein_altering|2/7|c.715C>T|p.Arg239Trp|818/2584|79%|...
字段 含义
T 替代等位体
missense_variant 效应类型（同义/错义/无义等）
MODERATE 影响程度（HIGH/MODERATE/LOW）
TP53 基因名称
ENSG… 基因 ID
c.715C>T 核酸变化 (cDNA)
p.Arg239Trp 蛋白质变化（单字母：RW，三字母：Arg→Trp）
79% 蛋白质位置百分比

14.3.3 致病性评估

ACMG 指南 将变异分为 5 级：

  1. 病原性 (Pathogenic) - 明确导致疾病
  2. 可能病原性 (Likely Pathogenic) - 很可能导致疾病
  3. 意义不明 (VUS, Variant of Uncertain Significance) - 无法判断
  4. 可能良性 (Likely Benign) - 很可能无害
  5. 良性 (Benign) - 无致病性

评估标准： - 群体频率（gnomAD, 1000 Genomes） - 在致病数据库中的注释（ClinVar） - 计算预测（CADD, SIFT, PolyPhen） - 功能研究证据 - 家族聚集性

14.4 IGV 交互式可视化

Integrative Genomics Viewer (IGV) 用于直观查看比对和变异。

14.4.1 IGV 核心功能

  1. 加载参考基因组和注释
    • 选择 hg38/hg19
    • 导入 GTF 基因注释
  2. 导入 BAM 文件
    • 自动显示 read 覆盖度
    • 颜色编码：
      • 绿色：正向比对
      • 红色：反向比对
      • 灰色：低 MAPQ（质量差）
      • 彩色错配：红(A) 蓝(C) 绿(G) 黄(T)，与参考不同时显示
  3. 导入 VCF 文件
    • 标记变异位置
    • 显示基因型和注释
  4. 肉眼检查技巧
    • 覆盖度均匀 → 无明显偏差
    • 错配位置一致 → 可能真实变异
    • 错配散布不规则 → 可能测序或比对错误
    • 配对 reads 距离正确 → 有效比对

14.5 实践：VCF 注释与致病性评估

14.5.1 Python 整合注释和过滤

import vcf
import math

class VariantPrioritizer:
    def __init__(self, annotated_vcf):
        """Initialize with annotated VCF (from SnpEff/VEP)"""
        self.vcf = vcf.Reader(filename=annotated_vcf)
        self.variants = []

    def parse_snpeff_annotation(self, ann_field):
        """Parse SnpEff ANN field"""
        if not ann_field:
            return None

        # SnpEff ANN format
        annotations = []
        for ann in ann_field:
            parts = ann.split('|')
            if len(parts) >= 15:
                annotations.append({
                    'alt': parts[0],
                    'effect': parts[1],
                    'impact': parts[2],
                    'gene_name': parts[3],
                    'gene_id': parts[4],
                    'feature_type': parts[5],
                    'hgvs_c': parts[9],
                    'hgvs_p': parts[10]
                })
        return annotations

    def classify_impact(self, impact_str):
        """Convert IMPACT string to numeric score"""
        impact_scores = {
            'HIGH': 4,
            'MODERATE': 3,
            'LOW': 2,
            'MODIFIER': 1
        }
        return impact_scores.get(impact_str, 0)

    def get_population_frequency(self, info_dict):
        """Extract population frequency (AF from gnomAD or other sources)"""
        for key in ['AF', 'gnomAD_AF']:
            if key in info_dict:
                af = info_dict[key]
                if isinstance(af, list):
                    af = af[0]
                return float(af) if af else None
        return None

    def prioritize_variants(self):
        """Prioritize variants by pathogenicity"""
        results = []

        for record in self.vcf:
            variant_info = {
                'chrom': record.CHROM,
                'pos': record.POS,
                'ref': record.REF,
                'alt': record.ALT[0] if record.ALT else None,
                'qual': record.QUAL
            }

            # Parse annotations
            if 'ANN' in record.INFO:
                anns = self.parse_snpeff_annotation(record.INFO['ANN'])
                if anns:
                    ann = anns[0]  # Use most severe
                    variant_info['effect'] = ann['effect']
                    variant_info['impact'] = ann['impact']
                    variant_info['gene'] = ann['gene_name']
                    variant_info['hgvs_p'] = ann['hgvs_p']
                    variant_info['impact_score'] = self.classify_impact(ann['impact'])

            # Get population frequency
            variant_info['af'] = self.get_population_frequency(record.INFO)

            # Calculate priority score
            variant_info['priority_score'] = self._calculate_priority(variant_info)

            results.append(variant_info)

        # Sort by priority score (descending)
        results.sort(key=lambda x: x.get('priority_score', 0), reverse=True)
        return results

    def _calculate_priority(self, variant):
        """Calculate variant priority score (0-100)"""
        score = 0

        # Impact score (max 60)
        impact_score = variant.get('impact_score', 0)
        score += impact_score * 15

        # Quality score (max 20)
        qual = variant.get('qual', 0)
        if qual:
            score += min(20, qual / 5)

        # Population frequency (max 20, rare variants score higher)
        af = variant.get('af')
        if af:
            if af < 0.01:
                score += 20  # Very rare
            elif af < 0.05:
                score += 10  # Rare
            else:
                score += 0   # Common
        else:
            score += 15  # Unknown frequency

        return min(100, score)

    def report_top_variants(self, top_n=10):
        """Print top N prioritized variants"""
        variants = self.prioritize_variants()

        print("\n" + "="*80)
        print("变异优先级排序（致病性从高到低）")
        print("="*80)

        for i, var in enumerate(variants[:top_n], 1):
            print(f"\n{i}. {var['chrom']}:{var['pos']} {var['ref']}->{var['alt']}")
            print(f"   基因: {var.get('gene', 'Unknown')}")
            print(f"   效应: {var.get('effect', 'Unknown')}")
            print(f"   影响: {var.get('impact', 'Unknown')}")
            print(f"   蛋白变化: {var.get('hgvs_p', 'Unknown')}")
            print(f"   群体频率: {var.get('af', 'Unknown')}")
            print(f"   质量评分: {var.get('qual', 'N/A')}")
            print(f"   优先级分数: {var.get('priority_score', 0):.1f}/100")

# 使用示例

15 从个体变异到群体研究：迈向 GWAS

经过前几个专题的完整训练，读者已掌握从原始测序数据到高质量注释变异集的全套流程。本节作为本章的收尾，简要介绍当分析对象从单个样本扩展为数百乃至数千个个体时，需要额外关注的几个核心概念。这些概念在后续 GWAS 章节将得到深入展开。

15.1 等位基因频率：从个体到群体的核心统计量

当同一套流程被应用于大量个体时，分析的核心不再是”这个人有什么变异”，而是”在这个群体里，某个位点的替代等位体有多常见”。等位基因频率（Allele Frequency，AF） 正是这一问题的答案：某个等位体在所有观测等位体中所占的比例。

次等位基因频率（Minor Allele Frequency，MAF） 是其中出现频率较低的那个等位体的频率，是衡量变异在群体中稀有程度的标准指标：

MAF 范围 变异分类 在 GWAS 中的意义
> 5% 常见变异（Common Variant） GWAS 的主要研究对象，统计功效高
1%–5% 低频变异（Low-frequency Variant） 需要更大样本才能可靠检测
< 1% 罕见变异（Rare Variant） 单位点分析功效不足，需聚合检验

MAF 过滤（通常保留 MAF ≥ 1%–5%）是 GWAS 数据预处理的标准步骤之一——极低频变异的统计功效过低，纳入分析只会增加多重检验负担而不带来可靠信号。

15.2 群体分层：必须警惕的混杂因素

不同地理来源的人群在基因组上存在系统性的频率差异。这一现象称为群体分层（Population Stratification），是 GWAS 中最重要的混杂因素。

{{Wiki:Population_stratification}}

核心问题：若一项病例-对照研究中，病例与对照的祖先来源不均衡（例如病例多为东亚人、对照多为欧洲人），那么两组之间大量的基因组频率差异与目标疾病完全无关，纯粹来自祖先背景的不同——却会被统计模型误判为疾病关联信号，产生大量假阳性。

直觉类比：
假设东亚人群中乳糖不耐受比例（某基因型频率）更高，
而该研究恰好病例多东亚、对照多欧洲，
即便乳糖不耐受基因与目标疾病毫无关联，
它也会在统计上显示"关联"——这是群体分层引起的假阳性。

标准解决方案：对全基因组 SNP 矩阵做主成分分析（PCA），前几个主成分会自动捕获样本的祖先背景，在后续关联模型中将这些主成分作为协变量加入，即可校正祖先混杂。具体方法与代码详见 GWAS 章节。

15.3 连锁不平衡：基因组的区块结构

基因组上相邻位点的等位体并非随机独立——由于染色体重组频率有限，紧邻的 SNP 之间存在非随机关联，称为连锁不平衡（Linkage Disequilibrium，LD）。直觉上，LD 意味着”知道了一个位点的等位体，就能在一定程度上推断附近位点的等位体”。

{{Wiki:Linkage_disequilibrium}}

LD 区块结构示意：

染色体：  |═══════════════════════════════════|
区块：    [Block A ████]  [Block B ████████]  [Block C ██]
特点：     区块内 SNP 高度相关；区块间相对独立

LD 对 GWAS 有两个关键实践含义：

  • 无需对每一个 SNP 都进行基因分型：用少数”标签 SNP”代表整个区块，通过基因型插补（Genotype Imputation）推断区块内其他位点的基因型，大幅降低成本。这是 GWAS 芯片能以数十万个 SNP 覆盖全基因组数百万个变异位点的根本原因。
  • GWAS 检测到的关联位点不一定是因果变异：命中位点可能只是与真正致病变异处于高 LD 的”搭便车者”；从 GWAS 信号定位到真正因果变异（Fine-mapping）是后续分析的核心挑战。

LD 的量化（\(r^2\) 系数）、基因型插补的算法原理，以及在 GWAS 精细定位中的应用，将在 GWAS 章节中展开。

15.4 Hardy-Weinberg 平衡：群体遗传学的基准状态

在深入讨论频率差异和自然选择之前，需要先建立一个理想基准：在没有任何干扰因素的情况下，一个群体的基因型频率应该满足什么规律？这个基准就是 Hardy-Weinberg 平衡（Hardy-Weinberg Equilibrium，HWE），由英国数学家 Godfrey Hardy 和德国医生 Wilhelm Weinberg 分别于 1908 年独立提出。

{{Wiki:Hardy–Weinberg_principle}}

15.4.1 六个理想假设

HWE 成立需满足以下六个假设，现实中这些假设几乎从不同时成立，这正是 HWE 作为诊断工具的价值所在：

假设 现实中的偏差来源
随机交配 配偶选择偏好、地理隔离、近亲交配
无突变 碱基突变率约 \(10^{-8}\)/代，微小但不为零
无自然选择 几乎所有蛋白编码位点都受不同程度的纯化选择
无遗传漂变 有限种群大小导致随机频率波动
无基因流动 迁徙将其他群体的等位体引入
无限大种群 人类 \(N_e \approx 10{,}000\)，远非无限大

15.4.2 频率计算示例

设某二等位位点中，A 等位体频率为 \(p\)，a 等位体频率为 \(q = 1 - p\)。在 HWE 下，三种基因型的期望频率由以下公式给出：

\[P(\text{AA}) = p^2, \quad P(\text{Aa}) = 2pq, \quad P(\text{aa}) = q^2\]

这个公式的直觉来自有放回随机抽取两个等位体：两次都抽到 A 的概率是 \(p \times p = p^2\)；一次 A 一次 a（顺序不限）的概率是 \(2 \times p \times q = 2pq\)。

# R: HWE 频率计算与检验（手工演示）
hwe_expected <- function(p, n_total) {
  q <- 1 - p
  data.frame(
    genotype   = c("AA (hom_ref)", "Aa (het)", "aa (hom_alt)"),
    expected_freq  = round(c(p^2, 2*p*q, q^2), 4),
    expected_count = round(c(p^2, 2*p*q, q^2) * n_total, 1)
  )
}

# 示例：p(A) = 0.7，1000 个个体
cat("=== HWE Expected Genotype Frequencies ===\n")
print(hwe_expected(p = 0.7, n_total = 1000))

# 卡方检验判断是否偏离 HWE
hwe_chisq <- function(obs_AA, obs_Aa, obs_aa) {
  n <- obs_AA + obs_Aa + obs_aa
  p <- (2 * obs_AA + obs_Aa) / (2 * n)
  q <- 1 - p
  exp_AA <- n * p^2
  exp_Aa <- n * 2 * p * q
  exp_aa <- n * q^2
  chi2 <- (obs_AA - exp_AA)^2 / exp_AA +
           (obs_Aa - exp_Aa)^2 / exp_Aa +
           (obs_aa - exp_aa)^2 / exp_aa
  pval <- pchisq(chi2, df = 1, lower.tail = FALSE)
  cat(sprintf("Observed  : AA=%d  Aa=%d  aa=%d\n", obs_AA, obs_Aa, obs_aa))
  cat(sprintf("Expected  : AA=%.1f  Aa=%.1f  aa=%.1f\n", exp_AA, exp_Aa, exp_aa))
  cat(sprintf("Chi² = %.3f  p = %.2e  → %s\n", chi2, pval,
              ifelse(pval < 1e-4, "FAIL HWE (possible genotyping error)",
                     "PASS HWE")))
}

cat("\n--- 正常位点 (接近 HWE) ---\n")
hwe_chisq(obs_AA = 492, obs_Aa = 427, obs_aa = 81)

cat("\n--- 杂合子严重缺失 (疑似等位基因丢失) ---\n")
hwe_chisq(obs_AA = 650, obs_Aa = 50, obs_aa = 300)

15.4.3 HWE 偏离的诊断价值

在变异质控流程中，HWE 偏离有着明确的技术含义：

偏离方向 表现 最可能原因
杂合子缺乏（\(H_\text{obs} < H_\text{exp}\)） 纯合子 AA、aa 偏多 等位基因丢失（allele dropout）、芯片探针失效、近亲交配
杂合子过多（\(H_\text{obs} > H_\text{exp}\)） 杂合子 Aa 偏多 样本混合（两个个体被错误地合并为一）、体细胞嵌合

HWE 检验的合理使用范围

HWE 过滤只适用于对照组，不适用于病例组——某些真实致病变异在患者中本就偏离 HWE（例如隐性遗传病中 aa 纯合子富集于病例）。GWAS QC 流程中通常只对对照组执行 HWE 过滤（阈值 \(p < 10^{-6}\)），以避免错误地将真实生物学信号当作基因分型错误剔除。

15.5 单倍型：等位体的组合遗传

前面的讨论一直将每个位点当作独立个体来看待。但真实基因组中，同一条染色体上的等位体并不是独立分配的——它们以整体形式遗传，直到重组事件将其打散。这种同一染色体上特定等位体的固定组合，称为单倍型（Haplotype）。

{{Wiki:Haplotype}}

15.5.1 单倍型与基因型的区别

设某染色体区段上有两个 SNP 位点（位点 1：A/a，位点 2：B/b），一个二倍体个体在这个区段上有两条染色体。基因型只告诉我们每个位点上有什么等位体，但不告诉我们它们是如何组合的：

基因型表示（无法区分相位）：
  个体甲：位点1 = Aa，位点2 = Bb

可能的两种相位：
  相位1（AB | ab）：一条染色体带 AB，另一条带 ab  →  AB / ab
  相位2（Ab | aB）：一条染色体带 Ab，另一条带 aB  →  Ab / aB

这两种相位虽然基因型完全相同，但单倍型截然不同！

相位推断（Phasing） 是从基因型数据中确定单倍型组合的计算过程，常用工具包括 SHAPEIT 和 EAGLE。准确的相位信息是基因型插补和 LD 分析的基础。

15.5.2 单倍型在医学研究中的例子

HLA（Human Leukocyte Antigen，人类白细胞抗原）区域是人类基因组中多态性最丰富的区域，位于 6 号染色体短臂。HLA 区域内众多基因以单倍型形式遗传，不同 HLA 单倍型与特定自身免疫性疾病（如强直性脊柱炎、1 型糖尿病）的关联，是单倍型分析在疾病研究中最经典的应用案例。

15.6 遗传漂变：有限种群的随机效应

到目前为止，我们讨论的等位基因频率变化都有明确的方向性原因（选择、迁徙）。但即便没有任何方向性压力，等位体频率也会随时间随机波动——这一现象称为遗传漂变（Genetic Drift）。

{{Wiki:Genetic_drift}}

15.6.1 直觉理解：硬币翻转的类比

想象一个硬币正面频率恰好为 50% 的公平硬币，但每次只翻 20 次（对应有限种群大小）。由于采样误差，观测到的正面频率可能是 40%、60% 甚至 70%，而非精确的 50%。种群越小，这种随机波动越剧烈。

遗传漂变的本质正是这种有限采样引起的随机频率波动：每一代个体只是上一代个体的有限样本，等位体频率的随机上下波动在小种群中尤为突出。长期积累后，某些等位体会被随机固定（频率升至 100%），另一些会被随机丢失（频率降至 0%）——无关乎其是否有利。

15.6.2 种群大小与漂变强度

漂变强度与有效种群大小（Effective Population Size，\(N_e\)）成反比——\(N_e\) 越小，频率波动越剧烈：

\(N_e\) 量级 漂变强度 代表情况
数十 极强 濒危物种、实验室近交系
数百 强 岛屿隔离种群、人工育种群体
数千 中等 早期人类祖先（出非洲前）
数万 弱 现代人类全球平均（\(N_e \approx 10{,}000\)）

为什么人类的 \(N_e\) 远小于实际人口？

现代人口约 80 亿，但遗传学估算的 \(N_e\) 仅约 10,000。这是因为 \(N_e\) 反映的是历史上平均的有效交配群体规模：人类经历过多次严重的种群瓶颈（例如约 7 万年前的图巴火山爆发后，人类祖先据估计减少至数千人），这些历史瓶颈极大地降低了整体遗传多样性并留下了长期印记。\(N_e\) 是对整个进化历史上遗传漂变累积效应的综合度量，而非现代人口的简单反映。

15.6.3 种群瓶颈与奠基者效应

两种特殊的遗传漂变情形在医学遗传学中有重要应用：

种群瓶颈（Population Bottleneck）：种群规模在某一历史时期骤降，只有少数个体存活并成为后代祖先。瓶颈期仅有的几个个体的基因组截面决定了后代群体的多样性格局——大量等位体在瓶颈期随机丢失，而某些本来罕见的等位体可能因”幸存者偏差”意外富集。

奠基者效应（Founder Effect）：少数个体迁移到新地区并建立新种群，这少数奠基者携带的等位体决定了新种群的遗传组成。

奠基者效应的经典例子——芬兰病（Finnish Disease Heritage）：

芬兰人群由于历史上的隔离和数次人口扩张，
大约 36 种遗传病在芬兰人中的发病率显著高于其他欧洲人群，
而在其他欧洲人群中常见的遗传病在芬兰人中反而罕见。

原因：约 2000–4000 年前，少数奠基者迁入芬兰并快速扩张，
偶然携带某些罕见等位体的奠基者使这些等位体在后代中频率升高，
而奠基者未携带的等位体则在芬兰人中至今罕见。

这一规律对研究具有创始人效应的孤立人群（如阿什肯纳兹犹太人、冰岛人、萨丁尼亚人）有重要意义——这些人群中特定遗传病的研究往往更容易，因为致病变异频率更高、背景单倍型更纯净。

15.7 自然选择在基因组上的印记

遗传漂变是”随机”的等位体频率变化，而自然选择（Natural Selection）则是”定向”的：携带特定等位体的个体在生存和繁殖上更具优势或劣势，导致该等位体频率系统性地升高或降低。

{{Wiki:Natural_selection}}

15.7.1 三种基本选择类型

正向选择（Positive Selection）：某等位体赋予携带者适应性优势，频率在自然选择下快速升高。正选择的基因组学特征是：该等位体在某一人群中频率异常高，且周围区域的 LD 异常长（“选择清除”，Selective Sweep——快速升频的等位体来不及被重组打散）。

经典例子：成人乳糖酶基因（LCT 上游调控位点 rs4988235）。约 10,000 年前农业社会兴起，能消化牛奶的个体具有营养优势，这一位点的 T 等位体在欧洲牧业人群中迅速从罕见变为主流（北欧人群频率约 80%），在中国汉族中频率仍低于 1%。

纯化选择（Purifying Selection）：破坏重要蛋白质功能的等位体使携带者适合度降低，在群体中被持续清除。这是作用于功能基因的最常见选择形式。纯化选择解释了为什么严重功能性突变（无义突变、移码突变）在人群中极为罕见，以及为什么蛋白质编码区的多态性远少于非编码区。

经典例子：镰刀型细胞贫血症（SCD）的 HBB 基因 c.20A>T。纯合子（aa）患严重贫血，繁殖成功率极低，纯化选择不断清除 a 等位体；但在疟疾流行地区，杂合子（Aa）对疟疾有抵抗力，平衡选择维持了 a 等位体在非洲人群中约 10%–20% 的频率。

平衡选择（Balancing Selection）：维持两个或多个等位体在群体中长期共存，通常通过杂合子优势（如镰状细胞贫血）或频率依赖选择实现。平衡选择的基因组学特征是：某位点的多态性在多个物种之间长期保持（跨越物种隔离时间），提示该位点两个等位体都有功能价值。

15.7.2 选择对等位基因频率谱（AFS）的影响

不同选择类型会以可预测的方式改变 AFS 的形状，这是现代进化基因组学的重要推断工具：

中性（无选择）：    大量低频变异 + 少量高频变异 → L 形分布
正向选择扫荡后：   目标等位体快速升至高频 → 高频峰出现
纯化选择：         清除高影响突变 → 极低频峰过低
平衡选择：         维持中等频率 → 中频峰偏高
种群扩张后：       大量新突变涌现 → 极低频变异骤增
种群瓶颈后：       低频变异丢失 → L 形分布"左侧截断"

15.8 遗传多样性的基本度量

理解了影响等位体频率的各种力量后，需要一套数值指标来量化一个群体的遗传多样性水平。这些指标既是群体遗传学研究的基础输出，也是 GWAS 数据质控的重要参考。

15.8.1 核苷酸多样性 π

核苷酸多样性（Nucleotide Diversity，π）是最常用的群体遗传多样性指标。其含义是：从群体中随机抽取两条单倍型序列，平均每个碱基位置上存在差异的概率。

\[\pi = \frac{\text{两两比较的平均差异碱基数}}{\text{序列总长度}}\]

人类基因组的全基因组平均 \(\pi \approx 0.1\%\)（即每 1000 个碱基中约有 1 个 SNP 位点有差异）。不同物种和不同人群的 \(\pi\) 值差异显著：

物种 / 人群 \(\pi\) 近似值 含义
非洲人群（约鲁巴） ~0.12% 最高，非洲是人类发源地，积累了最多多样性
欧洲人群 ~0.09% 出非洲瓶颈后多样性有所降低
东亚人群 ~0.08% 与欧洲人群相近
果蝇（D. melanogaster） ~1.5% 远高于人类（有效群体大很多）
猎豹（A. jubatus） ~0.01% 极低（历史瓶颈极严重）

非洲人群的遗传多样性显著高于非非洲人群，这与”走出非洲”假说完全一致：现代人类起源于非洲，非非洲人群是约 5–7 万年前从非洲迁出的小群体的后代，迁出事件本身就是一次强烈的奠基者效应。

15.8.2 杂合度

期望杂合度（Expected Heterozygosity，\(H_e\)）是另一个常用指标，定义为在 HWE 下群体中随机个体为杂合基因型的概率：

\[H_e = 1 - \sum_{i} p_i^2\]

其中 \(p_i\) 是第 \(i\) 个等位体的频率。二等位位点时，\(H_e = 2pq\)。\(H_e\) 是衡量某个位点或某个基因区域多态性程度的常用指标，在 GWAS 位点选择和进化分析中广泛使用。

# R: 计算多个位点的杂合度，并可视化分布
compute_heterozygosity <- function(allele_freqs) {
  # allele_freqs: 每行一个位点，列为各等位体频率（行和为 1）
  if (is.vector(allele_freqs)) {
    1 - sum(allele_freqs^2)     # 单个位点
  } else {
    1 - rowSums(allele_freqs^2) # 多个位点
  }
}

)

15.9 等位基因频率谱（AFS）：群体历史的快照

将前面所有内容综合起来，便可以理解等位基因频率谱（Allele Frequency Spectrum，AFS）——它是群体中所有变异位点按 MAF 分组后得到的频率分布直方图，综合反映了群体历史中漂变、选择和人群规模变化的累积效果。

{{Wiki:Allele_frequency_spectrum}}

在中性进化（无选择）且种群规模稳定的理想情况下，理论预期 AFS 呈 L 形：极低频变异最多（绝大多数变异是近期新突变，尚未积累到高频），高频变异最少。对真实人类基因组 AFS 的偏离，可以推断出群体历史事件：

AFS 直方图的形状含义（示意）：

变异数量
  ↑
  │█
  │█  ← 大量低频变异（新突变 + 纯化选择清除后残留）
  │█ █
  │█ █
  │█ █ █
  │█ █ █ █ █ ██
  └────────────────→  MAF (0% → 50%)
     低频  中频  高频

若左侧（低频）极端偏高 → 种群近期快速扩张
若左侧明显截断          → 历史瓶颈丢失了低频变异
若中频峰出现            → 平衡选择维持中等频率等位体

AFS 是验证基因组数据质量的实用工具——若你的 WGS 变异集 AFS 形状异常（如高频区异常富集，或低频区完全缺失），往往提示测序深度不均匀、过度过滤或样本混合问题，而非真实的生物学信号。

15.10 突变：一切遗传变异的原始来源

前面所有关于等位体频率、漂变和选择的讨论都有一个共同的前提：变异从哪里来？答案只有一个——突变（Mutation）。没有突变，等位体频率无论如何漂移或被选择，最终都会因固定效应而趋向单态（全为参考等位体）；正是持续不断的新突变，为进化提供了取之不尽的原材料。

{{Wiki:Mutation}}

15.10.1 点突变的类型：转换与颠换

碱基替换（点突变）按化学结构可分为两类：

  • 转换（Transition，Ti）：嘌呤与嘌呤之间（A↔︎G）、或嘧啶与嘧啶之间（C↔︎T）的互换。化学结构相似，热力学上更容易发生。
  • 颠换（Transversion，Tv）：嘌呤与嘧啶之间的互换（A↔︎C、A↔︎T、G↔︎C、G↔︎T）。结构差异大，发生概率相对较低。

从排列组合看，4 种碱基共有 12 种可能的单碱基替换，其中 4 种为转换、8 种为颠换；若突变完全随机，Tv:Ti 比值应约为 2:1。然而，实际测量的人类基因组 Ti/Tv 比值约为 2.0–2.1，转换显著过多——原因是 CpG 二核苷酸的超突变性（见下文）以及转换在化学机制上更容易发生。

Ti/Tv 比值作为数据质量指标：

数据类型 期望 Ti/Tv 偏低的含义
全基因组测序（WGS） 2.0–2.1 假阳性（测序错误）过多，应检查过滤阈值
全外显子测序（WES） 2.8–3.0 编码区功能约束使转换富集
RNA-seq SNP calling 3.0–3.5 转录区选择压力更强

WES 的 Ti/Tv 更高，是因为外显子区域受到更强的纯化选择——颠换对蛋白质结构的破坏往往更大，被更积极地清除，留下来的多态性中转换比例更高。Ti/Tv 显著偏低（如 < 1.5）是变异检测质量不达标的警示信号，提示需重新审视过滤参数或测序质量。

15.10.2 CpG 二核苷酸的超突变性

人类基因组中，CpG 位点（5′-C-G-3′，胞嘧啶后紧跟鸟嘌呤）的甲基化修饰使得胞嘧啶极易自发脱氨基（C→T），导致 CpG 位点的 C→T 突变率约为基因组平均水平的 10–15 倍。这种超突变性（CpG Hypermutability）有几个重要推论：

  1. CpG 位点的 C→T 转换贡献了人类突变总量的约 25%，尽管 CpG 在基因组中只占约 1% 的位点
  2. 基因组中观察到的 CpG 频率远低于随机预期（“CpG 缺失”，CpG Suppression），正是因为高突变率持续消耗 CpG 位点
  3. CpG 岛（CpG Island）——通常位于基因启动子附近、甲基化受到保护的 CpG 富集区域——因未甲基化而维持了较高的 CpG 密度

15.10.3 从头突变率与遗传病负担

人类从头突变率（De Novo Mutation Rate）约为 \(1.0\)–\(1.5 \times 10^{-8}\) 每碱基每代，全基因组约 30 亿碱基，意味着每个个体相较于其父母，携带约 50–70 个新的点突变。这一数字看似不多，但：

  • 新突变在所有已知变异中占绝大多数（约 99% 的人类常见 SNP 历史超过数万年）
  • 父亲年龄与从头突变数量正相关（高龄父亲增加约 1–2 个额外突变/年）——这是一些罕见发育障碍（如孤独症谱系障碍）发病率与父亲年龄正相关的分子基础
  • 从头突变在亲子三代（trio）测序研究中可直接检测，是孟德尔遗传病因果鉴定的主要方法

15.11 基因流动：种群间的遗传连通

遗传漂变和自然选择在相互隔离的种群内各自运作，会使种群间的遗传差异不断积累。基因流动（Gene Flow）是对抗这一分化趋势的力量：当个体（或配子）在种群间迁移并成功繁殖时，外来等位体便进入目标种群，使两个种群的频率趋于均一。

{{Wiki:Gene_flow}}

从进化力量的框架来看，突变、漂变、选择和基因流动共同构成了改变等位体频率的四大机制：

四种进化力量及其方向性：

突变（Mutation）    →  产生新的等位体（随机，速率极低）
遗传漂变（Drift）  →  随机改变频率（在小种群中更强）
自然选择（Selection）→  定向改变频率（有利等位体升频）
基因流动（Gene Flow）→  使种群间频率趋于均一

15.11.1 Fst：量化种群间遗传分化

群体分化指数（Fixation Index，\(F_{ST}\)）是最常用的种群间遗传分化度量，其直觉含义是”等位体频率在不同种群间的差异占总多态性的比例”：

\[F_{ST} = \frac{H_T - H_S}{H_T}\]

其中 \(H_T\) 是将所有个体视为单一大群体时的期望杂合度，\(H_S\) 是各子群体期望杂合度的加权平均值。\(F_{ST}\) 介于 0（各群体频率相同，无分化）和 1（每个群体固定不同等位体，完全分化）之间。

\(F_{ST}\) 范围 解读
0–0.05 轻微分化（同一连续种群，频繁基因流动）
0.05–0.15 中等分化（不同地理区域的人类亚群）
0.15–0.25 较高分化（不同大陆人群间）
> 0.25 极高分化（不同物种间，或长期隔离种群）

人类不同大陆人群间的全基因组平均 \(F_{ST}\) 约为 0.10–0.15，远低于一般动物的种群分化水平，反映了人类历史上频繁的迁徙与混合。个别受到强烈正向选择的位点 \(F_{ST}\) 可接近 1——例如，控制北欧人肤色的 SLC24A5 rs1426654 位点，在欧洲人（几乎全为 A 等位体）和非洲/东亚人（几乎全为 G 等位体）之间的 \(F_{ST}\) 约为 0.99，是自然选择使种群极端分化的典型证据。

15.11.2 混合（Admixture）：多个祖先群体的基因组遗迹

当两个长期隔离的种群重新接触并相互通婚时，其后代基因组将由来自不同祖先种群的染色体片段镶嵌拼接而成——这称为遗传混合（Admixture）。现代基因组学工具（如 STRUCTURE、ADMIXTURE）通过分析数万个 SNP，可以将每个个体的基因组分解为来自不同”祖先成分”（Ancestry Component）的比例，绘制出直观的”条形图”（Bar Plot）。

混合分析在法医与临床中的应用

遗传混合分析不仅有进化学意义，在临床遗传学中也有实际价值：多基因风险评分（PRS）在构建训练集时所用的人群背景直接决定了其在其他人群中的预测精度。一个基于欧洲人群构建的 PRS 在高度混合的美洲人群（如拉丁美洲人）中效果大幅下降，其根本原因正是不同祖先成分在效应量估计上的系统性偏差。准确的混合比例估计是跨人群 PRS 迁移校正的第一步。

15.12 Wright-Fisher 模型：遗传漂变的数学骨架

“有限种群导致随机漂变”这一直觉，有着精确的数学表述——Wright-Fisher 模型（Wright-Fisher Model）。理解这个模型，可以从直觉的”硬币翻转类比”跃升到可计算的量化预测。

15.12.1 模型的核心假设

Wright-Fisher 模型将种群抽象为以下最简框架：

  • 种群规模固定为 \(N\) 个二倍体个体（即 \(2N\) 个等位体拷贝）
  • 每一代的 \(2N\) 个等位体，通过对上一代 \(2N\) 个等位体有放回随机抽样产生
  • 无突变、无选择、随机交配

在这一框架下，若当前代某等位体 \(A\) 的拷贝数为 \(k\)（频率 \(p = k/2N\)），则下一代 \(A\) 的拷贝数 \(k'\) 服从二项分布：

\[k' \sim \text{Binomial}(2N,\ p)\]

因此，等位体频率在相邻两代间的期望变化量为零（\(\mathbb{E}[\Delta p] = 0\)，漂变无方向），但方差为：

\[\text{Var}(\Delta p) = \frac{p(1-p)}{2N}\]

这个公式非常直观：\(p = 0.5\) 时方差最大（中间频率最不稳定）；\(N\) 越大，方差越小（漂变越弱）。

15.12.2 固定时间的估算

一个初始频率为 \(p_0\) 的等位体，在纯漂变下最终要么固定（频率 → 1）要么丢失（频率 → 0），其固定概率恰好等于其当前频率 \(p_0\)。平均固定时间约为：

\[\bar{t}_{\text{fix}} \approx 4 N_e \text{ 代}\]

对于人类（\(N_e \approx 10{,}000\)），一个中性新突变被固定的平均时间约为 4 万代（约 100 万年）。这解释了为什么很多古老的人类多态性（杂合 SNP）在全球不同人群中均有分布——它们在人类走出非洲之前已存在，时间足够长。

# R: Wright-Fisher drift simulation — multiple allele frequency trajectories
# Shows how frequency trajectories diverge more with smaller population size

wf_simulation <- function(N, p0, n_gen, n_rep) {
  # N: diploid population size (2N allele copies)
  # p0: initial allele frequency
  # n_gen: number of generations to simulate
  # n_rep: number of independent replicate trajectories
  trajectories <- matrix(NA_real_, nrow = n_gen + 1, ncol = n_rep)
  trajectories[1, ] <- p0

  for (rep in seq_len(n_rep)) {
    p <- p0
    for (gen in seq_len(n_gen)) {
      # binomial sampling: draw 2N alleles, count successes (A alleles)
      k  <- rbinom(1, size = 2 * N, prob = p)
      p  <- k / (2 * N)
      trajectories[gen + 1, rep] <- p
      if (p == 0 || p == 1) {
        trajectories[(gen + 1):nrow(trajectories), rep] <- p  # absorbing state
        break
      }
    }
  }
  trajectories
}

set.seed(2024)
n_gen <- 200
p0    <- 0.3

traj_large <- wf_simulation(N = 500, p0 = p0, n_gen = n_gen, n_rep = 20)

par(mfrow = c(1, 2), mar = c(4, 4, 3, 1))

                       p0 * (1 - p0) / (2 * 50)))
abline(h = c(0, 1), lty = 2, col = "gray50")

运行上述代码会看到一个清晰对比：小种群（N=50）中，各条频率轨迹在 50–100 代内迅速发散，许多轨迹达到 0 或 1（固定或丢失）；大种群（N=500）中，轨迹在初始频率附近波动，200 代内几乎无轨迹到达固定/丢失——漂变的速率与 \(N_e\) 成反比，直观而精确地体现了公式中的 \(1/2N\) 因子。

Wright-Fisher 模型的局限

现实种群不满足 WF 模型的几个关键假设：世代重叠、性别比例不等、个体繁殖成功率差异（reproduction variance）、以及交配非随机——这些都会使实际漂变强度偏离 \(N_e = N\) 的预测，这正是为什么我们需要引入”有效种群大小 \(N_e\)“这一概念作为等效的 WF 种群规模：\(N_e\) 是与实际种群产生相同漂变强度的理想 WF 种群大小。

15.13 Tajima’s D：中性进化偏离的统计检验

掌握了 AFS 的形状含义之后，一个自然的问题是：如何将这种”形状偏离”量化为一个可以进行统计检验的数字？Tajima’s D（由日本遗传学家 Fumio Tajima 于 1989 年提出）正是为此而设计的，它通过比较两种估算核苷酸多样性的方法之间的差异，将 AFS 的形状偏离压缩为单一统计量。

15.13.1 两种多样性估计量

Tajima’s D 比较的是同一数据的两种等价估计量：

Watterson’s \(\hat{\theta}_W\)（基于分离位点数）：

\[\hat{\theta}_W = \frac{S}{\sum_{i=1}^{n-1} \frac{1}{i}}\]

其中 \(S\) 为序列中观测到的分离位点（segregating sites）总数，\(n\) 为采样个体数。\(\hat{\theta}_W\) 对低频和高频变异同等权重，只关心”有多少位点是多态的”。

\(\hat{\pi}\)（基于两两平均差异数）：

即前文定义的核苷酸多样性，对每对序列做平均。\(\hat{\pi}\) 对中等频率的变异给予更高权重（中频等位体对两两比较贡献更大），对极低频/极高频变异权重相对低。

15.13.2 Tajima’s D 的计算与解读

\[D = \frac{\hat{\pi} - \hat{\theta}_W}{\sqrt{\text{Var}(\hat{\pi} - \hat{\theta}_W)}}\]

在中性进化（无选择、无人口规模变化）下，两种估计量在期望上相等，\(D\) 近似为 0。

Tajima’s D 值 AFS 特征 可能的进化解释
\(D \approx 0\) 标准 L 形，符合中性期望 中性进化，种群规模稳定
\(D < 0\)（显著） 低频变异过多（左侧偏高） 纯化选择（清除有害突变） 或 种群快速扩张（大量新突变涌现）
\(D > 0\)（显著） 中等频率变异过多 平衡选择（维持中等频率）或 种群瓶颈（低频变异丢失）

直觉解读：\(D\) 本质上是在问”你的低频变异相对于预期是太多还是太少”。

  • 如果 \(D < 0\)（低频变异过多），说明大量变异处于极低频——这与近期新突变爆发（种群扩张）或持续清除高频有害突变（纯化选择）的预期一致
  • 如果 \(D > 0\)（中频变异过多），说明变异在群体中维持在中等频率——这是平衡选择（两个等位体都有优势）或历史瓶颈（低频变异被随机清除）的信号

Tajima’s D 的解读需要结合背景

\(D\) 的显著偏离可以同时由选择和人口历史事件（扩张/瓶颈）解释——两者对 AFS 的影响方向相同，仅凭 \(D\) 无法区分。实践中，通常先估算全基因组平均 \(D\) 作为人口历史基线，再识别显著偏离基线的局部基因组区域作为选择信号候选。此外，\(D\) 对采样深度（\(n\)）敏感，不同研究间的比较须谨慎。

15.13.3 经典案例

HBB 基因区域（非洲人群，疟疾流行区）：包含镰状细胞贫血症突变（c.20A>T）的区域，由于平衡选择使杂合子优势维持，\(D\) 值显著为正（约 +2.5 至 +4.0）——中等频率等位体远多于中性期望，是平衡选择最有力的基因组学证据之一。

MHC（主要组织相容性复合体）区域：人类基因组中多态性最高的区域，长期平衡选择（免疫系统需要多样性识别不同病原体）使该区域整体 \(D\) 显著为正。

蛋白质编码区全基因组平均：由于纯化选择持续清除有害突变，编码区整体 \(D\) 值略为负值（约 −0.3 至 −0.5），而非编码区则更接近 0，这一差异本身便是纯化选择在编码区发挥作用的间接证据。

# R: conceptual demonstration of Tajima's D
# We compare pi vs theta_W under different evolutionary scenarios

tajima_demo <- function(freq_vec, n_seq) {
  # freq_vec: derived allele frequencies of segregating sites
  # n_seq: number of sequences sampled
  S <- length(freq_vec)
  if (S == 0) return(list(pi = 0, theta_w = 0, D_approx = NA))

  # pi: mean pairwise differences (weighted by frequency)
  # For a site with derived freq p, contribution to pi = 2p(1-p) * n/(n-1)
  pi_hat <- mean(2 * freq_vec * (1 - freq_vec) * n_seq / (n_seq - 1))

  # Watterson's theta_W
  a1 <- sum(1 / seq_len(n_seq - 1))
  theta_w <- S / (a1 * 1)   # per-site theta_w requires knowing sequence length

  list(pi = round(pi_hat, 4),
       theta_w = round(theta_w / 1000, 4),  # rough per-site scaling
       ratio = round(pi_hat / (theta_w / 1000), 3))
}

# Scenario 1: neutral AFS (many rare variants, L-shape)
neutral_afs <- c(0.02, 0.03, 0.02, 0.05, 0.04, 0.01, 0.03, 0.02, 0.30, 0.45)

# Scenario 2: balancing selection (elevated intermediate frequencies)
balancing_afs <- c(0.35, 0.40, 0.42, 0.38, 0.45, 0.50, 0.36, 0.43)

# Scenario 3: purifying selection / expansion (excess rare variants)
purifying_afs <- c(0.005, 0.008, 0.003, 0.006, 0.002, 0.004, 0.009, 0.007, 0.001, 0.003)

cat("=== Tajima's D conceptual comparison ===\n\n")
for (scenario in list(
    list(name = "Neutral / stable",    freqs = neutral_afs),
    list(name = "Balancing selection", freqs = balancing_afs),
    list(name = "Purifying / expansion", freqs = purifying_afs)
)) {
  res <- tajima_demo(scenario$freqs, n_seq = 50)
  cat(sprintf("Scenario: %-25s | pi = %.4f | pi/theta_W ratio = %.3f\n",
              scenario$name, res$pi, res$ratio))
  cat(sprintf("  → High pi/theta_W (>1) implies D > 0; Low (<1) implies D < 0\n\n"))
}

  1. 全球 2% 的频率属于常见多态性，ACMG 的 BS1 标准（等位频率高于疾病预期发病率）本应降低致病评级；但东亚人群仅 0.05% 的频率说明该变异在高风险群体中仍属罕见，频率差异本身具有群体特异性。不能仅凭全球频率否定致病性，应分人群查看频率，结合 ClinVar 具体表型记录、家系共分离证据和功能实验综合评判。ACMG 框架要求多证据集成，单一指标不可决定最终分级。

  2. BQSR 的核心假设是：已知变异位点处的碱基差异归因于真实多态性而非测序错误，排除后的差异才用于统计错误率、训练校正模型。没有数据库时的策略：(1) 先宽松检测，将高置信变异集作为临时替代库（自举法）；(2) 跳过 BQSR，直接使用原始质量分；(3) 用全局均值代替位点特异性校正。上述方法均有局限，非模式生物中 BQSR 收益通常远小于人类基因组。

  5. 群体分层产生假阳性的根本原因是：不同祖先人群在全基因组范围内系统性地存在频率差异，这些差异与目标疾病无关，但当病例与对照祖先不均衡时，大量无关位点也会在频率上呈现病例 vs 对照的差异，被统计模型误判为关联。这与变异检测质量无关——即使每个变异的基因型都是高质量的，只要祖先背景存在偏差，假阳性问题就必然存在。解决方案是在研究设计阶段匹配祖先背景，或在分析阶段加入 PCA 协变量。

  6. 连锁不平衡意味着基因组可被划分为若干 LD 区块，同一区块内的 SNP 因历史上罕见的重组而倾向于以固定组合（单倍型）共同遗传，彼此高度相关（r² 接近 1）。每个区块只需基因分型少数几个”标签 SNP”，其余 SNP 的基因型就可通过 LD 关系和参考面板（如 1000 Genomes）推断得到（基因型插补）。人类常见变异约分布在 5 万个 LD 区块中，50 万标签 SNP 足以代表其中绝大多数区块，因此芯片数据经插补后实际覆盖的变异数量可扩展至数千万级别。
