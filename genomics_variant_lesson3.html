17  GWAS 统计模型

17.1 学习目标

完成本专题学习后，学生应能够：

  1. 描述 GWAS 标准质量控制（QC）流程的各个步骤及其生物学依据；
  2. 理解并比较卡方检验、逻辑回归与线性回归三类关联检验的适用条件；
  3. 阐释混合线性模型（Mixed Linear Model，MLM）如何控制群体分层与亲缘关系对关联分析的影响；

17.2 全基因组关联分析的基本逻辑

17.2.1 GWAS 的历史背景与范式转变

要理解 GWAS 的意义，必须将其置于分子标记技术几十年的演进历程之中。

第一阶段：限制性片段长度多态性与分子标记辅助选择（1980–1990 年代）

1980 年，Botstein 等人提出利用限制性片段长度多态性（Restriction Fragment Length Polymorphism，RFLP） 作为遗传标记绘制人类遗传图谱，开创了基于 DNA 多态性进行遗传定位的先河。RFLP 的原理是：不同个体在特定限制性内切酶识别位点附近的 DNA 序列存在差异，导致酶切片段长度不同，可通过 Southern 杂交检测。在农业领域，RFLP 和随后出现的 SSR（Simple Sequence Repeat，简单重复序列）、AFLP 等标记被大规模应用于 QTL 定位（Quantitative Trait Loci mapping）——在人工设计的分离群体（\(F_2\)、回交群体、重组近交系等）中，借助标记与性状的连锁关系，将控制数量性状的染色体区间定位至数十 cM 的范围。

在此基础上发展起来的分子标记辅助选择（Marker-Assisted Selection，MAS），是将分子标记信息直接用于育种决策的技术体系：选择与目标 QTL 紧密连锁的标记，在育种早期代就能预测后代是否携带目标等位基因，从而大幅压缩育种周期。MAS 在抗病基因导入、品质性状改良等领域取得了显著成效，至今仍是作物和畜禽育种的核心工具之一。然而，传统 QTL 定位与 MAS 存在根本局限：其一，分离群体必须人工构建，在人类疾病研究中无法实施；其二，定位精度粗糙，一个 QTL 区间往往包含数十至数百个基因，难以确定因果基因；其三，所用标记数量有限（数百至数千个），无法覆盖全基因组的精细变异信息。

第二阶段：候选基因研究与连锁分析（1990–2000 年代初）

随着人类基因组计划的推进，研究者获得了参考序列与大量已知基因的功能注释，候选基因研究（candidate gene study） 应运而生：依据对疾病生物学机制的先验认识，选取少量”可疑”基因的 SNP 进行靶向基因分型，直接检验其与疾病的关联。这一策略在少数生物学机制明确的疾病中取得了成功（如载脂蛋白 E APOE ε4 等位基因与阿尔茨海默病的关联），但其致命弱点在于：若真正的致病基因不在研究者的先验假设范围内，整个研究将一无所获，且大量基于先验假设的阳性结果在后续研究中无法重复。

与此同时，连锁分析（linkage analysis） 利用多代家系样本，寻找与疾病在家系内部共同遗传的染色体区段。连锁分析在孟德尔单基因疾病的基因定位中功勋卓著（如亨廷顿病、囊性纤维化、BRCA1/2），但对于多因素复杂疾病，连锁分析的分辨率和功效都十分有限——其定位精度通常只能达到数十 Mb 的范围，而且绝大多数常见复杂疾病中单个位点的效应量太小，家系中的共遗传信号极为微弱，几乎无法被连锁分析检出。

第三阶段：GWAS 的范式革命（2005 年至今）

GWAS 的出现标志着一次根本性的范式转变：不再依赖先验假设，不再需要构建特殊的分离群体，而是直接在自然人群中以无偏（hypothesis-free）的方式对整个基因组进行系统扫描。这一转变在技术上依赖三项关键进步：其一，SNP 芯片（SNP genotyping array） 的发展，单张芯片可同时对 50 万至 500 万个 SNP 进行基因分型，成本从最初的数千美元降至数十美元；其二，人类基因组单倍型图谱计划（HapMap Project） 绘制出了人类基因组的连锁不平衡结构，揭示了标记 SNP 与邻近未测位点之间的代理关系，使稀疏分布的芯片 SNP 得以”捕捉”更多遗传变异信息；其三，大规模流行病学队列 的建立提供了必要的样本量基础。与传统 QTL 定位相比，GWAS 直接在自然群体中利用历史重组事件积累的精细 LD 结构来定位关联位点，理论分辨率可达 kb 级，远优于家系连锁分析的数十 Mb。

2007 年，由 Wellcome Trust Case Control Consortium（WTCCC）发表的里程碑式研究，同时对 7 种常见疾病进行了全基因组扫描，共鉴定出 24 个独立关联区域，并首次系统性地展示了 GWAS 全流程的标准化框架，奠定了现代 GWAS 的方法论基础。

17.2.2 常见病-常见变异假说与遗传度缺失

GWAS 的理论基础之一是常见病-常见变异（Common Disease-Common Variant，CD-CV）假说：常见复杂疾病的遗传基础主要由群体中频率较高（MAF > 1%）但效应量较小的遗传变异共同决定，而非由少数频率极低、效应量极大的罕见变异所主导。这一假说支持了使用 SNP 芯片进行大规模人群筛查的策略——芯片上密集分布的标签 SNP（tag SNP）能够通过 LD 捕捉到其附近的因果变异，即使因果变异本身并未直接测定。

然而，随着研究的深入，这一假说面临挑战。大量 GWAS 成果揭示了一个令人困惑的现象：已发现的位点虽然众多，但其效应量普遍偏小（OR 通常在 1.05 到 1.3 之间），累加起来所能解释的遗传度远低于双胞胎研究估计的总遗传度，这被称为“遗传度缺失”（missing heritability） 问题。以身高为例，双胞胎研究估计其遗传度约为 80%，但早期 GWAS 发现的位点合计仅能解释不到 10% 的表型方差。这一缺口的来源目前已逐渐明朗：大量效应量极小（\(\beta < 0.05\) SD）的变异未达到显著性阈值但真实存在；罕见变异的贡献未被标准芯片捕捉；以及基因-基因交互和基因-环境交互效应的贡献。

这些发现促使研究者转向更大的样本量（如英国生物银行，UK Biobank，约 50 万样本）和对罕见变异的全基因组测序（WGS）研究，形成了 GWAS 与 WGS 互补的格局。

17.2.3 GWAS 的核心逻辑框架

全基因组关联分析的核心问题是：基因组上的哪些位点的遗传变异与目标性状存在统计关联？

其逻辑框架可以分解为三个层次：

  1. 遗传标记的系统筛查：在全基因组范围内，对数十万至数百万个 SNP 位点逐一进行统计检验；
  2. 关联强度的量化：以 p 值、效应量（odds ratio 或 beta 系数）及其置信区间刻画每个位点的关联程度；
  3. 混杂因素的控制：群体分层（population stratification）、亲缘关系（relatedness）和基因型测定错误是导致假阳性的主要来源，必须在分析前或分析中予以控制。

关键概念：GWAS 的统计假设

每次单 SNP 检验的零假设为：该 SNP 的基因型与目标性状在群体中无关联（即等位基因频率在表型组间无差异，或回归系数为零）。GWAS 在同一数据集上同时进行成千上万次独立检验，由此引入严重的多重比较问题（参见多重检验校正一节）。

GWAS 的分析结果通常以曼哈顿图（Manhattan plot） 呈现：横轴按染色体顺序排列所有 SNP 的基因组坐标，纵轴为关联检验的 \(-\log_{10}(p)\) 值。超过基因组显著性阈值（\(5 \times 10^{-8}\)，红色虚线）的 SNP 位点被视为候选关联位点。

17.3 连锁不平衡与单倍型区块

连锁不平衡（Linkage Disequilibrium，LD）是指基因组上两个或多个位点的等位基因在群体中并非随机组合——某些等位基因组合出现的频率高于或低于随机期望。LD 的存在是 GWAS 芯片设计和信号解读的生物学基础：芯片无需对每一个变异位点进行直接基因分型，而只需选取具有代表性的标签 SNP（tag SNP），借助 LD 结构间接捕捉周边的未测变异信息。然而，LD 同时也是 GWAS 结果精细定位（fine-mapping）和跨群体迁移的最大障碍。理解 LD 的数学定义、成因与群体差异，对于正确解读 GWAS 结果至关重要。

17.3.1 LD 的数学定义与度量

设基因组上存在两个双等位基因 SNP，位点 A 的等位基因为 \(A/a\)（频率分别为 \(p_A\)、\(q_A = 1 - p_A\)），位点 B 的等位基因为 \(B/b\)（频率分别为 \(p_B\)、\(q_B = 1 - p_B\)）。若两位点处于完全连锁平衡（Linkage Equilibrium），则单倍型 \(AB\) 的群体频率应等于 \(p_A \cdot p_B\)。LD 系数 \(D\) 定义为观测单倍型频率与随机期望频率之差：

\[ D = p_{AB} - p_A \cdot p_B \]

\(D > 0\) 表示等位基因 \(A\) 和 \(B\) 倾向于共同出现在同一条染色体上；\(D < 0\) 则表示 \(A\) 与 \(b\)、\(a\) 与 \(B\) 更倾向于共现。\(D\) 的绝对值受各等位基因频率限制，取值范围为 \([D_{\min},\, D_{\max}]\)，不同位点之间不可直接比较，因此需要归一化处理。

归一化 LD 系数 \(D'\) 将 \(D\) 除以其理论最大值：

\[ D' = \begin{cases} D / D_{\max} & \text{若 } D > 0 \\ D / |D_{\min}| & \text{若 } D < 0 \end{cases} \]

其中 \(D_{\max} = \min(p_A q_B,\, q_A p_B)\)，\(D_{\min} = -\min(p_A p_B,\, q_A q_B)\)。\(D' = 1\) 意味着两位点之间从未发生历史重组（即两位点自共同祖先以来始终保持在同一单倍型上），是”完全 LD”的历史遗传学标志；\(D' < 1\) 则说明存在至少一次历史重组事件将原始单倍型打断。

然而，\(D' = 1\) 并不意味着等位基因频率对齐——当两位点的等位基因频率不对等（如 \(p_A \gg p_B\)）时，即便 \(D' = 1\)，也只有少数样本能从位点 B 的基因型推断位点 A 的基因型。为解决这一问题，GWAS 中更广泛使用相关系数平方 \(r^2\)：

\[ r^2 = \frac{D^2}{p_A q_A \cdot p_B q_B} \]

\(r^2\) 的取值范围为 \([0, 1]\)，其统计意义直接对应线性回归框架中两变量的决定系数：位点 B 的基因型能解释位点 A 基因型总变异的 \(r^2\) 比例。对于 GWAS 而言，\(r^2\) 的关键作用体现在统计检验力的缩放上：若芯片上测量的是标签 SNP（tag SNP），其与真实因果位点（causal SNP）的 \(r^2 = \rho^2\)，则标签 SNP 关联检验的非中心参数（Non-Centrality Parameter，NCP）为：

\[ \text{NCP}_\text{tag} = r^2 \cdot \text{NCP}_\text{causal} \]

这意味着即便 \(r^2 = 0.8\)，标签 SNP 的有效统计检验力也仅相当于直接测量因果位点的 80%，因此 GWAS 芯片设计时通常要求覆盖区域内所有常见变异的标签 \(r^2 > 0.8\)。

重要概念：\(D'\) 与 \(r^2\) 的区别

\(D' = 1\) 是历史重组的检验工具，说明两位点”没有被重组打断过”；\(r^2 = 1\) 则要求两位点等位基因频率完全对等，在人群中可以相互完美预测。后者在实际数据中极为罕见；前者在距离较近的位点对中较为常见。GWAS 中优先使用 \(r^2\)，因为它直接决定标签效率和检验力，而 \(D'\) 更多用于精细定位和历史重组推断。

17.3.2 LD 的成因与衰减

LD 的根本来源是共同祖先（common ancestry）：两个位点如果同时出现在同一条古老染色体上，其等位基因组合就会传递给后代。随着世代的积累，重组事件会以概率 \(c\)（重组率）将两位点分开，每一代 LD 的衰减满足：

\[ D_t = D_0 \cdot (1 - c)^t \]

其中 \(t\) 为世代数，\(c\) 为两位点之间的重组率（对于物理距离约 1 Mb 以上的位点，\(c \approx 0.5\)，LD 迅速消失；对于相距 10 kb 内的位点，\(c\) 很小，LD 衰减极慢）。这一公式揭示了 LD 随物理距离增加而衰减的普遍规律。

{{Wiki:Linkage_disequilibrium}}

不同人群的 LD 衰减速率存在显著差异，这是 GWAS 跨群体分析中最重要的群体遗传学背景之一：

  • 非洲人群：由于现代人类起源于非洲，非洲人群历史最为古老，有效群体规模（effective population size，\(N_e\)）在整个演化史上维持较大，积累了最多的历史重组事件。因此，非洲人群的 LD 衰减速率最快，LD 区块平均长度约为 5–15 kb。
  • 欧洲和东亚人群：现代人类走出非洲后经历了奠基者效应（founder effect）和遗传瓶颈（bottleneck），有效群体规模大幅缩小，积累的重组事件较少，LD 区块明显更长，平均长度约为 50–250 kb。

在物理距离相同的情况下，非洲人群中的 \(r^2\) 往往更低，这意味着要达到同等的变异覆盖密度，针对非洲人群的 GWAS 芯片需要更高的 SNP 密度。此外，重组热点（recombination hotspot）在基因组上的分布高度不均匀：约 60% 的重组事件集中在约 6% 的基因组序列中（即热点区域），这些热点将基因组切割为相对稳定的 LD 区块，热点两侧的 LD 会发生急剧下降，形成区块边界。与重组热点相对的是遗传漂变（genetic drift）：在小群体中，随机漂变效应会增强局部区域的 LD，有时产生与重组热点无关的高 LD 区域。

17.3.3 单倍型区块与标签 SNP

单倍型区块（haplotype block）的操作性定义最广泛采用的是 Gabriel 等人于 2002 年提出的方法：若某一连续基因组区间内任意两个 SNP 之间的 \(D' > 0.8\)（置信区间下界），且整个区间内高 LD SNP 对的比例超过一定阈值，则将该区间定义为一个单倍型区块。根据 HapMap 项目和 1000 Genomes Project 的分析，人类基因组中约 60–80% 的序列可归入单倍型区块，每个区块平均包含 5–10 个常见 SNP（MAF > 0.05），区块内部的遗传多样性通常可以用 3–5 条主要单倍型来描述（覆盖 > 90% 的染色体）。

标签 SNP 选择的目标是从每个区块中选取数量最少、但使区块内所有未测 SNP 的 \(r^2\) 覆盖最大化的代表性位点。当前主流 GWAS 芯片的标签效率如下：

  • Illumina Global Screening Array（约 65 万 SNP）：对欧洲人群常见变异（MAF > 0.05）的 \(r^2 > 0.8\) 覆盖率约为 90–92%；
  • Illumina Infinium GSA-24 v3（约 85 万 SNP）：对欧洲人群覆盖率提升至约 94–95%；
  • 对非洲人群：由于 LD 衰减快、区块更短，同等密度芯片的覆盖率通常下降至 70–80%，需要专门设计的高密度芯片（如 H3Africa Array）。

HapMap 项目（2002–2009 年，覆盖四个参考人群）首次系统绘制了人类基因组 LD 图谱，为第一代 GWAS 芯片的设计奠定了基础。1000 Genomes Project（2008–2015 年，覆盖 26 个群体）进一步精化了非欧洲人群的 LD 结构，为芯片密度提升和跨群体分析提供了更精确的参考面板。

17.3.4 LD 对 GWAS 的影响

LD 对 GWAS 的影响具有双刃剑属性。就优势而言，LD 使得芯片上数十万个标签 SNP 能够代理基因组中数百万个未被直接测量的常见变异，是 GWAS 方法在经济上可行的根本原因。就劣势而言，LD 导致关联信号无法精确定位到真实的因果变异：在某一基因组区域检出的”lead SNP”（具有最强关联信号的 SNP）不一定是因果 SNP，它可能只是与若干候选因果变异均处于高 LD 状态的代理位点。

LD 还直接影响多重检验校正的有效性。GWAS 通常对数百万个 SNP 进行检验，理论上最严格的 Bonferroni 校正应将 \(p\) 值阈值设为 \(0.05 / M\)（\(M\) 为总 SNP 数）。然而，由于 LD 使相邻 SNP 之间存在强相关，实际独立检验数 \(M_\text{eff}\) 远小于 \(M\)。经验研究表明，对于覆盖整个人类基因组的百万级 SNP 芯片，\(M_\text{eff}\) 约为 100 万，因此 \(p < 5 \times 10^{-8}\) 的经验阈值（对应 \(0.05 / 10^6\)）被广泛用作 GWAS 全基因组显著性标准，而非真正的 Bonferroni 校正值。

跨群体 PRS 迁移的根本障碍

在欧洲人群中进行 GWAS 发现的 lead SNP，其在东亚或非洲人群中与因果变异的 LD 结构可能完全不同：同一 lead SNP 在欧洲人群中 \(r^2_\text{causal} = 0.9\)，但在非洲人群中可能降至 \(r^2_\text{causal} = 0.2\)，导致其关联信号强度大幅衰减，甚至消失。这是当前多基因风险评分（Polygenic Risk Score，PRS）跨群体预测精度下降的根本原因之一，也是推动在多样化人群中开展大规模 GWAS 的科学动机。

17.4 GWAS 质量控制

GWAS 数据的质量控制（Quality Control，QC）是整个分析流程中不可或缺的第一步，其目的是在统计检验之前系统性地识别并移除两类误差来源：其一是技术性误差，包括基因型判定失败、样本污染、芯片批次效应等导致的非随机测量错误；其二是生物学混杂，包括近亲样本的纳入和群体分层的存在。若跳过 QC 直接进行关联分析，上述误差将以假阳性（虚假关联）或假阴性（真实信号被噪声淹没）的形式扭曲最终结果，且这种扭曲往往无法在后期统计校正阶段被完全消除。QC 分为两个层次：样本水平 QC 旨在识别低质量或不合适的个体；位点水平 QC 旨在过滤不可靠的基因型位点。两者的执行顺序通常是先样本后位点，以避免少数劣质样本对位点统计量的计算产生影响。

17.4.1 样本水平 QC

样本基因型缺失率（Call Rate）

样本 call rate 定义为某一样本在全部 SNP 中成功基因分型的比例。低 call rate 样本（通常阈值为 < 0.95）反映该样本的 DNA 质量差或上样量不足，其基因型数据中随机缺失比例过高，会导致后续所有统计量的估计偏差增大。PLINK 通过 --mind 参数实施该过滤，删除缺失率超过阈值的样本。

杂合率（Heterozygosity Rate）

样本杂合率 \(H_\text{obs}\) 衡量的是某个体在全基因组范围内处于杂合状态（即两条染色单体携带不同等位基因）的 SNP 比例。在一个遗传均质的群体中，每个个体的 \(H_\text{obs}\) 应围绕群体均值呈正态分布，偏离均值过大（通常以 ±3 SD 为阈值）提示以下两种异常情形：

  • \(H_\text{obs}\) 异常偏低：提示样本近亲繁殖（inbreeding）或 DNA 污染导致的等位基因脱扣（allele dropout），后者常见于 DNA 降解样本；在芯片实验中，低杂合率也可能源于单倍型纯合区段（runs of homozygosity，ROH）异常增多。
  • \(H_\text{obs}\) 异常偏高：最常见的原因是样本污染（sample contamination）——两个不同个体的 DNA 混合后，同一位点上来自两个个体的等位基因都会被检测到，导致大量位点呈现”杂合”状态；此外，染色体非整倍体（如三体综合征）也会引起特定染色体上的杂合率升高。

需要注意，杂合率过滤应在样本 call rate 过滤之后进行。call rate 极低的样本本身就会使 \(H_\text{obs}\) 的估计不稳定，先删除这类样本可以使杂合率的群体分布更为可靠，从而令 ±3 SD 阈值更加准确。

性别核实（Sex Check）

性别核实通过比对 X 染色体的杂合率与样本元数据中记录的生物学性别来识别性别标注错误或样本混淆。其原理基于 X 染色体的遗传特性：正常女性（46,XX）在 X 染色体上的 SNP 大多处于杂合状态，X 染色体杂合率通常 > 0.2；正常男性（46,XY）仅有一条 X 染色体，理论上 X 染色体上所有 SNP 均为半合子（hemizygous），观测杂合率应接近 0（通常 < 0.2）。PLINK 计算 X 染色体上的近交系数 \(F_X\)：男性样本的 \(F_X\) 应接近 1，女性样本应接近 0。若某样本记录性别为男性但 \(F_X\) 偏低，或记录为女性但 \(F_X\) 偏高，则应标记为性别不一致，予以删除或审查。性别核实不仅能发现样本标注错误，也能检出 Klinefelter 综合征（47,XXY）等性染色体异常个体。

亲缘关系过滤（IBD Filtering）

在介绍亲缘关系过滤之前，需要先理解同源同祖（Identity by Descent，IBD） 这一核心遗传学概念。若两个个体在某位点携带的等位基因来自同一个共同祖先个体的同一条染色体拷贝，则称这两个等位基因是”同源同祖”的。这与同态同祖（Identity by State，IBS） 不同：两个等位基因可能碱基相同（IBS），但各自独立演化而来（非 IBD）；也可能真正来源于共同祖先（既 IBD 又 IBS）。在群体遗传学中，IBD 比例 \(\hat{\pi}\)（也称 pi-hat）是衡量两个个体之间亲缘关系程度的全基因组统计量，通过在大量 SNP 上估计共享等位基因来自同一祖先的概率得到。

GWAS 的关联检验假设样本之间相互独立。若数据集中包含家系成员（如兄弟姐妹、父母子女）或近亲个体，其表型和基因型的相关性将违背独立性假设，导致关联检验的 I 类错误膨胀。PLINK 通过估计样本对之间的同源同祖比例（Proportion of Identity-by-Descent，\(\hat{\pi}\)） 来量化亲缘关系：

\(\hat{\pi}\) 范围 亲缘关系
≈ 1.0 同卵双胞胎 / 重复样本
≈ 0.5 一级亲属（父母-子女、全同胞）
≈ 0.25 二级亲属（祖孙、半同胞）
≈ 0.125 三级亲属（表兄弟等）
< 0.05 无关个体

标准过滤阈值为 \(\hat{\pi} > 0.125\)：对每一对超过阈值的样本，删除其中 call rate 较低的那个，以最大程度地保留样本量。IBD 估计必须在 LD 剪枝后的独立 SNP 子集上进行，因为处于 LD 中的相邻 SNP 会人为抬高 \(\hat{\pi}\) 的估计值。

指标 常用阈值
样本 call rate < 0.95 删除
杂合率（Heterozygosity rate） 均值 ± 3 SD 删除
性别核实 与记录不符删除
亲缘关系（IBD, \(\hat{\pi}\)） \(\hat{\pi} > 0.125\) 时每对中删除 call rate 低者

17.4.2 位点水平 QC

SNP 基因型缺失率

SNP call rate 衡量某一位点在全部样本中成功基因分型的比例。缺失率高的 SNP（> 5%）提示该位点的探针设计质量差、与基因组重复区域交叉或存在结构变异干扰，其基因型判定结果不可靠，应予以删除。

次等位基因频率过滤

MAF 极低的 SNP（通常 < 0.01 甚至 < 0.05）在统计上面临两重问题：其一，样本中携带该等位基因的个体数量极少，卡方检验或逻辑回归的期望频次过小，导致统计假设（如大样本近似）失效；其二，低频变异的基因型判定错误率相对较高，单个错误基因型对频率估计的影响被放大。因此，标准 GWAS 通常对 MAF < 0.01 的位点进行过滤，部分研究进一步采用 MAF ≥ 0.05 的严格标准。

哈迪-温伯格平衡检验

在一个满足 HWE 的随机交配群体中，基因型频率可由等位基因频率唯一确定：\(P(AA) = p^2\)，\(P(Aa) = 2pq\)，\(P(aa) = q^2\)。若某位点的观测基因型频率严重偏离 HWE 期望值（p < \(10^{-6}\)），通常意味着基因型判定存在系统性错误——例如，等位基因脱扣效应导致杂合子被错误判定为纯合子，使观测杂合基因型频率显著低于 HWE 预期。

HWE 过滤仅应用于对照组

若某位点真实与疾病关联，病例组的基因型频率本身就会偏离 HWE。对病例组实施 HWE 过滤会将真实关联位点错误删除，导致假阴性。

差异缺失检验（Differential Missingness）

差异缺失是一种容易被忽视但影响严重的 QC 指标。其原理是：若某 SNP 在病例组中的基因型缺失率显著高于对照组（或反之），则该 SNP 的缺失模式本身就与疾病状态相关。在关联分析中，这种差异缺失会人为产生虚假的统计关联，因为有效样本量的不对称分布等价于一种系统性偏倚。PLINK 通过 --test-missing 对每个 SNP 进行病例-对照缺失率的卡方检验，p < \(10^{-5}\) 的位点应予以删除。该步骤必须在样本水平 QC 完成后执行，以避免低质量样本的缺失模式干扰位点层面的差异检验。

指标 常用阈值
SNP call rate < 0.95 删除
MAF < 0.01 删除
Hardy-Weinberg 平衡（HWE） p < \(1 \times 10^{-6}\) 删除（仅对照组）
差异缺失 p < \(1 \times 10^{-5}\) 删除

17.4.3 批次效应与 QC 一致性

批次效应（batch effect）是指同一 SNP 在不同基因分型批次、芯片版本或实验室环境中表现出系统性的、可重复的差异，而这种差异与研究对象的真实遗传状态无关。批次效应是多中心 GWAS 和荟萃分析中最常见的系统性误差来源之一，若不加以识别和处理，会直接导致 QQ 图的膨胀（\(\lambda_{GC}\) 偏高）和曼哈顿图上出现聚集的假阳性峰。

批次效应的常见来源包括以下几类：不同批次的基因型判定（genotype calling）使用不同版本的芯片或 calling 算法（如 GenCall vs. zCall）；DNA 提取方法（如柱提法 vs. 磁珠法）和存储时间（DNA 降解程度不同）导致信号强度分布偏移；样本的上样日期和实验室操作员差异引起芯片信号的系统性漂移；多中心研究中不同实验室的实验室内部质量标准和 DNA 浓度要求不一致。上述来源均可在不同批次样本之间造成基因型判定结果的系统性偏差，进而使批次成为一个隐性混杂变量。

批次效应的检测方法主要依赖比较不同批次之间关键 QC 指标的分布：

  • 比较各批次的 MAF 分布，相同 SNP 的 MAF 在不同批次间应高度一致（差值通常 < 0.02）；
  • 比较各批次的 HWE p 值分布，异常批次往往在特定染色体区段出现 HWE 显著偏离的富集；
  • 比较各批次的样本 call rate 和位点 call rate 的中位数与分布形态；
  • 将所有样本进行联合 PCA，若批次效应显著，不同批次的样本会在 PCA 散点图中形成沿 PC1 或 PC2 方向的聚类，而非按群体来源聚类。

批次效应的处理策略需根据效应的强度和研究设计灵活选择。最直接的方法是将批次标签作为协变量纳入关联回归模型，使 \(\beta_\text{SNP}\) 的估计在控制批次之后进行；这一方法对效应强度较弱的批次差异有效，且对标准逻辑回归和线性混合模型均适用。当批次效应在统计量层面（如 Z 分数）表现明显时，可使用 ComBat 等经验贝叶斯方法对统计量进行后验校正，但需注意这类方法可能过度校正、消除真实的群体信号。最严格的处理方案是仅保留批次间 QC 指标高度一致的 SNP，或将批次内 call rate 差异超过 5% 的位点直接删除。

批次效应是 QQ 图膨胀的隐性来源

在多中心 GWAS 中，如果 QQ 图显示系统性膨胀（\(\lambda_{GC} > 1.05\)）但 PCA 分析未发现明显群体分层，应首先怀疑批次效应而非遗传混杂。此时应按批次分层重新绘制 QQ 图，若批次内 \(\lambda_{GC}\) 接近 1.0 而合并后膨胀，则批次效应是主要原因，需在关联分析中明确控制批次协变量。

17.4.4 PLINK QC 命令示例

本节使用 1000 Genomes Project Phase 3 第 22 号染色体数据集（1kg_chr22_common）作为教学示范输入。该数据集基本信息如下：

属性 值
来源 1000 Genomes Project Phase 3，chr22
样本量 2,504 个体，覆盖非洲、欧洲、东亚、南亚、美洲五大超级群体
SNP 数 98,193 个常见变异（MAF ≥ 0.05，已过滤低频与多等位基因位点）
格式 PLINK binary（.bed/.bim/.fam）
总基因型缺失率 0.0054%（即整体 call rate 约 99.995%，质量极高）

该数据集已完成上游的双等位基因过滤（--biallelic-only）和初始 MAF 过滤，代表 VCF 转换为 PLINK 格式并完成基本预处理之后的典型 GWAS 起点。需要注意，此数据集的 .fam 文件中表型列均为缺失值（-9），性别列亦未记录（全部标注为 0），这是 1000 Genomes 公开数据的常见情况；在真实 GWAS 分析中，这两列应在下载数据后根据样本元数据（phenotype file、sex file）手动填写。

以下 QC 流程按标准步骤逐步执行，每步生成一个新的 PLINK 二进制文件，便于在任意步骤回溯检查：

# Step 1: 样本 call rate 过滤（--mind 0.05 即保留 call rate > 95% 的样本）
# 预期：1kg_chr22_common 总体缺失率极低，本步通常无样本被删除
plink \
  --bfile 1kg_chr22_common \
  --mind 0.05 \
  --make-bed \
  --out qc_step1

# Step 2: SNP call rate 过滤（保留在 ≥ 95% 样本中成功分型的 SNP）
# 预期：同上，高质量芯片数据此步删除位点极少
plink \
  --bfile qc_step1 \
  --geno 0.05 \
  --make-bed \
  --out qc_step2

# Step 3: MAF 过滤（输入已为 MAF ≥ 0.05；此处设 0.01 保留 1kg 数据中 MAF 稍低的位点）
# 注：若严格按 GWAS 惯例使用 MAF ≥ 0.05，可将 --maf 改为 0.05
plink \
  --bfile qc_step2 \
  --maf 0.01 \
  --make-bed \
  --out qc_step3

# Step 4: HWE 过滤（--hwe 默认仅对 .fam 中表型为 1=对照 的样本执行）
# 注：1kg 数据无表型，--hwe 实际作用于全部样本；
#     在有真实病例对照数据时务必确认该行为
plink \
  --bfile qc_step3 \
  --hwe 1e-6 \
  --make-bed \
  --out qc_step4

# Step 5: 计算每个样本的杂合率，用于后续在 R 中过滤杂合率异常样本
plink \
  --bfile qc_step4 \
  --het \
  --out het_check

# Step 6: LD 剪枝，为 IBD 估计和 PCA 准备近似独立的 SNP 子集
# 参数含义：窗口 50 SNP，步进 10 SNP，窗口内 r^2 > 0.1 则删除
plink \
  --bfile qc_step4 \
  --indep-pairwise 50 10 0.1 \
  --out pruned_snps

# Step 6b: 使用剪枝后 SNP 集估计样本间 IBD（pi-hat）

17.5 单 SNP 关联检验

17.5.1 遗传模型的选择

在进行关联检验时，必须首先指定一个遗传模型（genetic model），描述基因型与表型之间的关系。不同遗传模型的选择对统计检验力有显著影响。加性模型（additive model） 假设每增加一个次等位基因拷贝，表型效应等量增加。若以 0、1、2 对应 BB、AB、AA 进行编码，则加性效应意味着基因型 AB 的表型均值恰好处于 BB 和 AA 之间。加性模型是 GWAS 的默认选择，因为大量遗传变异的效应确实近似加性，且在等效应量下加性模型具有最高的统计检验力。显性模型（dominant model） 假设只要携带至少一个次等位基因，效应与纯合次等位基因相同，编码方式为 BB = 0，AB = AA = 1。隐性模型（recessive model） 假设只有纯合次等位基因才表现出效应，编码为 BB = AB = 0，AA = 1。在 GWAS 实践中，通常默认使用加性模型，辅以敏感性分析检验其他模型；若先验知识提示某变异的遗传模式，则应相应调整模型。

17.5.2 卡方检验（病例-对照研究）

对于二元表型（如患病 vs. 健康），最简单的关联检验方法是构建 2×3 列联表（病例/对照 × 三种基因型）并进行卡方检验。

17.5.2.1 统计量的推导与大样本近似

设 \(r\) 行 \(c\) 列的列联表中，第 \((i,j)\) 格的观测频次为 \(O_{ij}\)，对应的边际频率为行总计 \(R_i = \sum_j O_{ij}\) 和列总计 \(C_j = \sum_i O_{ij}\)，总样本量为 \(N = \sum_{i,j} O_{ij}\)。零假设为行变量（病例/对照）与列变量（基因型）相互独立，即 \(P(\text{row}=i, \text{col}=j) = P(\text{row}=i) \cdot P(\text{col}=j)\)。在此零假设下，期望频次为：

\[ E_{ij} = \frac{R_i \cdot C_j}{N} \]

Pearson 卡方统计量定义为：

\[ \chi^2 = \sum_{i=1}^{r}\sum_{j=1}^{c} \frac{(O_{ij} - E_{ij})^2}{E_{ij}} \]

当 \(N \to \infty\) 时，该统计量在零假设下渐近服从自由度为 \((r-1)(c-1)\) 的 \(\chi^2\) 分布。自由度的来源可以直观理解为：\(r \times c\) 个格子中，确定行总计和列总计后，独立可变的格子数为 \((r-1)(c-1)\)——即去掉一行和一列的约束后剩余的自由参数数量。对于 2×3 的基因型表，自由度为 \((2-1)(3-1)=2\)；对于 2×2 的等位基因表，自由度为 1。

大样本近似的适用条件为所有格子的期望频次 \(E_{ij} \geq 5\)。当期望频次过小时（常见于 MAF 极低的位点），Pearson 卡方的近似失效，应改用 Fisher 精确检验（Fisher’s exact test），后者无需大样本假设，直接基于超几何分布计算精确 p 值，但计算代价较高。

17.5.2.2 基因型检验与等位基因检验的比较

两种检验在数据组织方式、统计假设和适用场景上存在本质差异，理解这些差异是选择正确检验方法的前提。

列联表结构的差异

基因型检验构建 2×3 列联表：行为病例/对照，列为三种基因型（BB/AB/AA，对应次等位基因计数 0/1/2），每个个体贡献一行观测。等位基因检验将每个个体的两条单倍型”拆分”为独立观测，构建 2×2 列联表：行为病例/对照状态（每个个体重复两次），列为参考/效应等位基因。这一拆分使有效样本量从 \(N\) 个个体翻倍至 \(2N\) 条单倍型，但前提是两条单倍型在统计上相互独立——这正是 HWE 假设的核心内容。

维度 基因型检验（Genotype test） 等位基因检验（Allelic test）
列联表规格 2×3（个体为观测单元） 2×2（单倍型为观测单元）
自由度 df = 2 df = 1
HWE 依赖性 不依赖 HWE 要求 HWE 成立
加性效应下检验力 较低（多余 1 个 df） 较高（df 聚焦加性分量）
对非加性效应的敏感性 可同时检出显性、隐性、超显性效应 仅对加性分量敏感，对非加性效应检验力低
期望频次最小要求 所有 6 个格子 \(E_{ij} \geq 5\) 所有 4 个格子 \(E_{ij} \geq 5\)
MAF 极低时的稳健性 纯合次等位基因格子期望频次易 < 5 2×2 格子更易满足期望频次条件
PLINK 实现 --assoc model --model genotype --assoc（默认输出）
典型应用 遗传模式未知、疑有显性/隐性效应时 GWAS 主扫描（HWE 已过滤后）

检验力差异的统计机制

自由度决定了在相同非中心参数（NCP）下的检验力。等位基因检验 df = 1，基因型检验 df = 2；在效应为纯加性时，基因型检验多出的 1 个自由度消耗在检测偏离加性的分量上，而该分量的真实效应为零，相当于引入了额外的随机噪声，导致 p 值系统性偏大。以 NCP = 10 为例，df = 1 的卡方检验功效约 86%，而 df = 2 的检验功效约 76%。这一约 10 个百分点的差距在 GWAS 规模（百万 SNP，显著阈值 \(5 \times 10^{-8}\)）下会转化为数量可观的真实信号漏检。

HWE 偏离时等位基因检验的失效机制

等位基因检验将两条单倍型当作独立同分布（i.i.d.）的观测，但若某位点偏离 HWE（如杂合子不足，\(P(AB) < 2pq\)），则同一个体的两个等位基因之间存在统计依赖（给定一条等位基因为 \(A\)，另一条也为 \(A\) 的概率高于随机期望）。这种依赖使得等位基因检验的方差估计偏小，等价于人为缩小了标准误，导致统计量虚增、p 值偏小——即使位点真实无关联也可能出现显著结果。这是 GWAS QC 流程中对照组 HWE 过滤必须先于等位基因检验执行的直接原因。

以下模拟数据同时演示两种检验的实施步骤，并通过手动计算期望频次矩阵验证统计量的推导过程。模拟设定：样本量 \(n = 1{,}000\)，MAF = 0.3，加性效应（基线 log-odds = −1，\(\beta_{\text{SNP}} = 0.5\)）。

set.seed(42)
n    <- 1000
geno <- rbinom(n, 2, prob = 0.3)                    # genotype: 0/1/2 minor allele copies
case <- rbinom(n, 1, plogis(-1 + 0.5 * geno))       # binary phenotype under additive model

# ---- Build contingency tables ----
tbl_geno   <- table(case, geno)                     # 2x3: case/ctrl x genotype (0/1/2)

# allelic test: unpack each individual into 2 haplotype alleles (valid only under HWE)
# geno=0 -> (0,0);  geno=1 -> (0,1) or (1,0);  geno=2 -> (1,1)
allele1     <- ifelse(geno >= 1, 1L, 0L)
allele2     <- ifelse(geno == 2, 1L, 0L)
tbl_allele  <- table(rep(case, 2), c(allele1, allele2))  # 2x2: case/ctrl x allele (0/1)

# ---- Manual chi-square for 2x3 genotype table ----
row_g  <- rowSums(tbl_geno);  col_g  <- colSums(tbl_geno);  N_g  <- sum(tbl_geno)
exp_g  <- outer(row_g, col_g) / N_g
chi2_g <- sum((tbl_geno - exp_g)^2 / exp_g)
df_g   <- (nrow(tbl_geno) - 1) * (ncol(tbl_geno) - 1)        # (2-1)*(3-1) = 2
p_g    <- pchisq(chi2_g, df = df_g, lower.tail = FALSE)
cat(sprintf("Genotype test  (df=%d): chi2 = %.4f,  p = %.4e\n", df_g, chi2_g, p_g))

# ---- Manual chi-square for 2x2 allelic table ----
row_a  <- rowSums(tbl_allele); col_a  <- colSums(tbl_allele); N_a  <- sum(tbl_allele)
exp_a  <- outer(row_a, col_a) / N_a
chi2_a <- sum((tbl_allele - exp_a)^2 / exp_a)
df_a   <- (nrow(tbl_allele) - 1) * (ncol(tbl_allele) - 1)    # (2-1)*(2-1) = 1
p_a    <- pchisq(chi2_a, df = df_a, lower.tail = FALSE)
cat(sprintf("Allelic test   (df=%d): chi2 = %.4f,  p = %.4e\n", df_a, chi2_a, p_a))

卡方检验无法控制群体分层

卡方检验基于列联表，其统计框架不支持纳入任何协变量。当研究样本来自遗传背景不同的亚群体时，群体分层混杂无法通过卡方检验加以校正，此时应改用逻辑回归并将 PCA 主成分作为协变量纳入模型（参见下节）。

17.5.3 Cochran-Armitage 趋势检验

Cochran-Armitage 趋势检验（Cochran-Armitage trend test） 是加性遗传模型下最优的单自由度关联检验，由 Cochran（1954）和 Armitage（1955）独立提出。与基因型检验（df = 2）相比，趋势检验在真实效应为加性时具有更高的统计检验力；与等位基因检验相比，趋势检验不要求 HWE 成立，对偏离 HWE 的位点更为稳健。

17.5.3.1 统计量的推导

设 \(n_0, n_1, n_2\) 分别为基因型 BB、AB、AA 的总样本数，\(r_0, r_1, r_2\) 为各基因型中病例数，\(N = n_0 + n_1 + n_2\)，患病率 \(\bar{p} = (r_0 + r_1 + r_2)/N\)。在加性编码下，三种基因型赋分 \(x_j \in \{0, 1, 2\}\)。趋势检验统计量定义为：

\[ T = \frac{\left(\sum_j x_j r_j - \bar{p} \sum_j x_j n_j\right)^2}{\bar{p}(1-\bar{p})\left[\sum_j x_j^2 n_j - \frac{(\sum_j x_j n_j)^2}{N}\right]} \]

在零假设下，\(T\) 渐近服从 \(\chi^2_1\) 分布。分子为效应方向统计量的平方，分母为其方差估计，整体结构与 Pearson 相关系数的平方检验完全等价——趋势检验本质上是检验加性基因型得分（0/1/2）与病例状态之间的线性相关性。

趋势检验、等位基因检验与基因型检验的关系

三种检验统计量在加性效应下的检验力排序为：趋势检验 \(\approx\) 等位基因检验 \(>\) 基因型检验（df=2）。等位基因检验与趋势检验在 HWE 成立时数值上几乎等价，但等位基因检验在 HWE 偏离时有偏；趋势检验在 HWE 不成立时仍保持正确的 I 类错误控制。因此，PLINK 默认的 --assoc 输出的是等位基因检验结果，而更严格的分析流程（如 --trend）应采用趋势检验。

以下代码在上方同一模拟数据集（geno、case，set.seed(42)，\(n=1000\)，MAF = 0.3）上，将 Cochran-Armitage 趋势检验与已计算的两种卡方统计量并排比较，展示三者的 p 值差异：

# ---- Cochran-Armitage trend test (df=1, no HWE requirement) ----
# Reuses geno, case, tbl_geno from the chi-square section above.
# T = [sum_j x_j r_j - p_bar * sum_j x_j n_j]^2 /
#     [p_bar*(1-p_bar)*(sum_j x_j^2 n_j - (sum_j x_j n_j)^2/N)]
r_j   <- tapply(case, geno, sum)         # disease cases per genotype group (0/1/2)
n_j   <- table(geno)                     # total count per genotype group
x_j   <- as.numeric(names(n_j))         # additive scores: 0, 1, 2
N_tot <- length(case)
p_bar <- mean(case)                      # overall disease prevalence

numer <- (sum(x_j * r_j) - p_bar * sum(x_j * n_j))^2
denom <- p_bar * (1 - p_bar) *
         (sum(x_j^2 * n_j) - sum(x_j * n_j)^2 / N_tot)
T_ca  <- numer / denom
p_ca  <- pchisq(T_ca, df = 1, lower.tail = FALSE)

# ---- Three-way comparison (all from the same simulated dataset) ----
cat(sprintf("Genotype test  (df=2): p = %.4e\n", p_g))   # from chi-square section
cat(sprintf("Allelic test   (df=1): p = %.4e\n", p_a))   # from chi-square section
cat(sprintf("C-A trend test (df=1): p = %.4e\n", p_ca))

17.5.4 逻辑回归（Logistic Regression）

逻辑回归是病例-对照 GWAS 的标准方法，其优势在于可以将协变量（如年龄、性别、PCA 主成分）纳入模型，同时提供对效应量（OR）的自然解释。模型形式为：

\[ \log\frac{P(Y=1)}{P(Y=0)} = \beta_0 + \beta_{\text{SNP}} \cdot G + \beta_1 X_1 + \cdots + \beta_k X_k \]

其中 \(G\) 为 SNP 的加性编码（0/1/2，计次等位基因数），\(\beta_{\text{SNP}}\) 的指数 \(e^{\beta_{\text{SNP}}}\) 即为 优势比（Odds Ratio，OR）。

17.5.4.1 从模型到对数似然函数的完整推导

第一步：从线性预测量到概率

逻辑回归的出发点是为每个个体构造一个线性预测量（linear predictor）：

\[ \eta_i = \beta_0 + \beta_{\text{SNP}} \cdot G_i + \beta_1 X_{i1} + \cdots + \beta_k X_{ik} = \mathbf{x}_i^\top \boldsymbol{\beta} \]

\(\eta_i\) 的取值范围为 \((-\infty, +\infty)\)，而概率必须在 \([0,1]\) 内。为将 \(\eta_i\) 映射到概率，逻辑回归使用 logistic 函数（S 型函数） 作为逆连接函数（inverse link function）：

\[ \mu_i = P(Y_i = 1 \mid \mathbf{x}_i) = \sigma(\eta_i) = \frac{1}{1 + e^{-\eta_i}} = \frac{e^{\eta_i}}{1 + e^{\eta_i}} \]

这一步与模型公式的等价性可直接验证：将 \(P(Y=1)/P(Y=0)\) 取对数（即对 odds 取对数），得到对数优势（log-odds，logit）：

\[ \log\frac{\mu_i}{1-\mu_i} = \log\frac{\sigma(\eta_i)}{1-\sigma(\eta_i)} = \log\frac{e^{\eta_i}/(1+e^{\eta_i})}{1/(1+e^{\eta_i})} = \log e^{\eta_i} = \eta_i = \mathbf{x}_i^\top\boldsymbol{\beta} \]

因此，模型公式中写”对数优势等于线性预测量”与”患病概率等于 \(\sigma(\mathbf{x}_i^\top\boldsymbol{\beta})\)“完全等价——前者是模型的正向表达（便于解释系数），后者是反向表达（便于计算概率和构造似然函数）。同时，logistic 函数有一个重要性质：

\[ 1 - \sigma(\eta) = \sigma(-\eta) = \frac{1}{1 + e^{\eta}} \]

这将在后续化简中频繁用到。

第二步：构造单个样本的似然

给定参数 \(\boldsymbol{\beta}\)，第 \(i\) 个样本的观测结局 \(y_i \in \{0, 1\}\) 服从以 \(\mu_i\) 为参数的伯努利分布（Bernoulli distribution），其概率质量函数为：

\[ P(Y_i = y_i \mid \mathbf{x}_i;\, \boldsymbol{\beta}) = \mu_i^{y_i}(1-\mu_i)^{1-y_i} \]

这一写法可以统一表达两种情况：当 \(y_i = 1\) 时上式等于 \(\mu_i\)；当 \(y_i = 0\) 时等于 \(1-\mu_i\)。该概率即为参数 \(\boldsymbol{\beta}\) 关于第 \(i\) 个观测的似然贡献（likelihood contribution）。

第三步：构造全样本联合似然

假设 \(n\) 个样本独立采集（GWAS 无关个体的核心假设），联合似然为各样本似然贡献之积：

\[ L(\boldsymbol{\beta}) = \prod_{i=1}^n \mu_i^{y_i}(1-\mu_i)^{1-y_i} \]

连乘在数值计算中易下溢（underflow），且对乘积求导不便。取自然对数转换为对数似然：

\[ \ell(\boldsymbol{\beta}) = \log L(\boldsymbol{\beta}) = \sum_{i=1}^n \left[ y_i \log \mu_i + (1-y_i)\log(1-\mu_i) \right] \]

第四步：代入 logistic 函数化简

将 \(\mu_i = \sigma(\eta_i) = e^{\eta_i}/(1+e^{\eta_i})\) 和 \(1-\mu_i = 1/(1+e^{\eta_i})\) 代入：

\[ \log \mu_i = \log\frac{e^{\eta_i}}{1+e^{\eta_i}} = \eta_i - \log(1+e^{\eta_i}) \]

\[ \log(1-\mu_i) = \log\frac{1}{1+e^{\eta_i}} = -\log(1+e^{\eta_i}) \]

代入对数似然的求和式：

\[ \ell(\boldsymbol{\beta}) = \sum_{i=1}^n \left[ y_i\bigl(\eta_i - \log(1+e^{\eta_i})\bigr) + (1-y_i)\bigl(-\log(1+e^{\eta_i})\bigr) \right] \]

展开合并 \(\log(1+e^{\eta_i})\) 项：

\[ \boxed{\ell(\boldsymbol{\beta}) = \sum_{i=1}^n \left[ y_i\,\mathbf{x}_i^\top\boldsymbol{\beta} - \log\!\left(1 + e^{\mathbf{x}_i^\top\boldsymbol{\beta}}\right) \right]} \]

这正是逻辑回归对数似然函数的标准形式。第一项 \(y_i\eta_i\) 反映观测结局对预测的”奖励”（病例的 \(\eta_i\) 越大贡献越大），第二项 \(-\log(1+e^{\eta_i})\) 是对所有样本的惩罚项，防止 \(\boldsymbol{\beta}\) 无限增大。

对数似然函数的凹性保证唯一最大值

\(\ell(\boldsymbol{\beta})\) 关于 \(\boldsymbol{\beta}\) 是严格凹函数（strictly concave）：其 Hessian 矩阵 \(\mathbf{H} = -\mathbf{X}^\top\mathbf{W}\mathbf{X}\)（见下节推导）在 \(\mathbf{X}\) 列满秩时负定，因此对数似然没有局部极大值，任何迭代算法（Newton-Raphson、梯度上升）只要收敛就必然找到全局最大值。完全分离情形是唯一例外：当某列 \(\mathbf{x}\) 能完美分开 \(y=0\) 和 \(y=1\) 时，\(\ell(\boldsymbol{\beta})\) 单调递增无上界，MLE 不存在。

第五步：求导得到得分函数与 Hessian

对 \(\ell(\boldsymbol{\beta})\) 关于 \(\boldsymbol{\beta}\) 求一阶偏导，利用复合函数求导和 \(\partial\sigma(\eta)/\partial\eta = \sigma(\eta)(1-\sigma(\eta)) = \mu_i(1-\mu_i)\)：

\[ \frac{\partial \ell}{\partial \boldsymbol{\beta}} = \sum_{i=1}^n \left[ y_i \mathbf{x}_i - \frac{e^{\eta_i}}{1+e^{\eta_i}} \mathbf{x}_i \right] = \sum_{i=1}^n (y_i - \mu_i)\mathbf{x}_i = \mathbf{X}^\top(\mathbf{y} - \boldsymbol{\mu}) \]

令得分函数为零即为 MLE 的一阶条件 \(\mathbf{X}^\top(\mathbf{y} - \boldsymbol{\mu}) = \mathbf{0}\)，无解析解，需数值迭代。对得分函数再次求导得 Hessian：

\[ \mathbf{H} = \frac{\partial^2 \ell}{\partial \boldsymbol{\beta}\partial\boldsymbol{\beta}^\top} = -\sum_{i=1}^n \mu_i(1-\mu_i)\,\mathbf{x}_i\mathbf{x}_i^\top = -\mathbf{X}^\top\mathbf{W}\mathbf{X} \]

其中 \(\mathbf{W} = \text{diag}(\mu_1(1-\mu_1),\,\ldots,\,\mu_n(1-\mu_n))\) 为对角权重矩阵，每个对角元素 \(w_i = \mu_i(1-\mu_i) \in (0, 0.25]\) 恰好是伯努利分布在 \(\mu_i\) 处的方差。Fisher 信息矩阵 \(\mathcal{I}(\boldsymbol{\beta}) = -\mathbf{H} = \mathbf{X}^\top\mathbf{W}\mathbf{X}\) 的逆矩阵给出参数估计的渐近协方差：

\[ \text{Var}(\hat{\boldsymbol{\beta}}) \approx \mathcal{I}(\hat{\boldsymbol{\beta}})^{-1} = \left(\mathbf{X}^\top\hat{\mathbf{W}}\mathbf{X}\right)^{-1} \]

17.5.4.2 Newton-Raphson 迭代与 IRLS 等价性

Newton-Raphson 更新公式利用二阶 Taylor 展开将非线性优化局部线性化：

\[ \boldsymbol{\beta}^{(t+1)} = \boldsymbol{\beta}^{(t)} - \mathbf{H}^{-1} \nabla_{\boldsymbol{\beta}}\ell = \boldsymbol{\beta}^{(t)} + (\mathbf{X}^\top\mathbf{W}^{(t)}\mathbf{X})^{-1} \mathbf{X}^\top(\mathbf{y} - \boldsymbol{\mu}^{(t)}) \]

每次迭代更新三个量：预测概率 \(\boldsymbol{\mu}^{(t)}\)、权重矩阵 \(\mathbf{W}^{(t)}\) 和梯度 \(\mathbf{X}^\top(\mathbf{y}-\boldsymbol{\mu}^{(t)})\)。

NR 迭代与迭代加权最小二乘（Iteratively Reweighted Least Squares，IRLS） 完全等价。将更新公式改写为加权线性回归的形式，令工作响应向量（working response） 为：

\[ \mathbf{z}^{(t)} = \boldsymbol{\eta}^{(t)} + (\mathbf{W}^{(t)})^{-1}(\mathbf{y} - \boldsymbol{\mu}^{(t)}) \]

其中 \(\boldsymbol{\eta}^{(t)} = \mathbf{X}\boldsymbol{\beta}^{(t)}\)。则每步 NR 更新等价于以权重 \(\mathbf{W}^{(t)}\) 对 \(\mathbf{z}^{(t)}\) 拟合加权线性回归：

\[ \boldsymbol{\beta}^{(t+1)} = \left(\mathbf{X}^\top\mathbf{W}^{(t)}\mathbf{X}\right)^{-1} \mathbf{X}^\top\mathbf{W}^{(t)}\mathbf{z}^{(t)} \]

IRLS 的物理意义是：每次迭代在当前参数估计附近，用一个线性化的加权最小二乘问题近似原非线性优化问题，并将该近似问题的解作为下一步的参数估计。这一等价关系是 R 的 glm() 内部实现逻辑回归的数学基础，也是 EIGENSTRAT 在逻辑回归中应用 FWL 定理的技术支撑（见后续 PCA 校正节）。

17.5.4.3 OR 的定量特征与解读限制

OR 的基本含义：\(e^{\beta_{\text{SNP}}}\) 表示每增加一个次等位基因，患病的优势（odds = \(p/(1-p)\)） 变为原来的 \(e^{\beta_{\text{SNP}}}\) 倍，而非患病概率变为该倍数。这一区别在低患病率（< 10%）时影响很小（OR ≈ RR），但在高患病率时 OR 会系统性高估相对风险 RR。

OR 与 RR 的定量关系（Woolf 近似）：

\[ \text{RR} = \frac{\text{OR}}{(1 - p_0) + p_0 \cdot \text{OR}} \]

其中 \(p_0\) 为参考基因型（次等位基因计数 = 0）的患病率。当 \(p_0 = 0.05\) 且 OR = 1.5 时，RR \(\approx\) 1.48；当 \(p_0 = 0.30\) 且 OR = 1.5 时，RR \(\approx\) 1.30——高患病率时 OR 高估 RR 达 15%。

OR 的置信区间由参数估计的标准误推导：

\[ 95\%\ \text{CI of OR} = \exp\left(\hat{\beta}_{\text{SNP}} \pm 1.96 \cdot \widehat{\text{SE}}(\hat{\beta}_{\text{SNP}})\right) \]

其中 \(\widehat{\text{SE}} = \sqrt{[(\mathbf{X}^\top\hat{\mathbf{W}}\mathbf{X})^{-1}]_{jj}}\)（第 \(j\) 个参数对应的对角元素）。置信区间不包含 1 与 Wald p 值显著（\(< 0.05\)）完全等价。

GWAS 中 OR 的典型范围：复杂疾病 GWAS 发现的常见变异 OR 通常在 1.05 至 1.3 之间，罕见功能变异可达 2–5。OR 远小于孟德尔单基因疾病的效应量（如 BRCA1 致病突变的 OR > 10），反映复杂疾病的高度多基因性：每个常见变异仅贡献微弱但真实的风险增量。

完全分离（Complete Separation）导致 MLE 发散

当某个协变量能完美预测结局（如某种基因型仅出现在病例组）时，逻辑回归的 MLE 不收敛——对应系数趋向 \(+\infty\) 或 \(-\infty\)，标准误也随之发散。这一问题称为完全分离（complete separation） 或 Hauck-Donner 效应。在 GWAS 中，MAF 极低的位点（如 MAC < 10）最容易发生完全分离，此时 glm() 仍会返回结果但会打印警告”fitted probabilities numerically 0 or 1”，且 OR 和 p 值完全不可信。解决方案包括：过滤 MAC < 20 的位点（标准 GWAS QC），或使用 Firth 偏校正逻辑回归（logistf R 包）。

以下模拟数据演示含协变量的逻辑回归关联检验，并通过手动 NR 迭代验证 glm() 结果，同时输出 OR 的置信区间。模拟设定：\(n = 1{,}000\)，MAF = 0.3，性别（二元）随机分配；\(\beta_{\text{SNP}} = 0.6\)，基线 log-odds = −2（患病率约 12%），期望 OR ≈ \(e^{0.6} \approx 1.82\)。

set.seed(42)
n    <- 1000
geno <- rbinom(n, 2, 0.3)
sex  <- rbinom(n, 1, 0.5)
case <- rbinom(n, 1, plogis(-2 + 0.6 * geno + 0.4 * sex))

# ---- glm() reference ----
fit          <- glm(case ~ geno + sex, family = binomial)
coef_summary <- summary(fit)$coefficients
b_glm        <- coef_summary["geno", "Estimate"]
se_glm       <- coef_summary["geno", "Std. Error"]

cat(sprintf("glm()  OR  = %.3f  (95%% CI: %.3f – %.3f)\n",
            exp(b_glm),
            exp(b_glm - 1.96 * se_glm),
            exp(b_glm + 1.96 * se_glm)))
cat(sprintf("glm()  p   = %.3e\n", coef_summary["geno", "Pr(>|z|)"]))

# ---- Manual Newton-Raphson MLE ----
# Derive: score = X'(y - mu),  Hessian = -X'WX,  NR step = -(H)^{-1} score
X    <- cbind(intercept = 1, geno = geno, sex = sex)
y    <- case
beta <- rep(0, ncol(X))

for (iter in seq_len(200)) {
  eta   <- as.vector(X %*% beta)
  mu    <- 1 / (1 + exp(-eta))            # predicted probability: sigma(eta)
  W     <- mu * (1 - mu)                  # diagonal of weight matrix W
  score <- t(X) %*% (y - mu)             # gradient: X'(y - mu)
  hess  <- -t(X) %*% (X * W)             # Hessian:  -X'WX
  delta <- solve(-hess, score)            # Newton step: (X'WX)^{-1} X'(y-mu)
  beta  <- beta + delta
  if (max(abs(delta)) < 1e-10) break
}

# Asymptotic covariance: Var(beta_hat) = (X'WX)^{-1}
fisher_info <- t(X) %*% (X * W)          # = X'WX evaluated at convergence
se_nr       <- sqrt(diag(solve(fisher_info)))
z_nr        <- beta / se_nr
p_nr        <- 2 * pnorm(abs(z_nr), lower.tail = FALSE)

cat(sprintf("NR MLE OR  = %.3f  (95%% CI: %.3f – %.3f)\n",
            exp(beta[2]),
            exp(beta[2] - 1.96 * se_nr[2]),
            exp(beta[2] + 1.96 * se_nr[2])))
cat(sprintf("NR MLE p   = %.3e\n", p_nr[2]))

# ---- OR vs RR: Woolf approximation at p0 = baseline prevalence ----
p0   <- plogis(-2)                        # prevalence when geno=0, sex=0
OR   <- exp(beta[2])
RR   <- OR / ((1 - p0) + p0 * OR)        # Woolf formula
cat(sprintf("\nOR = %.3f,  estimated RR = %.3f  (baseline prevalence p0 = %.3f)\n",
            OR, RR, p0))

17.5.4.4 逻辑回归中的 PCA 校正

群体分层对逻辑回归的干扰机制

在真实的病例-对照 GWAS 中，研究样本几乎不可避免地来自遗传背景不均质的亚群体。这种不均质性表现为两个方向的系统差异：其一，不同亚群体在大量 SNP 位点上的等位基因频率存在差异，这是演化历史、奠基者效应和遗传漂变的共同结果；其二，不同亚群体的疾病患病率或表型均值可能因生活环境、医疗可及性及非遗传危险因素的不同而产生差异。当这两种差异同时存在时，任何一个等位基因频率在群体间有差异的 SNP，其基因型都会与表型产生统计相关——即使该 SNP 在每一个亚群体内部对表型均无任何因果效应。这一混杂机制在逻辑回归框架中的体现是：目标 SNP 的回归系数 \(\beta_{\text{SNP}}\) 不再是因果效应的无偏估计，而是因果效应与群体差异的混合信号，其方向甚至可能与真实效应相反。

EIGENSTRAT 方法的统计基础

Price 等人于 2006 年提出的 EIGENSTRAT 方法，是目前逻辑回归 GWAS 中控制群体分层的标准框架。其核心思想源于多元回归中的混杂控制原理：若能找到一组协变量，能够充分解释群体来源所引起的系统性变异，将其纳入回归模型后，\(\beta_{\text{SNP}}\) 的估计便只反映在控制了群体来源之后、个体层面的 SNP-表型关联。

EIGENSTRAT 将这一”充分描述群体结构的协变量”定义为基因型矩阵的前若干个主成分。具体而言，设 \(\mathbf{G}\) 为 \(n \times M\) 的中心化标准化基因型矩阵（\(n\) 为样本量，\(M\) 为背景 SNP 数量），对每个位点 \(l\) 的基因型编码按下式标准化：

\[ \tilde{G}_{il} = \frac{G_{il} - 2p_l}{\sqrt{2p_l(1-p_l)}} \]

其中 \(p_l\) 为第 \(l\) 个 SNP 的次等位基因频率估计值。标准化的目的是消除各位点因 MAF 差异引起的方差不等，使每个 SNP 对主成分的贡献与其群体分化程度（而非其频率本身）成正比。对标准化矩阵 \(\tilde{\mathbf{G}}\) 进行奇异值分解（Singular Value Decomposition，SVD）：

\[ \tilde{\mathbf{G}} = \mathbf{U} \boldsymbol{\Sigma} \mathbf{V}^\top \]

其中 \(\mathbf{U}\) 的各列即为样本空间中的主成分方向（样本得分向量），\(\boldsymbol{\Sigma}\) 的对角元素为奇异值，对应各主成分所解释的遗传变异量。第 \(k\) 个主成分得分向量 \(\mathbf{u}_k\)（\(\mathbf{U}\) 的第 \(k\) 列乘以对应奇异值）即为每个个体在第 \(k\) 个遗传结构维度上的坐标，具有群体遗传学上的直观意义：在两个遗传分化显著的群体样本中，PC1 通常能将两个群体完全分离；在多群体混合样本中，前几个主成分共同描述群体间的连续祖先结构。

纳入主成分后的逻辑回归模型

将前 \(K\) 个主成分作为固定效应协变量加入逻辑回归，得到校正后的关联检验模型：

\[ \log\frac{P(Y=1)}{P(Y=0)} = \beta_0 + \beta_{\text{SNP}} \cdot G + \sum_{k=1}^{K} \gamma_k \cdot \text{PC}_k + \beta_{\text{sex}} \cdot \text{sex} + \cdots \]

在此模型中，\(\gamma_k\) 为第 \(k\) 个主成分对患病 log-odds 的线性贡献，其估计值定量描述了群体结构对表型的影响；\(\beta_{\text{SNP}}\) 的估计则在”控制了全基因组群体结构之后”进行，理论上只捕捉亚群体内部的 SNP-表型关联，从而消除群体间差异带来的混杂路径。这一操作等价于：先从目标 SNP 的基因型向量和表型向量中各自移除可被主成分解释的部分（即对主成分做偏回归），再对残差进行关联检验——这正是混杂控制的经典统计原理（Frisch-Waugh-Lovell 定理的直接应用）。

Frisch-Waugh-Lovell 定理与 PCA 校正的几何直觉

Frisch-Waugh-Lovell（FWL）定理是计量经济学中的经典结论，其在线性回归框架下有精确的表述：设多元线性回归模型为 \(\mathbf{y} = \beta_g \mathbf{g} + \mathbf{Z} \boldsymbol{\gamma} + \boldsymbol{\varepsilon}\)，其中 \(\mathbf{y}\) 为表型向量，\(\mathbf{g}\) 为目标 SNP 的基因型向量，\(\mathbf{Z}\) 为 \(n \times K\) 的主成分矩阵（协变量矩阵）。FWL 定理指出：将 \(\mathbf{y}\) 和 \(\mathbf{g}\) 分别对 \(\mathbf{Z}\) 做线性回归，取得残差向量 \(\tilde{\mathbf{y}} = (\mathbf{I} - \mathbf{P}_Z)\mathbf{y}\) 和 \(\tilde{\mathbf{g}} = (\mathbf{I} - \mathbf{P}_Z)\mathbf{g}\)（其中 \(\mathbf{P}_Z = \mathbf{Z}(\mathbf{Z}^\top\mathbf{Z})^{-1}\mathbf{Z}^\top\) 为 \(\mathbf{Z}\) 的投影矩阵），再将 \(\tilde{\mathbf{y}}\) 对 \(\tilde{\mathbf{g}}\) 做简单线性回归，所得系数 \(\hat{\beta}_g\) 与直接拟合完整多元回归模型所得的 \(\hat{\beta}_g\) 完全相同，标准误与检验统计量亦完全一致（仅自由度需对应调整）。

FWL 定理提供了一个直观的几何解释：主成分矩阵 \(\mathbf{Z}\) 张成一个 \(K\) 维的线性子空间（即”群体结构空间”），算子 \((\mathbf{I} - \mathbf{P}_Z)\) 将任意向量投影到与该子空间正交的补空间中。对 \(\mathbf{y}\) 和 \(\mathbf{g}\) 分别取残差 \(\tilde{\mathbf{y}}\)、\(\tilde{\mathbf{g}}\)，等价于将表型信息和基因型信息都投影到”去除群体结构之后的正交补空间”中，再在此残差空间中进行关联检验。换言之，PCA 校正的本质是：在与 PC 正交的空间中寻找 SNP 与表型的关联，只有在该空间中两者仍然相关，才能被判定为真实的遗传关联，而不是由群体来源差异引起的混杂信号。

对于逻辑回归，FWL 定理没有精确的等价形式，因为逻辑回归的目标函数（对数似然）是非线性的，残差化的两阶段估计与同时纳入协变量的一步估计在数值上存在偏差。然而，当样本量足够大且主成分协变量对表型的解释比例不过高时，近似的 FWL 效果仍然成立。正是基于这一近似关系，EIGENSTRAT 算法在实现层面采用迭代线性化（iterative re-weighted least squares，IRLS）的策略：在每次 Newton-Raphson 迭代中，将当前迭代步骤的工作响应变量和权重矩阵对 PC 矩阵做线性残差化，等价于在线性化近似框架下应用 FWL 定理，从而在逻辑回归中实现近似精确的协变量控制。

FWL 定理的认识论价值在于揭示了一条极具实用价值的计算捷径：对于全基因组数百万个 SNP 的逐一回归，最耗时的步骤是将各协变量（PC）纳入模型并求解。若预先将表型向量 \(\mathbf{y}\) 和每一个 SNP 向量 \(\mathbf{g}_j\) 分别对 PC 矩阵做一次线性回归取残差，则后续可对所有 SNP 的残差向量 \(\tilde{\mathbf{g}}_j\) 与 \(\tilde{\mathbf{y}}\) 进行快速卡方检验或 \(t\) 检验，而无需每次都求解包含 \(K\) 个协变量的完整线性模型。由于表型残差 \(\tilde{\mathbf{y}}\) 只需计算一次，SNP 残差 \(\tilde{\mathbf{g}}_j\) 的计算可以高度并行化，整体计算复杂度从 \(O(M \cdot (n K^2 + K^3))\) 降低至 \(O(nK^2 + MnK)\)，在百万级 SNP 和数千 PC 协变量的规模下计算效率可提升一个数量级以上。

实践建议：两阶段残差化加速全基因组扫描

在线性模型 GWAS（如连续性表型的 OLS 回归）中，可以利用 FWL 定理实现高效的两阶段分析：（1）将表型 \(\mathbf{y}\) 对 PC 矩阵 \(\mathbf{Z}\)（以及其他固定协变量）做线性回归，保存残差 \(\tilde{\mathbf{y}}\)；（2）对每个 SNP 向量 \(\mathbf{g}_j\) 也做相同的线性回归取残差 \(\tilde{\mathbf{g}}_j\)；（3）对所有 SNP 计算 \(\tilde{\mathbf{y}}\) 与 \(\tilde{\mathbf{g}}_j\) 的 Pearson 相关系数并转化为 \(F\) 统计量。REGENIE、BOLT-LMM 等现代全基因组回归软件均在内部实现了类似的两阶段优化。对于逻辑回归，上述方法为近似而非精确，建议以软件直接支持的逻辑回归模式为准，仅在需要快速探索性分析时使用线性近似。

主成分数量 \(K\) 的选择

\(K\) 的选择在实践中需要平衡两种错误：\(K\) 过小则群体分层校正不充分，导致假阳性膨胀；\(K\) 过大则引入多余的自由度，降低统计检验力，且可能将真实的基因效应与主成分混淆（当目标 SNP 本身就是群体分化位点时）。标准的经验做法是纳入前 10 个主成分，这一选择已在大量 GWAS 中验证具有充分的校正效果，同时对检验力的损耗可以接受。更严格的方法是通过 Tracy-Widom 检验对主成分的显著性进行统计检验，仅纳入达到显著性阈值的主成分；或利用 QQ 图和基因组膨胀因子 \(\lambda_{GC}\) 迭代评估不同 \(K\) 值下的校正效果，选取使 \(\lambda_{GC}\) 最接近 1.0 的 \(K\)。在存在明确的多群体采样设计（如多中心招募、多民族队列）时，也可以根据先验的群体标签选取不同的 \(K\)，或采用混合线性模型（参见后续章节）进行更全面的校正。

黄金法则：主成分的含义可以可视化验证

在纳入主成分之前，始终应绘制 PC1 vs. PC2 散点图，并将样本按已知或估计的群体来源着色。若主成分确实捕捉到了群体结构，则不同群体的样本应在散点图中形成可辨认的聚类或渐变带。若散点图中无明显结构（样本均匀分布），则说明当前数据集的群体分层效应较弱，此时纳入主成分对结果影响有限，但仍建议保留以保证分析规范性。

以下模拟数据定量展示群体分层对逻辑回归的影响及 PCA 校正的效果。模拟设定：两个亚群体各 500 个样本，共 500 个背景 SNP；群体 1 的各位点 MAF 较高（0.4–0.7），群体 2 较低（0.05–0.3）；目标 SNP 在各群体内部与表型无因果关系，但两群体的基础患病率不同（群体 1 约 30%，群体 2 约 10%），造成混杂。

set.seed(42)
n_pop2 <- 500
n_snp2 <- 500

# Background genotype matrix for PCA
maf1_lr   <- runif(n_snp2, 0.4, 0.7)
maf2_lr   <- runif(n_snp2, 0.05, 0.3)
gmat1_lr  <- mapply(function(m) rbinom(n_pop2, 2, m), maf1_lr)
gmat2_lr  <- mapply(function(m) rbinom(n_pop2, 2, m), maf2_lr)
G_lr      <- rbind(gmat1_lr, gmat2_lr)            # 1000 x 500

# Target SNP (column 1): no true causal effect
geno_lr   <- G_lr[, 1]

# Binary phenotype: driven by population membership, NOT by target SNP
# Pop1 baseline log-odds = 0 (~50% prevalence shifted), Pop2 = -2 (~12%)
base_lo   <- c(rep( 0.0, n_pop2), rep(-2.0, n_pop2))
case_lr   <- rbinom(n_pop2 * 2, 1, plogis(base_lo))

# ---- PCA on background SNPs ----
G_bg_lr   <- scale(G_lr[, -1])                    # 499 background SNPs
svd_lr    <- svd(G_bg_lr / sqrt(nrow(G_bg_lr) - 1))
PC_lr     <- svd_lr$u %*% diag(svd_lr$d)

# ---- Newton-Raphson logistic regression (no glm) ----
logit_nr <- function(X_in, y_in, max_iter = 200, tol = 1e-9) {
  b <- rep(0, ncol(X_in))
  for (i in seq_len(max_iter)) {
    mu    <- 1 / (1 + exp(-as.vector(X_in %*% b)))
    W     <- mu * (1 - mu)
    score <- t(X_in) %*% (y_in - mu)
    hess  <- -t(X_in) %*% (X_in * W)
    delta <- solve(-hess, score)
    b     <- b + delta
    if (max(abs(delta)) < tol) break
  }
  fi <- t(X_in) %*% (X_in * W)
  se <- sqrt(diag(solve(fi)))
  z  <- b / se
  p  <- 2 * pnorm(abs(z), lower.tail = FALSE)
  list(beta = b, se = se, p = p, OR = exp(b))
}

# Model 1: naive
m1 <- logit_nr(cbind(1, geno_lr), case_lr)

# Model 2: PCA-corrected (PC1 + PC2)
m2 <- logit_nr(cbind(1, geno_lr, PC_lr[, 1], PC_lr[, 2]), case_lr)

cat(sprintf("Naive       OR = %.3f,  p = %.2e\n", m1$OR[2], m1$p[2]))
cat(sprintf("PCA-correct OR = %.3f,  p = %.2e\n", m2$OR[2], m2$p[2]))

期望结果解读

无校正模型中，目标 SNP 的 p 值应极小（假阳性），OR 偏离 1.0；加入 PC1、PC2 校正后，p 值应升至不显著水平，OR 回归至 1.0 附近，说明原有”关联”完全来自群体分层混杂，并非真实生物学效应。

17.5.5 线性回归（连续性状）

对于连续表型（如血压、身高、产量），GWAS 使用普通最小二乘（Ordinary Least Squares，OLS） 线性回归：

\[ Y = \beta_0 + \beta_{\text{SNP}} \cdot G + \beta_1 X_1 + \cdots + \beta_k X_k + \varepsilon, \quad \varepsilon \sim \mathcal{N}(0, \sigma^2\mathbf{I}) \]

\(\beta_{\text{SNP}}\) 为每增加一个次等位基因副本时表型均值的绝对变化量；其显著性通过 \(t\) 检验（等价于 \(F\) 检验）评估，自由度为 \(n - p\)（\(p\) 为参数总数含截距）。

17.5.5.1 OLS 估计量的推导与最优性

OLS 的目标是最小化残差平方和 \(\text{RSS} = \|\mathbf{y} - \mathbf{X}\boldsymbol{\beta}\|^2\)。对 \(\boldsymbol{\beta}\) 求导并令其为零，得正规方程（normal equations）：

\[ \mathbf{X}^\top\mathbf{X}\hat{\boldsymbol{\beta}} = \mathbf{X}^\top\mathbf{y} \]

当 \(\mathbf{X}\) 列满秩时，解析解为：

\[ \hat{\boldsymbol{\beta}}_{\text{OLS}} = (\mathbf{X}^\top\mathbf{X})^{-1}\mathbf{X}^\top\mathbf{y} \]

残差方差的无偏估计为 \(\hat{\sigma}^2 = \mathbf{e}^\top\mathbf{e}/(n-p)\)，其中 \(\mathbf{e} = \mathbf{y} - \mathbf{X}\hat{\boldsymbol{\beta}}\)；参数的协方差矩阵为 \(\text{Var}(\hat{\boldsymbol{\beta}}) = \hat{\sigma}^2(\mathbf{X}^\top\mathbf{X})^{-1}\)。

Gauss-Markov 定理保证了 OLS 估计量在特定条件下的最优性：若误差满足（1）零均值 \(\mathbb{E}[\varepsilon_i] = 0\)，（2）同方差 \(\text{Var}(\varepsilon_i) = \sigma^2\)，（3）不相关 \(\text{Cov}(\varepsilon_i, \varepsilon_j) = 0\)（\(i \neq j\)），则 \(\hat{\boldsymbol{\beta}}_{\text{OLS}}\) 是所有线性无偏估计量（Linear Unbiased Estimators） 中方差最小的，即 OLS 为BLUE（Best Linear Unbiased Estimator）。

Gauss-Markov 条件在 GWAS 中的违反

当样本存在亲缘关系或群体分层时，条件（3）被违反——同一家系或同一群体内的个体残差存在系统性相关，OLS 的协方差矩阵估计偏小，导致 \(t\) 统计量虚增。混合线性模型（MLM）通过将协方差结构 \(\mathbf{V} = \sigma_g^2\mathbf{K} + \sigma_e^2\mathbf{I}\) 显式建模，用广义最小二乘（GLS）替代 OLS，恢复 BLUE 性质（参见后续 MLM 节）。

17.5.5.2 效应量指标：\(\beta\)、标准化效应量与 PVE

绝对效应量 \(\hat{\beta}_{\text{SNP}}\) 的单位与表型相同，难以跨研究比较（血压单位为 mmHg，身高单位为 cm）。GWAS 通常在分析前对表型进行标准化（均值 0，方差 1），此时 \(\hat{\beta}_{\text{SNP}}\) 变为标准化效应量：每增加一个次等位基因，表型变化的标准差倍数，记为 \(\hat{\beta}^*\)。标准化效应量具有无量纲性，可直接跨性状、跨研究比较。

单个 SNP 解释的表型方差比例（Proportion of Variance Explained，PVE） 定义为：

\[ \text{PVE} = \frac{\hat{\beta}_{\text{SNP}}^2 \cdot \text{Var}(G)}{\text{Var}(Y)} = \frac{2p(1-p)\hat{\beta}_{\text{SNP}}^2}{\text{Var}(Y)} \]

其中 \(\text{Var}(G) = 2p(1-p)\)（加性编码下双等位基因 SNP 的方差）。在标准化表型（\(\text{Var}(Y) = 1\)）且已知 MAF \(p\) 时，\(\text{PVE} = 2p(1-p)(\hat{\beta}^*)^2\)。GWAS 发现的常见变异单位点 PVE 通常在 \(0.01\%\) 至 \(1\%\) 之间，身高 GWAS 中效应最强的位点（HMGA2 附近）PVE \(\approx 0.3\%\)。

\(R^2\)、\(F\) 统计量与 \(t\) 统计量的等价关系

对于单 SNP 检验，\(t\) 统计量、\(F\) 统计量和偏 \(R^2\) 之间存在精确的换算关系：

\[ F = t^2, \qquad \text{partial-}R^2 = \frac{F}{F + (n - p)} \approx \frac{t^2}{n} \quad (n \gg p) \]

因此，\(\hat{\beta}_{\text{SNP}}\) 的 \(t\) 统计量本身就是效应量的度量：在相同样本量和 MAF 下，\(t\) 值越大表示效应量越大。GWAS 软件（PLINK、REGENIE）的输出文件通常同时报告 \(\hat{\beta}\)、SE、\(t\)（或 \(Z\)）和 \(p\)，由这四个量可以相互推导出 PVE。

17.5.5.3 残差诊断

线性回归的推断依赖误差的正态性和同方差性。在 GWAS 中，连续性状（如身高、BMI）往往存在轻微偏态或异方差，残差诊断是评估模型假设是否成立的常规步骤。

正态 Q-Q 图（Normal Q-Q Plot）：将标准化残差的分位数与标准正态分位数比较，若点落在对角线附近则残差近似正态；偏离表明存在厚尾（heavy tail）或偏态，通常需要对表型进行 Box-Cox 变换或 rank-based inverse normal transformation（RINT）。

Scale-Location 图：纵轴为 \(\sqrt{|\text{标准化残差}|}\)，横轴为拟合值；若存在异方差（如方差随表型均值增大），图中会出现明显的上升或下降趋势。

以下模拟数据演示含协变量的线性回归关联检验，包括手动 OLS 推导、效应量指标计算和残差诊断。模拟设定：\(n = 1{,}000\)，MAF = 0.3；\(\beta_{\text{SNP}} = 0.5\)，性别效应 0.8，残差标准差 1.5，截距 2.0。

set.seed(42)
n     <- 1000
geno  <- rbinom(n, 2, 0.3)
sex   <- rbinom(n, 1, 0.5)
pheno <- 2.0 + 0.5 * geno + 0.8 * sex + rnorm(n, sd = 1.5)

# ---- Manual OLS via matrix algebra ----
X_mat     <- cbind(intercept = 1, geno = geno, sex = sex)
y_vec     <- pheno
n_obs     <- nrow(X_mat)
p_par     <- ncol(X_mat)

XtX       <- t(X_mat) %*% X_mat          # p x p: X'X
Xty       <- t(X_mat) %*% y_vec          # p x 1: X'y
beta_ols  <- solve(XtX, Xty)             # OLS solution: (X'X)^{-1} X'y

resid_ols  <- y_vec - X_mat %*% beta_ols
sigma2_hat <- sum(resid_ols^2) / (n_obs - p_par)   # unbiased residual variance
se_ols     <- sqrt(diag(sigma2_hat * solve(XtX)))  # sqrt of diagonal of Var(beta)
t_stat     <- beta_ols / se_ols
p_ols      <- 2 * pt(abs(t_stat), df = n_obs - p_par, lower.tail = FALSE)

cat(sprintf("beta_SNP (OLS)  = %.4f  (true = 0.5)\n", beta_ols["geno"]))
cat(sprintf("SE              = %.4f\n", se_ols["geno"]))
cat(sprintf("t               = %.4f,  p = %.4e\n", t_stat["geno"], p_ols["geno"]))

# ---- Effect size metrics ----
maf      <- mean(geno) / 2                        # estimated MAF
var_g    <- 2 * maf * (1 - maf)                  # Var(G) under additive encoding
var_y    <- var(y_vec)
PVE      <- var_g * beta_ols["geno"]^2 / var_y   # proportion of variance explained
cat(sprintf("\nMAF             = %.3f\n", maf))
cat(sprintf("Var(G)          = %.4f\n", var_g))
cat(sprintf("PVE             = %.4f%%  (true ≈ %.4f%%)\n",
            PVE * 100, var_g * 0.5^2 / (var_g * 0.5^2 + 0.8^2 * 0.25 + 1.5^2) * 100))

# Standardised effect size: beta* (effect in SD units of standardised phenotype)
pheno_std <- scale(pheno)                         # mean 0, variance 1
fit_std   <- lm(pheno_std ~ geno + sex)
beta_star <- coef(fit_std)["geno"]
cat(sprintf("beta* (standardised) = %.4f  [1 minor allele ↔ %.3f SD]\n",
            beta_star, beta_star))

  theme_classic()

GWAS 连续性状的标准预处理流程

在运行线性回归之前，连续性状通常经过以下预处理：（1）正态化变换——对偏态表型（BMI、CRP 等）先取对数或进行 rank-based inverse normal transformation（RINT），使残差更接近正态；（2）协变量回归残差化——将表型对年龄、性别、年龄²、技术批次等固定协变量做线性回归，取残差后进行关联检验，等价于在完整模型中同时纳入这些协变量（FWL 定理保证两者系数相同）；（3）表型标准化——将经过残差化的表型标准化为均值 0、标准差 1，使 \(\hat{\beta}\) 具有可比的跨性状含义。

线性回归中的 PCA 校正

线性回归与逻辑回归的 PCA 校正原理完全相同：将前 \(K\) 个主成分作为协变量纳入模型。对于线性模型，FWL 定理严格成立——预先将表型和所有 SNP 对主成分矩阵做残差化，与直接在含主成分的完整模型中估计 \(\hat{\beta}_{\text{SNP}}\) 数值上完全等价。完整的三向对比模拟（无校正 / 真实标签校正 / PCA 校正）在后续”群体分层的危害”小节中演示。

17.5.6 Wald 检验、Score 检验与似然比检验

在逻辑回归和混合线性模型框架下，对单个 SNP 效应 \(\beta_{\text{SNP}}\) 的显著性可以通过三种等价的渐近检验来评估，理解三者的区别有助于正确解读不同软件的输出格式。

17.5.6.1 三种检验的统计原理

Wald 检验（Wald test） 是 GWAS 软件中最常用的方法。在 MLE 收敛后，直接利用参数估计值和标准误计算统计量：

\[ W = \frac{\hat{\beta}_{\text{SNP}}^2}{\widehat{\text{Var}}(\hat{\beta}_{\text{SNP}})} \sim \chi^2_1 \]

Wald 检验的优点是计算高效——每个 SNP 只需拟合一次含 SNP 的完整模型，无需额外拟合零假设模型。其局限在于当参数真值远离零（即效应量很大）时，标准误的估计可能不稳定，导致 Wald 统计量偏小（Hauck-Donner 效应）；但在 GWAS 常见的小效应量场景下，此问题几乎不影响结论。

Score 检验（Score test / Lagrange multiplier test） 仅在零假设下（\(\beta_{\text{SNP}} = 0\)）估计参数，利用得分函数（对数似然对参数的一阶导数）在零假设处的值进行检验：

\[ S = \frac{\left[\frac{\partial \ln L}{\partial \beta_{\text{SNP}}}\bigg|_{\hat{\boldsymbol{\beta}}_0}\right]^2}{\mathcal{I}(\hat{\boldsymbol{\beta}}_0)} \sim \chi^2_1 \]

Score 检验的最大优势是计算效率：对于全基因组数百万个 SNP，只需拟合一次不含 SNP 的零假设模型（含截距和协变量），随后对每个 SNP 计算在该空模型残差下的得分统计量，无需为每个 SNP 单独迭代至收敛。REGENIE、SAIGE 等现代大规模 GWAS 工具在内部均采用 Score 检验框架。

似然比检验（Likelihood Ratio Test，LRT） 同时拟合含 SNP 的完整模型和不含 SNP 的零假设模型，取两个对数似然之差：

\[ \Lambda = 2[\ln L(\hat{\boldsymbol{\beta}}) - \ln L(\hat{\boldsymbol{\beta}}_0)] \sim \chi^2_1 \]

LRT 在理论上是三种检验中最优的（在 Neyman-Pearson 框架下最接近最优检验），在中等样本量下表现优于 Wald；缺点是计算量是 Wald 的两倍（每个 SNP 需拟合两次模型）。GEMMA 提供 -lmm 2 选项即为 LRT，适合精细定位阶段对少数候选位点的精确评估。

17.5.6.2 三者的渐近等价性

在样本量趋于无穷时，三种检验的统计量渐近等价：\(W \approx S \approx \Lambda\)。在 GWAS 的实际应用中（通常 \(N > 1000\)，效应量 OR \(\in [1.0, 1.5]\)），三者结果差异极小，p 值在同一数量级内。以下代码验证三种检验在小模拟数据集上的数值一致性：

set.seed(42)
n    <- 1000
geno <- rbinom(n, 2, 0.3)
sex  <- rbinom(n, 1, 0.5)
case <- rbinom(n, 1, plogis(-2 + 0.6 * geno + 0.4 * sex))

# Full model (with SNP)
fit_full <- glm(case ~ geno + sex, family = binomial)
# Null model (without SNP)
fit_null <- glm(case ~ sex,        family = binomial)

# Wald test
b_snp  <- coef(fit_full)["geno"]
se_snp <- sqrt(diag(vcov(fit_full)))["geno"]
p_wald <- pchisq((b_snp / se_snp)^2, df = 1, lower.tail = FALSE)

# LRT
lrt_stat <- 2 * (logLik(fit_full) - logLik(fit_null))
p_lrt    <- pchisq(as.numeric(lrt_stat), df = 1, lower.tail = FALSE)

# Score test (manual)
mu0      <- fitted(fit_null)
score_val <- sum(geno * (case - mu0))        # dL/d(beta_SNP) at null
info_val  <- sum(geno^2 * mu0 * (1 - mu0))  # Fisher info at null
p_score  <- pchisq(score_val^2 / info_val, df = 1, lower.tail = FALSE)

cat(sprintf("Wald  p = %.4e\n", p_wald))
cat(sprintf("LRT   p = %.4e\n", p_lrt))
cat(sprintf("Score p = %.4e\n", p_score))

软件选择指南

  • 探索性全基因组扫描：优先选用支持 Score 检验的工具（REGENIE、SAIGE），在百万级 SNP 上效率最高；
  • 精细定位或候选位点验证：使用 LRT（GEMMA -lmm 2）获得更精确的 p 值；
  • 报告与元分析：GWAS 结果通常以 Wald 统计量对应的 \(\beta\) 和 SE 报告，便于元分析汇总。

17.6 混合线性模型（MLM）与群体分层控制

17.6.1 群体分层的危害与历史教训

当研究样本来自遗传背景不同的亚群体时，如果某一表型在各亚群中的频率不同，而各亚群的等位基因频率也存在系统差异，则即使某个 SNP 与表型没有因果关系，其关联检验也可能出现显著的 p 值。这种现象称为群体分层混杂（population stratification confounding）。

群体分层问题在 GWAS 早期造成了不少混乱。一个典型案例是：某项早期 GWAS 宣称发现了身高相关的 SNP，后来发现该 SNP 实际上是区分欧洲不同亚群体的祖先标记，其与身高的”关联”完全来源于北欧人群平均身高高于南欧人群的人口学差异，与基因本身无关。这一教训推动了群体分层控制方法的快速发展。EIGENSTRAT 方法（Price et al., 2006）是第一个被广泛接受的基于 PCA 的群体分层校正方案：对全基因组 SNP 矩阵进行主成分分析，提取出反映群体遗传结构的主成分（PC），将前 k 个 PC 作为协变量纳入回归模型，从而吸收群体间的系统性差异。

17.6.2 群体分层的危害

以下模拟数据定量展示群体分层混杂的危害与 PCA 校正的效果。模拟设定：两个亚群体各 500 个体；群体 1 的 MAF = 0.6、表型均值 = 5.0，群体 2 的 MAF = 0.2、表型均值 = 3.0；在各亚群内部 SNP 与表型无因果关系（表型由正态分布独立生成）。比较未校正模型与加入群体标签协变量后的 p 值，可直观看到混杂效应的量级。

# Simulation: population stratification confounding
set.seed(2024)
n_pop <- 500  # individuals per population

# Population 1: high disease prevalence, high MAF
pop1_geno  <- rbinom(n_pop, 2, 0.6)
pop1_pheno <- rnorm(n_pop, mean = 5.0, sd = 1.0)

# Population 2: low disease prevalence, low MAF
pop2_geno  <- rbinom(n_pop, 2, 0.2)
pop2_pheno <- rnorm(n_pop, mean = 3.0, sd = 1.0)

geno_all  <- c(pop1_geno, pop2_geno)
pheno_all <- c(pop1_pheno, pop2_pheno)
pop_label <- factor(c(rep("Pop1", n_pop), rep("Pop2", n_pop)))

# Naive test (no stratification correction)
fit_naive <- lm(pheno_all ~ geno_all)
p_naive <- summary(fit_naive)$coefficients["geno_all", "Pr(>|t|)"]

# Corrected test (add population as covariate)
fit_corrected <- lm(pheno_all ~ geno_all + pop_label)
p_corrected   <- summary(fit_corrected)$coefficients["geno_all", "Pr(>|t|)"]

cat(sprintf("Naive p-value:     %.2e\n", p_naive))
cat(sprintf("Corrected p-value: %.2e\n", p_corrected))

上方的”校正”利用了已知的群体标签，相当于直接告诉模型”哪个样本来自哪个群体”。然而在真实 GWAS 中，样本的群体来源往往未知或不精确；PCA 校正的核心思路是：从全基因组多 SNP 基因型矩阵中提取主成分，以数据驱动的方式发现并量化群体结构，再将这些主成分作为协变量纳入回归模型。以下代码在同一模拟框架下，扩展为每群体 800 个个体（共 1,600 个样本）、500 个 SNP 的基因型矩阵，演示完整的 PCA 校正流程，并与无校正结果及真实标签校正结果三向比较。

# Extended simulation: PCA-based stratification correction

17.6.3 混合线性模型的数学框架

主成分分析（PCA）校正虽然简便，但存在两个局限性：其一，PCA 主成分仅能捕捉群体间的主要遗传结构差异，对于细粒度的亚群体差异或连续的遗传混合（admixture）格局，有限个主成分无法完全捕捉；其二，样本间的亲缘关系（如家系成员的纳入）不能被 PCA 充分校正。针对这两个问题，混合线性模型（Mixed Linear Model，MLM） 提供了更为全面的解决方案。

混合线性模型通过在模型中引入随机效应来同时控制群体分层和亲缘关系：

\[ \mathbf{y} = \mathbf{X}\boldsymbol{\beta} + \mathbf{Z}\mathbf{u} + \boldsymbol{\varepsilon} \]

其中： - \(\mathbf{y}\)：\(n \times 1\) 表型向量； - \(\mathbf{X}\)：固定效应设计矩阵，包含 SNP 基因型编码和协变量（如 PCA 主成分）； - \(\boldsymbol{\beta}\)：固定效应系数向量； - \(\mathbf{u}\)：随机遗传效应，\(\mathbf{u} \sim \mathcal{N}(\mathbf{0},\ \sigma_g^2 \mathbf{K})\)； - \(\mathbf{K}\)：\(n \times n\) 亲缘关系矩阵（kinship matrix / genomic relatedness matrix，GRM）； - \(\boldsymbol{\varepsilon}\)：残差，\(\boldsymbol{\varepsilon} \sim \mathcal{N}(\mathbf{0},\ \sigma_e^2 \mathbf{I})\)。

因此表型的边际分布为：

\[ \mathbf{y} \sim \mathcal{N}\!\left(\mathbf{X}\boldsymbol{\beta},\ \sigma_g^2 \mathbf{K} + \sigma_e^2 \mathbf{I}\right) \]

亲缘关系矩阵 K 由基因组全局 SNP 估计，其第 \((i,j)\) 元素反映个体 \(i\) 与个体 \(j\) 的遗传相似度。常用的计算方式（VanRaden 2008）为：

\[ K_{ij} = \frac{1}{M} \sum_{l=1}^{M} \frac{(G_{il} - 2p_l)(G_{jl} - 2p_l)}{2p_l(1-p_l)} \]

其中 \(G_{il}\) 为个体 \(i\) 在第 \(l\) 个 SNP 的基因型编码，\(p_l\) 为次等位基因频率，\(M\) 为 SNP 总数。

17.6.4 模型参数的估计：REML

MLM 中方差分量 \(\sigma_g^2\) 和 \(\sigma_e^2\) 通常采用约束最大似然法（Restricted Maximum Likelihood，REML） 进行估计。与普通最大似然估计（MLE）相比，REML 通过对数据施加线性约束消除固定效应的影响，从而得到方差分量的无偏估计，在样本量较小时尤为重要。REML 估计的计算瓶颈在于矩阵求逆，时间复杂度为 \(O(n^3)\)。当 \(n > 10,000\) 时，标准 REML 迭代将变得极为耗时。为此，GEMMA 采用了谱分解优化，将复杂度降至 \(O(n^2)\)，而 REGENIE 进一步通过随机数学技巧实现了近似线性时间复杂度，使 GWAS 分析可扩展至 UK Biobank 规模的队列。

17.6.5 SNP 遗传力（\(h^2_{\text{SNP}}\)）的估计

混合线性模型不仅能够校正群体分层，其方差分量估计还直接提供了关于遗传架构的定量信息。REML 估计的方差分量 \(\hat{\sigma}_g^2\) 和 \(\hat{\sigma}_e^2\) 可用于计算 SNP 遗传力（SNP-based heritability，\(h^2_{\text{SNP}}\)）：

\[ h^2_{\text{SNP}} = \frac{\hat{\sigma}_g^2}{\hat{\sigma}_g^2 + \hat{\sigma}_e^2} \]

\(h^2_{\text{SNP}}\) 衡量的是表型总方差中可被当前基因芯片所捕获的常见 SNP（通常 MAF > 0.01）所解释的比例。它与经典双生子研究估计的狭义遗传力（narrow-sense heritability，\(h^2\)） 之间存在系统性差距（即”缺失遗传力”问题，missing heritability）：前者仅计入常见变异的加性效应，而后者还包含稀有变异、显性效应和上位效应的贡献。

GCTA-GREML 是估计 \(h^2_{\text{SNP}}\) 的标准工具。其实现思路是：将全基因组 SNP 构建 GRM，作为 MLM 随机效应的协方差结构，通过 REML 估计方差分量。以下命令演示了使用 GCTA 估计身高的 SNP 遗传力：

# Step 1: Compute GRM from all QC-passed SNPs
gcta64 \
  --bfile qc_step4 \
  --make-grm \
  --out grm_all \
  --thread-num 8

# Step 2: GREML to estimate SNP heritability
gcta64 \
  --grm grm_all \
  --pheno height.phen \
  --qcovar pca_covariates.txt \
  --reml \
  --out greml_height \
  --thread-num 8

# Output: greml_height.hsq
# Key columns: Variance (sigma2_g, sigma2_e, total), h2

黄金法则：\(h^2_{\text{SNP}}\) 的解读

\(h^2_{\text{SNP}}\) 的估计是整个 GWAS 发现的理论上界：若某性状的 \(h^2_{\text{SNP}}\) = 0.20，则基于当前芯片数据最多只能解释 20% 的表型方差，无论样本量增加到多大。身高的 \(h^2_{\text{SNP}} \approx 0.45\)，而双生子研究估计的 \(h^2 \approx 0.80\)，两者的差距（约 0.35）构成了理论上的”缺失遗传力”，部分来源于稀有变异和结构变异，部分来源于常见变异之间的上位交互效应。

17.6.7 R 模拟实验：MLM 对亲缘关系混杂的校正效果

实验目标与设计思路

本模拟实验旨在从数值层面验证混合线性模型（MLM）相对于普通线性回归（OLS）在控制亲缘关系混杂方面的优越性。实验设计如下：构造一个包含两类个体的样本集——一部分为无关个体（unrelated），另一部分为高度相关的”家系簇”（related clusters，模拟同胞关系）；目标 SNP 与表型在群体内部无真实因果效应，但表型在家系簇内部存在显著的家系内相关（即共享遗传背景和环境效应），同时目标 SNP 在家系簇中的 MAF 与无关个体存在差异，从而制造出亲缘关系驱动的虚假关联。通过对比三个模型的检验结果，可以直观看到：

  • OLS（无校正）：I 类错误膨胀，对无因果效应的 SNP 产生显著的假阳性 p 值；
  • OLS + PCA 校正：对群体分层有一定控制，但无法充分吸收家系内相关性；
  • MLM（GRM 随机效应）：通过亲缘关系矩阵将家系内相关结构完整建模为随机效应，SNP 固定效应估计趋于无偏，假阳性 p 值恢复至期望水平（接近均匀分布）。

亲缘关系矩阵（GRM）的计算原理

在模拟实验中，GRM \(\mathbf{K}\) 依据 VanRaden（2008）方法由全基因组背景 SNP 矩阵估计。对于标准化后的基因型矩阵 \(\tilde{\mathbf{G}}\)（\(n \times M\)，每列按 \(\tilde{G}_{il} = (G_{il} - 2p_l)/\sqrt{2p_l(1-p_l)}\) 标准化），GRM 可由外积直接计算：

\[ \mathbf{K} = \frac{1}{M} \tilde{\mathbf{G}} \tilde{\mathbf{G}}^\top \]

\(\mathbf{K}\) 的第 \((i,j)\) 元素反映个体 \(i\) 与 \(j\) 在全基因组范围内的遗传相似度：对角元素（\(K_{ii}\)）约为 1（个体与自身完全相同）；全同胞的 \(K_{ij}\) 约为 0.5；无关个体的 \(K_{ij}\) 接近 0。

MLM 的固定效应估计：广义最小二乘（GLS）

在方差分量 \(\sigma_g^2\) 和 \(\sigma_e^2\) 已知的情况下，MLM 的固定效应估计等价于广义最小二乘（Generalized Least Squares，GLS）。定义表型的协方差矩阵为：

\[ \mathbf{V} = \sigma_g^2 \mathbf{K} + \sigma_e^2 \mathbf{I} \]

则固定效应的 GLS 估计为：

\[ \hat{\boldsymbol{\beta}}_{\text{GLS}} = \left(\mathbf{X}^\top \mathbf{V}^{-1} \mathbf{X}\right)^{-1} \mathbf{X}^\top \mathbf{V}^{-1} \mathbf{y} \]

其协方差矩阵为 \(\text{Var}(\hat{\boldsymbol{\beta}}_{\text{GLS}}) = \left(\mathbf{X}^\top \mathbf{V}^{-1} \mathbf{X}\right)^{-1}\)。与 OLS 相比，GLS 通过 \(\mathbf{V}^{-1}\) 对观测值进行”去相关”加权：家系内高度相关的个体对估计的信息贡献被相应缩减，使关联统计量不再被家系内相关性所放大，从而实现对亲缘关系混杂的控制。

在实际软件（GEMMA、BOLT-LMM）中，方差分量 \(\sigma_g^2\) 和 \(\sigma_e^2\) 通过 REML 从数据中估计，随后代入上式完成固定效应推断。本模拟同样实现了完整的 REML 迭代：以梯度上升法最大化受限对数似然，配合步长减半线搜索（line search）保证收敛，最终得到数据驱动的方差分量估计值，再代入 GLS 公式推断固定效应。

以下代码实现上述完整流程，并以三向比较（OLS / OLS+PCA / MLM-REML-GLS）的形式输出目标 SNP 的检验结果：

set.seed(2025)

n_unrel <- 400          # unrelated individuals
n_fam   <- 20           # number of family clusters
n_sib   <- 10           # siblings per cluster
n_rel   <- n_fam * n_sib
n       <- n_unrel + n_rel
n_snp   <- 300          # background SNPs for GRM and PCA

# ---- Step 1: Simulate genotype matrix ----
# Unrelated: MAF drawn uniformly across SNPs
maf_bg       <- runif(n_snp, 0.1, 0.5)
G_unrel      <- mapply(function(m) rbinom(n_unrel, 2, m), maf_bg)

# Related (family clusters): siblings share ~50% of alleles
# Simulate by drawing a "parental haplotype" per family, then adding noise
G_rel_list <- lapply(seq_len(n_fam), function(f) {
  parent_geno <- rbinom(n_snp, 2, maf_bg)         # one shared parental genotype per family
  sib_mat <- t(replicate(n_sib, {
    # At each SNP: with prob 0.5 inherit the parental genotype (IBD),
    # otherwise draw independently from population MAF (non-IBD).
    # Expected pairwise IBD between any two siblings ≈ 0.5^2 = 0.25 per SNP pair,
    # giving an overall PI_HAT ≈ 0.5, consistent with full siblings.
    ifelse(rbinom(n_snp, 1, 0.5) == 1, parent_geno,
           rbinom(n_snp, 2, maf_bg))
  }))
  sib_mat
})
G_rel <- do.call(rbind, G_rel_list)               # n_rel x n_snp

G_all <- rbind(G_unrel, G_rel)                    # n x n_snp

# Target SNP: slightly higher MAF in related individuals (creates confounding)
maf_target_unrel <- 0.3
maf_target_rel   <- 0.5
geno_target_mlm  <- c(rbinom(n_unrel, 2, maf_target_unrel),
                      rbinom(n_rel,   2, maf_target_rel))

# ---- Step 2: Simulate phenotype ----
# Family-shared effect (polygenic background): correlated within families
fam_effect <- rep(rnorm(n_fam, sd = 2.0), each = n_sib)  # shared within sib
pheno_mlm  <- c(rnorm(n_unrel, mean = 0, sd = 1.5),
                fam_effect + rnorm(n_rel, sd = 1.0))
# NOTE: target SNP has NO causal effect on phenotype

# ---- Step 3: Compute GRM from background SNPs ----
p_snp    <- colMeans(G_all) / 2                   # estimated MAF per SNP
denom    <- sqrt(2 * p_snp * (1 - p_snp))
G_std    <- sweep(G_all, 2,
                  2 * p_snp, FUN = "-")           # center
G_std    <- sweep(G_std,  2, denom, FUN = "/")    # scale
G_std[is.nan(G_std)] <- 0                         # guard against MAF=0 or 1
K        <- (G_std %*% t(G_std)) / n_snp          # n x n GRM

# ---- Step 4: Compute PCA from background SNPs (for OLS+PCA) ----
svd_mlm  <- svd(G_std / sqrt(n - 1))
PC_mlm   <- svd_mlm$u %*% diag(svd_mlm$d)        # sample PC scores

# ---- Step 5: OLS — naive model (no correction) ----
X_ols    <- cbind(1, geno_target_mlm)
b_ols    <- solve(t(X_ols) %*% X_ols) %*% t(X_ols) %*% pheno_mlm
res_ols  <- pheno_mlm - X_ols %*% b_ols
s2_ols   <- sum(res_ols^2) / (n - 2)
se_ols2  <- sqrt(diag(s2_ols * solve(t(X_ols) %*% X_ols)))
p_ols    <- 2 * pt(abs(b_ols[2] / se_ols2[2]), df = n - 2, lower.tail = FALSE)

# ---- Step 6: OLS + PCA (top 2 PCs as covariates) ----
K_pc     <- 2
X_pca2   <- cbind(1, geno_target_mlm, PC_mlm[, seq_len(K_pc)])
b_pca2   <- solve(t(X_pca2) %*% X_pca2) %*% t(X_pca2) %*% pheno_mlm
res_p2   <- pheno_mlm - X_pca2 %*% b_pca2
s2_p2    <- sum(res_p2^2) / (n - 2 - K_pc)
se_pca2  <- sqrt(diag(s2_p2 * solve(t(X_pca2) %*% X_pca2)))
p_pca2   <- 2 * pt(abs(b_pca2[2] / se_pca2[2]),
                   df = n - 2 - K_pc, lower.tail = FALSE)

# ---- Step 7: REML estimation of variance components ----
# REML iteratively maximises the restricted log-likelihood:
#   log L_R = -0.5 * [ log|V| + log|X'V^{-1}X| + y'Py ]
# where P = V^{-1} - V^{-1}X(X'V^{-1}X)^{-1}X'V^{-1} is the projection matrix.
# Gradients with respect to sigma2_g and sigma2_e (AI-REML / average-information):
#   d(log L_R)/d(sigma2_g) = 0.5 * [-tr(PK)  + y'PKPy]
#   d(log L_R)/d(sigma2_e) = 0.5 * [-tr(P)   + y'PPy ]
# We use a simple gradient-ascent update with step size damping for stability.

X_mlm <- cbind(1, geno_target_mlm)
y_mlm <- pheno_mlm

reml_loglik <- function(sg2, se2) {
  V_r   <- sg2 * K + se2 * diag(n)
  # Use Cholesky for stable inversion
  ch    <- tryCatch(chol(V_r), error = function(e) NULL)
  if (is.null(ch)) return(-Inf)
  V_inv_r <- chol2inv(ch)
  XtVX_r  <- t(X_mlm) %*% V_inv_r %*% X_mlm
  ch2     <- tryCatch(chol(XtVX_r), error = function(e) NULL)
  if (is.null(ch2)) return(-Inf)
  P_r <- V_inv_r - V_inv_r %*% X_mlm %*% chol2inv(ch2) %*% t(X_mlm) %*% V_inv_r
  # REML log-likelihood (up to constant)
  log_det_V    <- 2 * sum(log(diag(ch)))
  log_det_XVX  <- 2 * sum(log(diag(ch2)))
  yPy          <- as.numeric(t(y_mlm) %*% P_r %*% y_mlm)
  -0.5 * (log_det_V + log_det_XVX + yPy)
}

reml_gradient <- function(sg2, se2) {
  V_r     <- sg2 * K + se2 * diag(n)
  ch      <- chol(V_r)
  V_inv_r <- chol2inv(ch)
  XtVX_r  <- t(X_mlm) %*% V_inv_r %*% X_mlm
  P_r     <- V_inv_r - V_inv_r %*% X_mlm %*%
             solve(XtVX_r) %*% t(X_mlm) %*% V_inv_r
  # Gradient w.r.t. sigma2_g
  PKg <- P_r %*% K
  dg  <- 0.5 * (-sum(diag(PKg)) + as.numeric(t(y_mlm) %*% PKg %*% P_r %*% y_mlm))
  # Gradient w.r.t. sigma2_e
  de  <- 0.5 * (-sum(diag(P_r)) + as.numeric(t(y_mlm) %*% P_r %*% P_r %*% y_mlm))
  c(dg, de)
}

# Gradient ascent with step-halving line search
sg2  <- var(y_mlm) * 0.5    # initialise near total variance
se2  <- var(y_mlm) * 0.5
lr   <- 0.01                 # initial learning rate
tol  <- 1e-6
max_iter_reml <- 200

for (it in seq_len(max_iter_reml)) {
  ll_cur <- reml_loglik(sg2, se2)
  grad   <- reml_gradient(sg2, se2)

  # Line search: halve step until likelihood improves
  step <- lr
  for (ls in seq_len(20)) {
    sg2_new <- max(sg2 + step * grad[1], 1e-6)
    se2_new <- max(se2 + step * grad[2], 1e-6)
    if (reml_loglik(sg2_new, se2_new) > ll_cur) break
    step <- step / 2
  }

  delta <- sqrt((sg2_new - sg2)^2 + (se2_new - se2)^2)
  sg2   <- sg2_new
  se2   <- se2_new
  if (delta < tol) break
}

cat(sprintf("REML estimates: sigma2_g = %.4f,  sigma2_e = %.4f\n", sg2, se2))
cat(sprintf("Heritability h2 = sigma2_g / (sigma2_g + sigma2_e) = %.3f\n",
            sg2 / (sg2 + se2)))

# ---- Step 8: MLM-GLS using REML variance components ----
V_reml   <- sg2 * K + se2 * diag(n)
V_inv_r  <- chol2inv(chol(V_reml))

XtVX_r   <- t(X_mlm) %*% V_inv_r %*% X_mlm
XtVy_r   <- t(X_mlm) %*% V_inv_r %*% y_mlm
b_mlm    <- solve(XtVX_r, XtVy_r)          # GLS fixed-effect estimator
se_mlm   <- sqrt(diag(solve(XtVX_r)))
t_mlm    <- b_mlm[2] / se_mlm[2]
p_mlm    <- 2 * pt(abs(t_mlm), df = n - 2, lower.tail = FALSE)

# ---- Step 9: Print results ----
cat(sprintf("\nTarget SNP has NO true causal effect (ground truth beta = 0)\n\n"))
cat(sprintf("OLS naive        : beta = %6.4f,  p = %.3e\n", b_ols[2],  p_ols))
cat(sprintf("OLS + PCA (K=%d) : beta = %6.4f,  p = %.3e\n", K_pc,
            b_pca2[2], p_pca2))
cat(sprintf("MLM-REML-GLS     : beta = %6.4f,  p = %.3e\n", b_mlm[2],  p_mlm))

# ---- Step 10: Visualize GRM heatmap ----
library(ggplot2)

# Order samples: unrelated first, then by family
ord       <- c(seq_len(n_unrel),
               n_unrel + seq_len(n_rel))
K_ord     <- K[ord, ord]

K_df      <- data.frame(

期望结果解读

  • OLS naive：\(\hat{\beta}\) 显著偏离 0，p 值极小（假阳性）。目标 SNP 在家系样本中频率偏高，而家系样本的表型均值因家系效应整体偏移，两者的相关性完全来源于混杂而非因果效应。
  • OLS + PCA：p 值有所改善，但仍可能偏小。前 2 个主成分能够区分相关与无关样本的整体遗传结构差异，但无法完整建模家系内部的相关性——PCA 将家系内相关压缩为连续坐标，而非直接建模协方差结构。
  • MLM-GLS：p 值应接近不显著（> 0.05），\(\hat{\beta}\) 接近 0。GRM 将每对样本间的遗传相似度直接编码为协方差结构，\(\mathbf{V}^{-1}\) 对家系内高度相关的观测进行降权，使关联统计量恢复无偏性。GRM 热图中右下角的块状结构直观展示了家系簇的内部高相似性，正是这种结构被 MLM 建模并”校正”的对象。
